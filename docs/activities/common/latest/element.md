# Common Element - akaBot Docs

Content
:   The Common Element Activities Pack contains all the activities used for creating automation projects. These activities enable the robots to:

    * Perform browser interaction and window manipulation.
    * Simulate human interaction, such as performing mouse and keyboard commands or typing and extracting text.
