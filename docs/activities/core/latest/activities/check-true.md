# Check True - akaBot Docs

RCA.Activities.Core.CheckTrue

## **Description**

This activity allows you to check whether the expression is true.

![image-20220505095635-1.jpeg](/img/9b6f2f_image-20220505095635-1.jpeg)

(\* for Mandatory)

## **Properties**

**Input**

* **Boolean Expression (Boolean)**\* - The Boolean expression that you want to evaluated stored in a variable.

**Misc**

* **Display Name (String)** - The name of this activity. You can edit the name of the activity to organize and structure your code better.  
  E.g: [997467815] Check True
* **Public (Checkbox)** - If you check it, the data of this activity will be shown in the log. Be careful, consider data security before using it.

**Output**

* **Result (Boolean)** - The output of the text that you get from this activity.
