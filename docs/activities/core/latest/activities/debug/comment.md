# Comment - akaBot Docs

RCA.Activities.Core.Comment

## **Description**

The activity allows you comment to a workflow.

![image-20220505092245-1.jpeg](/img/e3cbe3_image-20220505092245-1.jpeg)

(\* for Mandatory)

## **Properties**

**Input**

* **Text (String)**\* - Write your comments in the Text property.  
  E.g: This activity is hard to understand

**Misc**

* **Public (Checkbox)** - If you check it, the data of this activity will be shown in the log. Be careful, consider data security before using it.
* **Display Name (String)** - The name of this activity. You can edit the name of the activity to organize and structure your code better.  
  E.g: [556910226] Comment
