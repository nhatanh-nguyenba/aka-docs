# Add Log Fields - akaBot Docs

RCA.Activities.Core.AddLogFields

## **Description**

This activity allows you to add customized log fields to each and every Log Message.

![image-20220513103428-1.png](/img/7d6799_image-20220513103428-1.png)

(\* for Mandatory)

## **Properties**

**Input**

* **Fields (Collection of String)**\* - The collection of string arguments to be added from subsequent log lines.

**Misc**

* **Public (Checkbox)** - If you check it, the data of this activity will be shown in the log. Be careful, consider data security before using it.
* **Display Name (String)** - The name of this activity. You can edit the name of the activity to organize and structure your code better.  
  E.g: [056739368] Add Log Fields
