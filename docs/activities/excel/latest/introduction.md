# Excel Activities - akaBot Docs

Content
:   The Excel Activities offers deep integration with Microsoft Excel and ships with a number of activities that automate the tasks you perform in Excel. These activities enable the robots to:

    * Read information from a cell, columns, rows or ranges, and write to other spreadsheets or workbooks.
    * Execute macros, and even extract formulas.

    Also, there are other types of activities in this pack that aid in the creation and execution of the automation projects themselves, such as logical operators and expressions.
