# Introduction

The **Form Builder activities** package contains activities for displaying and manipulating Form Builder forms in automation workflows.

These activities enable robots to:

* Display a form from a JSON schema file.
* Pass data into form fields and retrieve submitted data.
* Get or set field values while a form is open.
* Enable, disable, focus, reset, and close a displayed form.

Use **Display Form** as the container activity for most Form Builder operations.
