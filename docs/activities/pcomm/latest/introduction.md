# Introduction

The **PCOMM** (IBM Personal Communications) activity package enables you to automate tasks on terminal emulation sessions. By interfacing directly with IBM Personal Communications, these activities allow you to connect to mainframe systems, simulate typing, navigate screens, and retrieve text or verify terminal states.

This package is essential for automating legacy host systems (such as IBM 3270 or 5250) without relying on manual user interaction or visual scraping.
