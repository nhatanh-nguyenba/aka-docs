# Release Notes

## v3.1.0

**Features & Enhancements**

* **Enhanced User Documentation**: Rewrote activity property descriptions to be more user-friendly and aligned with standard workflow design terminology.
* **Property Label Alignment**: Updated and standardized property labels in documentation to exactly match the properties panel layout in akaBot Studio (such as renaming `Rows`/`Cols` to `Row`/`Column` and `Text` to `Wait String`).
* **Visual Workflows**: Added screenshots and visual aids for key terminal activities to assist in designing automated processes.

## v2.1.0

* **Initial Release**: Supported core terminal emulation activities for IBM Personal Communications (PCOMM), including Connection Scope, Send Keys, Move Cursor, Send Command Key, Get Text, Get Text Rect, Set Text, Set Text Rect, Start/Stop Communication, Wait For, and Wait For String.
