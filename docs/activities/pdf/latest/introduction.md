# RCA.Activities.PDF

RCA.Activities.PDF provides activities for reading PDF text, reading PDF with OCR, exporting PDF pages as images, merging and splitting PDF files, encrypting/decrypting PDF files, getting page count, and reading PDF document properties.

## Activities

* Merge PDF Files
* Split PDF Files
* Get PDF Page Count
* Export PDF Page As Image
* Decrypt PDF
* Encrypt PDF
* Get PDF properties
* Read PDF Text
* Read PDF With OCR
