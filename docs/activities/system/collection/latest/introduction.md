# Collections - akaBot Docs

The **Collections activities** sub-package contains activities that work with [Collection<T> Class](https://docs.microsoft.com/en-us/dotnet/api/system.collections.objectmodel.collection-1), enabling the adding and clearing or removing **item**in a specified collection.

All of these activities can be found in the **Toolbox** panel, under **System > Collections**.
