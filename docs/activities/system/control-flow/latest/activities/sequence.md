# Sequence - akaBot Docs

System.Activities.Statements.Sequence

## **Description**

Sequence is a container used to organize a series of activities that are executed sequentially in a predefined order.

![image-20230417141912-1.png](/img/sequence.png)

\* indicates required fields.

## **Properties**

**Misc**

* **Public (Checkbox)**: If selected, the activity data will be logged. Consider data security before using this property.
* **Display Name (String)**: The name of this activity. You can edit it to better organize and structure your workflow.  
  Example: `[552074308] Sequence`
