# Attended - Unattended Agent

## Content

| Criteria | Attended | Unattended |
| --- | --- | --- |
| **Strength** | - Assist individual users in handling small and repetitive tasks.<br/>- Collaborate with humans to complete long-running processes. | Running long processes or automations without the need of human interaction. |
| **Operational Method** | Processes are triggered by users or specific user events. | Not depend on users to trigger processes. They are assigned tasks and scheduled to run through akaBot Center. |
| **Machine Requirement for Agent Execution** | Agent needs to be installed on the same machine as the one where the user conducts daily tasks.| Agents can be installed on any machine connected to akaBot Center. |
| **When Do Your Business Need This Type of Agent?** | When you need a personal assistant for an end-user (HR Personnel, Call Center Operator, or Data Validation Specialist) to help them with small day-to-day tasks or specific parts of a process. | When your company requires numerous Agents operating on a large scale in the background to streamline repetitive or prolonged tasks for multiple users or entire departments. |
