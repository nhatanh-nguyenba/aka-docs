# Akabot Hub

Akabot Hub is the central marketplace and collaboration platform for automation assets. Teams publish reusable workflows, activity packages, templates, and best-practice documentation to Hub so that the rest of the organization can discover and reuse them — accelerating delivery and preventing duplicate effort.

## Key features

- **Asset marketplace** — Browse and search workflows, activity packages, templates, and connectors published by your teams or the Akabot community.
- **One-click import** — Import any asset directly into Akabot Studio with a single click.
- **Versioning** — Each asset has a full version history; consumers can pin to a specific version or always use the latest.
- **Ratings & reviews** — Leave feedback on assets to help others choose the best solution.
- **Documentation hosting** — Each asset page includes a README, usage examples, and changelog.
- **Governance** — Administrators can require approval before an asset is published publicly within the organization.

## Asset types

| Type | Description |
|------|-------------|
| **Workflow templates** | Pre-built workflow skeletons for common use cases (invoice processing, onboarding, etc.). |
| **Activity packages** | Custom NuGet activity packages installable directly in Studio. |
| **Connectors** — | Pre-configured HTTP connectors for popular SaaS applications. |
| **Process documentation** | Process definition documents (PDDs) and solution design documents (SDDs). |

## Getting started

1. Open the Hub portal from Akabot Center.
2. Search for an asset relevant to your automation project.
3. Click **Import to Studio** or download the package manually.
4. Publish your own asset by clicking **Publish** from Studio or from the Hub portal.
