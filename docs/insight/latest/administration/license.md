## Manage license

Insights provides a license package management and tracking system, which allows you to monitor the number of accounts associated with each package. The number of users can be increased by purchasing additional license packages.

**Note:** It is recommended to purchase a license pack at least 3 months before the first license pack expires

### Add new license

1. Access to Insights.
2. Go to the **Setting** menu in the top-right corner and select **Administration**.
3. Go to the **License**and select **Add license** button.
4. A new dialog box opens, click **Select license file (\*.lic)**

![image-20230420164436-1.png](/static/img/image-20230420164436-1.png)

5. Select the license file provided by the vendor and click **Confirm**.
