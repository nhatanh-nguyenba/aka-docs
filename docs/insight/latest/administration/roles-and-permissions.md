## Manage User Roles

Insights supports 2 main permissions: ADMINISTRATOR\_RESOURCE & DATASET\_RESOURCE. From 2 basic permissions, users can be assigned different roles.

The impact of permissions on the system is outlined in the following table:

|  |  |  |
| --- | --- | --- |
| **Permissions** | **ADMINISTRATOR\_RESOURCE** | **DATASET\_RESOURCE** |
| Assign User | X |  |
| Configure Business Dataset |  | X |

### Create Role

1. Access to Insights.
2. Go to the **Setting** menu in the top-right corner and select **Administration**.
3. Go to the **Roles**and select **Create new** button.

![image-20230420153140-2.png](/static/img/image-20230420153140-2.png)

4. A new dialog box opens, enter role name and select the level of impact on the system.

**Note:** The increasing influence ranges from left to right, from none to delete.

      5. Select **Save** to confirm the action.

### Edit Role

1. Go to the 3-dot menu on the leftmost side of the role card and select **Update.**

![image-20230420154728-3.png](/static/img/image-20230420154728-3.png)

2. A new dialog box opens, enter the new role name and re-select the level of impact on the system.

3. Select **Save** to confirm the action.

### View Role

1. Go to the 3-dot menu on the leftmost side of the role card and select **View.**

![image-20230420155357-4.png](/static/img/image-20230420155357-4.png)

2. A new dialog box opens, select **Cancel** to close the dialog box.
