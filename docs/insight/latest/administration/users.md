## Manage User Access

As an Insights administrator, you can manage user access to Insights. You can assign permissions to users using an existing Center account to log in to the Insights.

### Assign Users

1. Access to Insights.
2. Go to the **Setting** menu in the top-right corner and select **Administration**.

![image-20230420143351-1.png](/static/img/image-20230420143351-1.png)

3. Go to the 3-dot menu on the leftmost side of the user card and select **Edit User.**

     4. Assign roles and Tableau accounts to your users.

### Unassign Tableau user

1. Go to the 3-dot menu on the leftmost side of the user card and select **Unassign tableau user.**
2. A new dialog box opens. Click **Unassign** to confirm the action.
