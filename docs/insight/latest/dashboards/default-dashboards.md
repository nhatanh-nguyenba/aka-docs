## 2. Edit default dashboards

**Step 1:** Choose the dashboard you would like to edit at the Dashboards screens

![image-20230413171635-2.png](/static/img/image-20230413171635-2.png)

Step 2: At the dashboard details screen, click button **Edit dashboard**

![image-20230413171515-1.png](/static/img/image-20230413171515-1.png)

Step 3: After clicking, the Edit view will be opened as below. From here, you can edit your default dashboard.

![image-20230413171828-3.png](/static/img/image-20230413171828-3.png)

**Note**: To become proficient in creating/editing dashboards, please refer to the [Tableau Documentation](https://help.tableau.com/current/pro/desktop/en-us/gettingstarted_overview.htm) (the embedded platform of akaBot Insight).
