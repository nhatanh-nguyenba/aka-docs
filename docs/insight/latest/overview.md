# Akabot Insight

Akabot Insight is the analytics and reporting module that turns raw execution data into actionable business intelligence. Operations managers, CoE teams, and executives use Insight dashboards to track ROI, spot bottlenecks, and make data-driven decisions about automation investments.

## Key features

- **Pre-built dashboards** — Out-of-the-box views for process performance, robot utilization, queue health, and error trends.
- **Custom dashboards** — Drag-and-drop dashboard builder with charts, KPI cards, tables, and filters.
- **Real-time data** — Metrics refresh automatically as robots execute; no manual refresh needed.
- **Business metrics** — Map automation outcomes to business KPIs such as FTE savings, processing time, and cost per transaction.
- **Scheduled reports** — Export PDF or Excel reports and deliver them to stakeholders on a schedule.
- **Drill-down** — Click any metric to navigate to individual execution logs for root-cause analysis.

## Key metrics tracked

| Metric | Description |
|--------|-------------|
| **Execution success rate** | Percentage of workflow runs that completed without errors. |
| **Average handling time** | Mean time to process one transaction or queue item. |
| **Robot utilization** | Percentage of time each robot was actively running vs. idle. |
| **FTE equivalent saved** | Estimated hours of manual work automated per period. |
| **Queue throughput** | Items processed per hour across all active queues. |
| **Exception rate** | Business and application exceptions per 100 transactions. |

## Getting started

1. Open Akabot Center and navigate to **Insight**.
2. Select a pre-built dashboard or create a custom one.
3. Configure the date range, environment, and process filters.
4. Share or schedule the dashboard for your team.
