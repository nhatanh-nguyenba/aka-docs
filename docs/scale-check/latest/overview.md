# Scale Check

## Introduction 
Scale Check is a Core Service Invoice Platform that provides e-invoice processing services through APIs and independent microservices. Instead of using one large all-in-one system, Scale Check offers separate services that businesses can use individually or combine into invoice processing flows based on their needs.

Scale Check helps businesses:
- Automate e-invoice processing.
- Reduce manual work when downloading, looking up, and verifying invoices.
- Standardize invoice data before sending it to ERP or accounting software.
- Integrate easily with existing systems through API and Webhook.

Main feature groups:
- [Supplier Invoice Download](./user-guide/main-features/review-services.md#supplier-invoice-download): Download invoices from supplier websites and monitor the download results.
- [Tax Authority Invoice Download](./user-guide/main-features/review-services.md#tax-authority-invoice-download): Set up tax authority accounts, create automatic download schedules, monitor run history, and view downloaded invoices.
- [Tax Authority Invoice Lookup](./user-guide/main-features/review-services.md#tax-authority-invoice-lookup): Check whether an invoice is valid.
- [Taxpayer Lookup](./user-guide/main-features/review-services.md#taxpayer-lookup): Check the company operating status.
- [Processing Flows](./user-guide/main-features/processing-flows.md): Set up invoice processing flows.
- [Invoice List](./user-guide/main-features/invoice-list.md): View, search, and export consolidated invoice data.
