# Set Up Users and Permissions
Administrators can create new users, limit data access by branch or department, and assign roles based on each user responsibility.

## Add a Branch

![branch-scalecheck](/static/img/branch-scalecheck.png)

1. Go to Branch Management.
2. Select Add Branch.
3. Enter the branch information and click Save.

![create-brand-scalecheck](/static/img/create-brand-scalecheck.png)

## Permissions

![role-permissions-scalecheck](/static/img/role-permissions-scalecheck.png)

1. Open the Roles & Permissions tab.
2. Click Add New if your business needs a custom role.
3. Enter a name and assign the permissions that match the tasks of the user group.
4. Click Save to finish.

![permissions-scalecheck.png](/static/img/permissions-scalecheck.png)

## Create a User

![creat-user-scalecheck](/static/img/creat-user-scalecheck.png)

1. Go to System Settings > Permissions & Users.
2. On the Users tab, click Add New.
3. Enter the user information.
4. If the user is a company administrator, turn on Company Admin.
5. Select a Branch / Department if you want to limit the user data access.
6. Select the appropriate role.
7. Click Save to finish.

![user-name-scalecheck](/static/img/user-name-scalecheck.png)
