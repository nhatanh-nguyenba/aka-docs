# ST-DBP-023 - Empty Workflow

**Rule ID:** ST-DBP-023

**Scope:** Workflow

## Description

The rule detects workflow files that do not contain any execution activities, which may indicate incomplete development or redundant files

## Recommendation

Implement automation logic in the workflow or remove the file if it is no longer needed
