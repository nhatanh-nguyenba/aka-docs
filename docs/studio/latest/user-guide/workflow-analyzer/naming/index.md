# Naming Rules

This section contains rules that check if items in your akaBot Studio projects follow standard naming conventions. Following consistent naming conventions makes it easier to understand the purpose of variables and arguments, and helps maintain your automation projects.

The following naming rules are available in the Workflow Analyzer:
