# ST-NMG-003 - Parent Variable Shadowing

**Rule ID:** ST-NMG-003

**Scope:** Workflow

## Description

Checks if a variable declared in a child scope has the same name as a variable/argument in a parent scope, shadowing it

![st-nmg-003](/static/img/st-nmg-003.png)

## Recommendation

Rename the variable in the child scope to avoid shadowing
