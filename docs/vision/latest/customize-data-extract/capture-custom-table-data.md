# Capture Custom Table Data
A basic element in the extraction schema is the data field. However, akaBot Vision enables the capture of even more complex structures like tables.

## Adding a predefined table field
If you are missing some fields when capturing data from the document, navigate to the Fields to Capture tab in the queue settings. In this tab, you can manage pre-trained data fields and select which of them should be extracted.

![image-20220421003652-1.png](/static/img/image-20220421003652-1.png)

## Adding a new table
In order to create a new table, click [Add Field], select Data Type = “table” and then input the value you want to create a new table.

![image-20220421003652-2.png](/static/img/image-20220421003652-2.png)
