# Export Manually
Staff can get the result of documents in akaBot Vision by exporting them.

After documents are confirmed, a staff can export results. In this article, we will show how to export documents

**Step 1**: Go to "Confirmed" tab

![image-20221021140511-1.png](/static/img/image-20221021140511-1.png)

**Step 2**: Choose documents that need to be exported then click "Export" icon

![image-20221021140817-3.png](/static/img/image-20221021140817-3.png)

**Step 3:** Staff choose file type that staff wants to export then click "OK" button

![image-20221021143844-4.png](/static/img/image-20221021143844-4.png)
