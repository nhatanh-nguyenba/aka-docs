# Report Center
**Report Center** is used to set up which information that users want to be reported via email.

Users can access and set up the Report Center by following these steps:

**Step 1**: Click Tenant Settings at the top menu bar, then choose Report Center on the left side

![1704879934184-829.png](/static/img/1704879934184-829.png)

**Step 2**: Type the email address will receive the report![image-20221115171226-2.png](/static/img/image-20221115171226-2.png)

**Step 3**: Decide how often the report should be sent to email

![image-20221115171340-3.png](/static/img/image-20221115171340-3.png)

**Step 4**: Choose the information will be mentioned in the report

![1704962101986-186.png](/static/img/1704962101986-186.png)

**Step 5**: Click "Save" to complete the settings

![1704962204408-151.png](/static/img/1704962204408-151.png)
