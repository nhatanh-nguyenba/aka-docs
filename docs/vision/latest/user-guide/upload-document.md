# Upload Document
Now, you can start uploading your documents by selecting the “Import” button or by sending your documents to the specified email address that is given (this can also be done via API as well). You can import PDFs or scanned images of your documents.

![1704876509645-404.png](/static/img/1704876509645-404.png)

![image-20220420184331-2.png](/static/img/image-20220420184331-2.png)

Click “OK” and the AI will automatically begin extracting the data fields. The AI extraction typically takes up to 5 seconds for 1 A4 page.

![image-20220420184331-3.png](/static/img/image-20220420184331-3.png)

After processing, the documents can be found in the “To review” tab in case human review is needed.

![image-20220420184331-4.png](/static/img/image-20220420184331-4.png)
