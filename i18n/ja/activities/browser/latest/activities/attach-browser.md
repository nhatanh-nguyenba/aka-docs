# ブラウザを接続する

RCA.Activities.Browser.AttachBrowser

## **説明**

すでに開いているネイティブ ブラウザに接続し、その中で複数のアクションを実行できるコンテナ。

![Browser_AttachBrowser](/static/img/b5a5f0_a15035e-screenshot_2021-05-25_154745.jpg)

（\*必須）

## **アクティビティの本文**

* **Do** - ブラウザ内で実行するアクティビティ。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。
True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False (デフォルト) - プロセスの実行の継続をブロックします。
* **完了時または失敗時にブラウザを終了する (チェックボックス)** - 実行の終了時または失敗時にブラウザを閉じるかどうかを指定します。デフォルト値はチェックなしです。

**入力**

* **ブラウザの種類** - このアクティビティで使用するブラウザの選択。選択肢は 1 つだけです: Chrome

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [4234234] ブラウザを接続
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。このチェックボックスはデフォルトではオフになっています

**出力**

* **出力ブラウザ** - タイプ = 「ブラウザ」のアクティビティの出力変数。出力名に空白を使用しないでください。
