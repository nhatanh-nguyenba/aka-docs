# ブラウザのスコープ

RCA.Activities.Browser.BrowserScrope

## **説明**

ブラウザ スコープ アクティビティは、すでに開いているブラウザをアタッチしてブラウザ内でアクションを実行できるコンテナを作成します。

![Browser_BrowserScope](/static/img/c5655a_c41d394-screenshot_2021-05-25_155316.jpg)  
（\*必須）

## **アクティビティ本体内**

* **Do** - ブラウザ内で実行するアクティビティ。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。
True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False (デフォルト) - プロセスの実行の継続をブロックします。
* **完了時または失敗時にブラウザを終了する (チェックボックス)** - 実行の終了時または失敗時にブラウザを閉じるかどうかを指定します。デフォルト値はチェックされています。

**入力**

* **Browser (String)**\* - ブラウザー タイプの入力。入力は、Open Browser アクティビティの出力から取得できます。

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [234234234] ブラウザのスコープ
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。デフォルト値はチェックなしです。

**出力**

* **出力ブラウザ (文字列)** - タイプ = 「ブラウザ」のアクティビティの出力。出力名に空白を使用しないでください。
