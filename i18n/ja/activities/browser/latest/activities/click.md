# クリック

RCA.Activities.Browser.Click

## **説明**

Click アクティビティは、クリック アクションを Web ページに送信するために使用されます。

![Browser_Click](/static/img/d3cc56_5fcb1fe-screenshot_2021-05-25_155024.jpg)

（\*必須）

## **アクティビティの本文**

* **ターゲット要素を選択**\* - 存在を確認する要素を選択します。このアクティビティは、その要素の場所を指定する文字列変数 (セレクター) を生成します。
* **クリック タイプ** - クリック イベントをシミュレートするときに使用されるマウス クリックのタイプを指定します。デフォルトでは、シングルクリックが選択されています。
**・CLICK\_SINGLE** - 1 回のクリック。
**・CLICK\_DOUBLE** - 2 回続けてクリックします。
**・CLICK\_UP** – マウス ボタンを放します
**・CLICK\_DOWN** - マウスボタンを押したままにします。
* **マウス ボタン** - クリック操作に使用されるマウス ボタン (左、右)。デフォルトでは、マウスの左ボタンが選択されています。
**・MOUSE\_LEFT** - マウスの左ボタンを使用してクリックするアクションを設定します。
**・MOUSE\_RIGHT** – マウスの右ボタンを使用してクリックするアクションを設定します。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。
True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False (デフォルト) - プロセスの実行の継続をブロックします。
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
例: 30000

**入力**

* **クリック タイプ (ドロップダウン リスト)** - クリック イベントをシミュレートするときに使用されるマウス クリックのタイプを指定します。デフォルトでは、シングルクリックが選択されています。
**・CLICK\_SINGLE** - 1 回のクリック。
**・CLICK\_DOUBLE** - 2 回続けてクリックします。
**・CLICK\_UP** – マウス ボタンを放します
**・CLICK\_DOWN** - マウスボタンを押したままにします。
* **マウス ボタン (ドロップダウン リスト)** - クリック操作に使用されるマウス ボタン (左、右)。デフォルトでは、マウスの左ボタンが選択されています。
**・MOUSE\_LEFT** - マウスの左ボタンを使用してクリックするアクションを設定します。
**・MOUSE\_RIGHT** – マウスの右ボタンを使用してクリックするアクションを設定します。
* **セレクター (文字列)\*** - アクティビティの実行時に特定の UI 要素を検索するために使用されるテキスト プロパティ。これは実際には、探している GUI 要素とその親の一部の属性を指定する XML フラグメントです。
* **表示を待機 (チェックボックス)** - アクティビティを実行する前にターゲットが表示されるまでオートメーションを待機させる場合は、このボックスをオンにします。これはデフォルトでチェックされています

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [5656456] をクリック
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。デフォルトはチェックなしです。

**オプション**

* **InputMethod (ドロップダウン リスト) -** システムがターゲット要素上でクリック アクションを実行する方法を定義します。
  + デフォルト: 環境に基づいて最適な方法を自動的に選択します
  + ハードウェア: 実際のマウス クリックをシミュレートします (表示される UI とアクティブな画面が必要です)。
  + ソフトウェア: システム レベルでバックグラウンド クリックを実行します (物理的なマウスの移動は必要ありません)。
* **キー修飾子 (KeyModifiers)** - キー修飾子を追加できます。次のオプションが使用可能です: Alt、Ctrl、Shift、Win

**スクロール -** アクションを実行する前にシステムが画面をスクロールしてターゲット要素を表示する方法を制御します*。*

* **水平スクロール:** 数値（ピクセル）を入力すると水平にスクロールします。
  + 正の値 → 右にスクロール
  + 負の値 → 左にスクロール
  + 例: 200
* **要素をスクロールして表示 (チェックボックス):** システムは、ターゲット要素が表示されるまで自動的にスクロールします。デフォルト設定はチェックが入っていません。
* **垂直スクロール:** 数値 (ピクセル) を入力して垂直スクロールします。
  + 正の値 → 下にスクロール
  + 負の値 → 上にスクロール
  + 例: 300
