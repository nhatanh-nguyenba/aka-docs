# データの抽出

RCA.Activities.Browser.ExtractData

## **説明**

データの抽出アクティビティを使用すると、指定した Web ページからデータを取得できます。

![Browser_ExtractData](/static/img/173e9e_f342d2e-screenshot_2021-05-25_160007.jpg)

（\*必須）

## **アクティビティ本体内**

* **Config Json (String)**\* - Json ファイルを使用すると、指定された Web ページからデータを抽出できます。テキストは引用符で囲む必要があります。
例: 「project.json」
* **テキスト html (文字列)** - テキスト文字列は、指定された Web ページから抽出する情報を指定します。テキストは引用符で囲む必要があります。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。
True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False (デフォルト) - プロセスの実行の継続をブロックします。

**入力**

* **Config Json (String)** - Json ファイルを使用すると、指定された Web ページからデータを抽出できます。テキストは引用符で囲む必要があります。
* **HTML (文字列) の抽出** - 指定された Web ページから抽出する HTML。テキストは引用符で囲む必要があります。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [342342314] データの抽出

**出力**

* **出力 Json (文字列)\***- タイプ = '文字列' の出力データ。出力名には空白を使用できません。
