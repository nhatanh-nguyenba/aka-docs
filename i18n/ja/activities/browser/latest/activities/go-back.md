# 戻る

RCA.Activities.Browser.GoBack

## **説明**

Go Back アクティビティは、現在のブラウザーの履歴リスト内の URL を 1 つ前に戻します。

![Browser_GoBack](/static/img/dc4c6f_bdce0db-screenshot_2021-05-25_154910.jpg)

（\*必須）

## **アクティビティの本文**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。
True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False (デフォルト) - プロセスの実行の継続をブロックします。

**その他**

* **表示 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [2446667] 戻る
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。デフォルトではチェックが入っていません。
