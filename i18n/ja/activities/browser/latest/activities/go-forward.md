# 前進する

RCA.Activities.Browser.GoForward

## **説明**

Go Forward アクティビティは、現在のブラウザーの履歴リスト内の URL を 1 つ進めます。

![Browser_GoForward](/static/img/7aa8f2_9db89ed-screenshot_2021-05-25_154922.jpg)

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。
True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False (デフォルト) - プロセスの実行の継続をブロックします。

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。例: [3131265578] 先に進む
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。デフォルトではチェックが入っていません。
