# アラートを処理する

RCA.Activities.Browser.HandleAlert

## **説明**

アラートの処理アクティビティを使用すると、さまざまな方法でアラート ポップアップ ボックスを処理できます。

![image-20220505142619-1.png](/static/img/7116a9_image-20220505142619-1.png)

（\*必須）

## **アクティビティ本体内**

* **ハンドル オプション (ドロップダウン リスト)**
・ACCEPT - アラートを受け入れて [OK] ボタンをクリックします。
・DISMISS - ポップアップウィンドウが表示されたらすぐに「キャンセル」または「終了」ボタンをクリックします。
・GETTEXT - アラートボックスからテキストを取得します。
・SENDKEYS – アラートボックスにテキスト文字列を送信します。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。
True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False (デフォルト) - プロセスの実行の継続をブロックします。
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
例: 20000

**入力**

* **ハンドル オプション (ドロップダウン リスト)**
・ACCEPT - アラートを受け入れて [OK] ボタンをクリックします。
・DISMISS - [キャンセル]ボタンまたは[終了]ボタンをクリックしてアラートを閉じます。
・GETTEXT - アラートボックスからテキストを取得します。
・SENDKEYS – アラートボックスにテキスト文字列を送信します。
* **KeysToSend (String)\***- アラート ボックスに渡されるテキストの文字列 (SENDKEYS ハンドル オプションを使用)。
例:`{ENTER}`、`{TAB}`..

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: アラートを処理します。

**出力**

* **アラート テキスト (文字列)**\* - アラート ボックスから返されるテキスト。
