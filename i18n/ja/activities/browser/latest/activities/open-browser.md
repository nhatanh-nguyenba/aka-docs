# ブラウザを開く

RCA.Activities.Browser.OpenBrowser

## **説明**

「ブラウザを開く」アクティビティを使用すると、ブラウザを起動し、ブラウザ内で複数のアクティビティを実行することで、指定された URL に移動できます。

![1774410564895-514.png](/static/img/a4dc32_1774410564895-514.png)

（\*必須）

## **アクティビティの本文**

* **挿入される URL (文字列)**\* - ブラウザーで開きたい URL。 URL は引用符で囲む必要があります。
例: 「[https://akabot.com/](https://akabot.com/%E2%80%9D)」
* **Do** - ブラウザ内で実行するアクティビティ。

## **プロパティ**

**ブラウザの実行可能パス**

* Edge 実行可能パス: Visual Basic 式を入力します。例: 「C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe」
* Firefox 実行可能パス: Visual Basic 式を入力します。例: 「C:\Program Files\Mozilla Firefox\firefox.exe」

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。
True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False (デフォルト) - プロセスの実行の継続をブロックします。
* **完了時または失敗時にブラウザを終了する (チェックボックス)**- 実行が終了したときまたは失敗したときにブラウザを閉じるかどうかを指定します。デフォルト値は true です。これにより、このアクティビティによって開かれたウィンドウ ブラウザのみが閉じられます。

**入力**

* **ブラウザの種類 (ドロップダウン リスト)** - このアクティビティで使用するブラウザの選択。選択肢は 4 つあります: Chrome、Firefox、IE、Edge
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
例: 30000
* **URL (文字列)**\* – ブラウザを開いたときにこのリンク先 URL を開始します。
例: 「https://akabot.com/」

**プロキシ**

* **HTTP プロキシ (文字列)** - アクセスしようとしているサイトの IP アドレス。これは文字列なので、引用符で囲む必要があります。例えば。 「127.0.0.0」
* **ユーザー名 (文字列)** - プロキシへのアクセスに使用されるユーザー アカウントのユーザー名。これは文字列なので、引用符で囲む必要があります。
例: 「ユーザー名1」
* **パスワード (文字列)** - プロキシへのアクセスに使用されるユーザー アカウントのパスワード。これは文字列なので、引用符で囲む必要があります。
例: 「パスワード@123」

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。

例: [34235252] ブラウザを開く

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。デフォルト値はチェックなしです。

**オプション**

* **引数:**
  + データを渡す必要がない場合 => フィールドを空のままにして、**OK** をクリックします。
  + 入力データを渡す必要がある場合 (方向 = In) => 「引数の作成」をクリック => 「値」列に入力します: 例: 値 = https://akabot.com/
* **デフォルトのダウンロード フォルダー (文字列)** - ファイルを保存するデフォルトのフォルダー
例: 「D:\ユーザー\」
* **エッジ モード (チェックボックス):** チェックボックスが選択されている場合、ブラウザは Internet Explorer (IE) モードで実行され、IE レンダリングを必要とする従来の Web サイトとの互換性が可能になります。デフォルト値はチェックなしです。
* **セッションをクリーンにする(チェックボックス)** - ブラウザを開く前にセッションをクリーンにするかどうかを指定します。デフォルト値はチェックなしです。
* **コマンド プロンプト ウィンドウを非表示にする (チェックボックス)** - Web ドライバーのコマンド プロンプト ウィンドウを非表示にします。デフォルト値はチェックされています。
* **最大化 (チェックボックス)** - ブラウザのサイズを最大化します。デフォルト値はチェックなしです。
* **ダウンロード プロンプトを表示 (チェックボックス)** - Web ドライバーのコマンド プロンプト ウィンドウを表示します。デフォルト値はチェックされています。
* **InitialBrowserUrl (チェックボックス) を使用する** - 特定の URL でブラウザを起動するために使用されます。デフォルト値はチェックなしです。
* **ユーザー プロファイル設定 -** ユーザーが引数 (名前、方向、タイプ、値) を使用してブラウザ プロファイルを構成できるようにします。 「引数の作成」をクリック => 「名前」と「値」を含む 2 つの列を入力します。例: "名前" = "プロファイル.名前"、"値" = "プロファイル 1"

**出力**

* **出力ブラウザ (ブラウザ)** - タイプ = 「ブラウザ」のアクティビティの出力。出力名に空白を使用しないでください。ブラウザセッションに関するすべての情報を保存します。ブラウザ変数のみがサポートされます。
