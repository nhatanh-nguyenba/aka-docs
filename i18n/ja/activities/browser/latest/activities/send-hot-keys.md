# ホットキーを送信する

RCA.Activities.Browser.SendHotKeys

## **説明**

Send Hot Keys アクティビティを使用すると、キーボード ショートカットをブラウザ要素に送信できます。

![Browser_SendHotKeys](/static/img/5d7892_e9e1b2c-browser_sendhotkey.png)

（\*必須）

## **アクティビティ本体内**

* **ターゲット要素を選択** - キーボード ショートカットを送信するブラウザ上の要素を選択します。このアクティビティは、その要素の場所を指定する文字列変数 (セレクター) を生成します。
* **Alt (チェックボックス)** - 修飾キー。
* **Ctrl (チェックボックス)** - 修飾キー。
* **Shift (チェックボックス)** - 修飾キー。
* **特殊キー (ドロップダウン リスト)\***- 送信するキー。使用可能なすべての特殊キーのリストがドロップダウン リストに表示されます。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。
True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False (デフォルト) - プロセスの実行の継続をブロックします。
* **TimeOut MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
例: 30000

**入力**

* **Alt (チェックボックス)** - 修飾キー。
* **Ctrl (チェックボックス)** - 修飾キー。
* **セレクター (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
* **Shift (チェックボックス)** - 修飾キー。
* **特殊キー (ドロップダウン リスト)**\* - 送信するキー。使用可能なすべての特殊キーのリストがドロップダウン リストに表示されます。
* **待機表示 (チェックボックス)** - チェックボックスには、True または False の 2 つの値があります。
チェック (デフォルト): オートメーションは、ターゲットが表示されるまで待機してからアクティビティを実行します。
チェックを外すと、オートメーションはアクティビティを実行する前にターゲットが表示されるまで待機しません。

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [234235245] ホットキーを送信
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。

**スクロール**: アクションを実行する前にシステムが画面をスクロールしてターゲット要素を表示する方法を制御します*。*

* **水平スクロール:** 数値 (ピクセル) を入力して水平にスクロールします。
  + 正の値 → 右にスクロール
  + 負の値 → 左にスクロール
  + 例: 200
* **要素をスクロールして表示 (チェックボックス):** システムは、ターゲット要素が表示されるまで自動的にスクロールします。デフォルト設定はチェックが入っていません。
* **垂直スクロール:** 数値 (ピクセル) を入力して垂直スクロールします。
  + 正の値 → 下にスクロール
  + 負の値 → 上にスクロール
  + 例: 300
