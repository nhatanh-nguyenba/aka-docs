# Web属性の設定

RCA.Activities.Browser.SetWebAttribute

## **説明**

「Web 属性の設定」は、Web 要素の属性の値を設定します。

![image-20220505134907-1.png](/static/img/b0ac8f_image-20220505134907-1.png)

（\*必須）

## **アクティビティの本文**

* **ターゲット要素を選択**\* - ブラウザ上のフィールドを選択して項目を選択します。このアクティビティは、そのフィールドの場所を指定する文字列変数 (セレクター) を生成します。
* **属性名 (文字列)**\* - 選択される属性の式。これは文字列なので、引用符で囲む必要があります。
* **属性値 (文字列)**\* - 指定された属性の値。このプロパティは文字列変数をサポートしているため、引用符で囲む必要があります。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
例: 30000

**入力**

* **属性名 (文字列)\***- 選択される属性の式。これは文字列なので、引用符で囲む必要があります。
* **属性値 (文字列)**\* - 指定された属性の値。このプロパティは文字列変数をサポートしているため、引用符で囲む必要があります。
* **セレクター (文字列)\*** - アクティビティの実行時に特定の UI 要素を検索するために使用されるテキスト プロパティ。これは実際には、探している GUI 要素とその親の一部の属性を指定する XML フラグメントです。
* **表示を待機 (チェックボックス)** - アクティビティを実行する前にターゲットが表示されるまでオートメーションを待機させる場合は、このボックスをオンにします。これはデフォルトでチェックされています

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [366546456] Web 属性の設定
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。デフォルトではチェックが入っていません。

**スクロール**: アクションを実行する前にシステムが画面をスクロールしてターゲット要素を表示する方法を制御します*。*

* **水平スクロール:** 数値（ピクセル）を入力して水平にスクロールします。
  + 正の値 → 右にスクロール
  + 負の値 → 左にスクロール
  + 例: 200
* **要素をスクロールして表示 (チェックボックス):** システムは、ターゲット要素が表示されるまで自動的にスクロールします。デフォルト設定はチェックが入っていません。
* **垂直スクロール:** 数値 (ピクセル) を入力して垂直スクロールします。
  + 正の値 → 下にスクロール
  + 負の値 → 上にスクロール
  + 例: 300
