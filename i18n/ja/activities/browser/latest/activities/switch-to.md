# 切り替え先

RCA.Activities.Browser.SwitchTo

## **説明**

Switch To アクティビティを使用すると、ブラウザー内でさまざまな UI 要素を切り替えることができます。

![image-20220505133627-1.png](/static/img/7f3a6b_image-20220505133627-1.png)

（\*必須）

## **アクティビティの本文**

* **切り替え先 (ドロップダウン リスト)** - 切り替えるタイプを選択します
・FRAME - iframe。
・DEFAULT\_CONTENT - メインの Web ページ。
・PARENT\_FRAME - ネストされた iframe がある場合に使用されます。
・WINDOW - タブを切り替えます。
・ALERT - 指定したメッセージを含むアラートボックス。
・ACTIVE\_ELEMENT - 現在フォーカスされている要素。
* **値 (文字列)**\* - 指定された要素の値。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。
True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False (デフォルト) - プロセスの実行の継続をブロックします。

**入力**

* **タイプに切り替える (ドロップダウン リスト)** - 切り替える HTML 要素のタイプを選択します。
・FRAME - iframe。
・DEFAULT\_CONTENT - メインの Web ページ。
・PARENT\_FRAME - ネストされた iframe がある場合に使用されます。
・WINDOW - タブを切り替えます。
・ALERT - 指定したメッセージを含むアラートボックス。
・ACTIVE\_ELEMENT - 現在フォーカスされている要素。
* **Switch To Value (String)**\* - 指定された要素の値。

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [345365] 切り替え先
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。デフォルトではチェックが入っていません。

**オプション**

* **待機タイムアウト MS:** 切り替えを実行する前に、ターゲットのタブ/ウィンドウが使用可能になるまでシステムが待機する最大時間 (ミリ秒) を指定します。数値を入力します。例: 30000
