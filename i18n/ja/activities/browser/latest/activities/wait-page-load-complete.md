# ページの読み込みが完了するまで待ちます

RCA.Activities.Browser.WaitPageLoadComplete

## **説明**

「ページの読み込み完了を待つ」を使用すると、Web ページが完全に読み込まれるまで待つことができます。

![image-20220505140621-1.png](/static/img/9d1869_image-20220505140621-1.png)

（\*必須）

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。
True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False (デフォルト) - プロセスの実行の継続をブロックします。

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [34116574] ページの読み込み完了を待ちます
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。デフォルトではチェックが入っていません。

**出力**

* **待機成功 (ブール値)** - ブール変数には 2 つの可能な値があります: True または False
**・True** - ページは完全に読み込まれました。
**・False** - ページは完全には読み込まれませんでした。
