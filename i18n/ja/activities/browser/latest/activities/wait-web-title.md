# 待機ページのタイトル

RCA.Activities.Browser.WaitWebTitle

## **説明**

待機ページ タイトルは、ページ タイトルが事前定義された要件に従っているかどうかを検証します。
値。

![image-20220505141738-1.png](/static/img/73bc4f_image-20220505141738-1.png)

（\*必須）

## **アクティビティの本文**

* **比較演算子 (ドロップダウン リスト)** - 数学的な計算や基本的な操作を簡単に実行できます
**・EQUALS** – ページのタイトルは、事前定義された値とまったく同じです。
**・CONTAINS** – ページ タイトルには事前定義された値が含まれます。
**・MATCHES** – ページ タイトルには、事前定義された値と同じ正規表現が含まれています
* **値 (文字列)**\* - ページ タイトルを確認するための事前定義された値。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。
True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False (デフォルト) - プロセスの実行の継続をブロックします。

**入力**

* **比較演算子 (ドロップダウン リスト)** - 数学的な計算や基本的な操作を簡単に実行できます
**・EQUALS** – ページのタイトルは、事前定義された値とまったく同じです。
**・CONTAINS** – ページ タイトルには事前定義された値が含まれます。
**・MATCHES** – ページ タイトルには、事前定義された値と同じ正規表現が含まれています
* **タイトル値 (文字列)** - ページ タイトルを確認するための事前定義された値。

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [132567] 待機ページのタイトル
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。デフォルトではチェックが入っていません。

**オプション**

* **待機タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
例: 20000

**出力**

* **結果 (ブール値)** - ブール変数には 2 つの可能な値があります: True または False
**・True** - ページ タイトルは要件を満たしています。
**・False** - ページは要件を満たしていません。
