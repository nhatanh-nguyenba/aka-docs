# 待機ページの URL

RCA.Activities.Browser.WaitPageUrl

## **説明**

待機ページ URL は、ページ URL が事前定義された値に従って要件の形式であるかどうかを検証します。

![image-20220505142016-2.png](/static/img/613d9d_image-20220505142016-2.png)

（\*必須）

## **アクティビティの本文**

* **比較演算子 (ドロップダウン リスト)** - 数学的な計算や基本的な操作を簡単に実行できます
**・EQUALS** – ページ URL は事前定義された値とまったく同じです。
**・CONTAINS** – ページ URL には事前定義された値が含まれています。
**・MATCHES** – ページ URL に事前定義された値と同じ正規表現が含まれています
* **値 (文字列)**\* - ページ URL を確認するための事前定義された値

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。
True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False (デフォルト) - プロセスの実行の継続をブロックします。

**入力**

* **比較演算子 (ドロップダウン リスト)** - 数学的な計算や基本的な操作を簡単に実行できます
**・EQUALS** – ページ URL は事前定義された値とまったく同じです。
**・CONTAINS** – ページ URL には事前定義された値が含まれています。
**・MATCHES** – ページ URL に事前定義された値と同じ正規表現が含まれています
* **URL 値 (文字列)**\* - ページ URL を確認するための事前定義された値

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [2356558] 待機ページ URL
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。デフォルトではチェックが入っていません。

**オプション**

* **待機タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
例: 20000

**出力**

* **結果 (ブール値)** - ブール変数には、True または False の 2 つの値が考えられます。
**・True** - ページ URL は要件を満たしています
**・False**- ページ URL が要件を満たしていません。
