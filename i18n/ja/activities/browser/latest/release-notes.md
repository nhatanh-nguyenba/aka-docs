# リリースノート

### **ブラウザ v2.3.0.5**

リリースノート:

1. 追加: クリックアクティビティの入力メソッド。
2. 追加: アイテムの選択アクティビティ (テキストおよび値による) へのワイルドカードのサポート。注: アスタリスク (\*) のみがサポートされています。
3. 追加: 「ブラウザを開く」>「引数」プロパティに「enableDriverLog」を挿入して、Selenium ドライバーのログ記録を有効にします。

### **ブラウザ v2.3.0.4**

リリースノート:

1. 追加: Type Into アクティビティの入力メソッド。

### **ブラウザ v2.3.0.2**

リリースノート:
1. 更新: プロパティ WaitVisible のデフォルト値を true に設定します
2. 追加: Click アクティビティの Wait Clickable プロパティ

### **ブラウザ v2.3.0.1**

リリースノート:
1. Selenium WebDriver、IEDriverServer のアップグレード
- バージョン4.2.0から4.5.0へ

2. ブラウザを開く:
- 追加: タイムアウトに達するまで再試行
- 追加: ユーザーが初期化段階で WebDriver を構成するための引数を入力できるようにします (IE モードのみ)
PageLoadStrategy = Normal // 値: Default、Normal、Eager、None
EnableNativeEvents = true // 値: true、false
RequireWindowFocus = false // 値: true、false

3. 要素が存在します:
- 更新: 操作中に例外をスローしないでください
- タイムアウト時にブール値を返す

4. 待機アクティビティ
- 適用対象: 待機要素の存在、消滅、タイトル、ページの読み込み
- 更新: タイムアウト時に条件が一致しない場合にのみ例外をスローする

5. 切り替え先
- 更新: タイムアウトになるまで操作を再試行します

6. ホットキーを入力、クリック、送信
- 追加: [デフォルト] > [シミュレート] > [ハードウェア] からの自動フォールバック メカニズム (IE モードのみ)

### **ブラウザ v2.2.0.0**

リリースノート

- 「テキスト」フィールドにデータがない場合にデザイナーに赤いアイコンが表示されない、入力アクティビティの問題を修正しました。
- WaitVisible がチェックされていない場合に表示されないテキストボックスにデータを入力できないという Type Into アクティビティの問題を修正しました。
- 複数のキーの組み合わせを選択するホットキーの送信アクティビティが機能しない問題を修正しました。
- Windows の特殊キーが機能しない、ホット キーの送信アクティビティの問題を修正しました。
- スクリプト ファイルにスペースのみが含まれている場合に誤ったメッセージが表示される JS の挿入アクティビティの問題を修正しました。
- 「ブラウザを終了」アクティビティで IE ブラウザを終了できない問題を修正しました。
- データの抽出、構造データの抽出アクティビティで、エラーで続行 = True の場合に例外エラーが表示される問題を修正しました。

### **アクティビティをインストールするにはどうすればよいですか?**

1. パッケージを手動でダウンロードする

- [here](https://ws3.akabot.com/s/dYtYRXcrKoCbB5u)をクリックしてアクティビティ ファイルをダウンロードします。

- \*.nupkg ファイルをフォルダー *C:\ProgramData\akaBot\Packages\* に配置します。

- Studio > Package Manager で、リストからこのアクティビティを検索してインストールします。

2. Studio パッケージ マネージャーを使用する

- [スタジオ] > [パッケージ マネージャー] > [設定] > [ユーザー パッケージ ソース] で、次のリポジトリを追加します: https://gitlab.com/api/v4/projects/75840319/packages/nuget/index.json

- リストからこのアクティビティを検索してインストールします。
