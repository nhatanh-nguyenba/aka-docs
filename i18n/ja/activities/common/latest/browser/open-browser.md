# ブラウザを開く

RCA.Activities.Common.OpenBrowser

## **説明**

「ブラウザを開く」アクティビティを使用すると、ブラウザを起動し、ブラウザ内で複数のアクティビティを実行することで、指定された URL に移動できます。

![image-20220505134910-1.png](/static/img/044606_image-20220505134910-1.png)

（\*必須）

## **アクティビティ本体内**

* **挿入される URL (文字列)**\* - ブラウザーで開きたい URL。 URL は引用符で囲む必要があります。
例： 「[https://akabot.com/”](https://akabot.com/%E2%80%9D)」
* **Do** - ブラウザ内で実行するアクティビティ。

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - このプロパティは、エラーが発生した場合にオートメーションを継続するタイミングを指定します。可能な値は True または False の 2 つだけです。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。
* **完了または障害が発生したときにブラウザを終了する (ブール値)** - 完了または障害が発生した後にブラウザを自動的に終了します。

**入力**

* **ブラウザの種類 (ドロップダウン リスト)** - このアクティビティで使用するブラウザの選択。 Chrome、Firefox、Edge の 3 つの選択肢があります。
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの実行を待機する時間 (ミリ秒単位)。デフォルトの時間は 30000 ミリ秒 (30 秒) です。
例: 30000
* **URL (文字列)**\* – ブラウザを開いたときにこのリンク先 URL を開始します。
例： 「[https://akabot.com/”](https://akabot.com/%E2%80%9D)」

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。デフォルトはチェックなしです。

**オプション**

* **非表示 (チェックボックス)** - 非表示のブラウザを開きます。
* **最大化 (チェックボックス)** - ブラウザ ウィンドウを最大化します。
* **プライベート モード (チェックボックス)** - プライベート/igcognito セッションを開きます。
* **起動引数 (文字列)** - 起動時にブラウザに直接渡される高度な起動引数
* **ユーザー データ フォルダー モード (UserDataFolderMode)** - 設定する UserDataFolderMode。特定のユーザー データ フォルダーでブラウザを起動するために使用されます。
* **ユーザー データ フォルダー パス (文字列)** - ブラウザーが使用するユーザー データ フォルダー。設定されていない場合、デフォルトは %LocalAppData%\akaBot\BrowserProfiles\BrowserType になります。

**出力**

* **出力ブラウザ (NativeBrowser)** - type = 'Browser' のアクティビティの出力変数。
