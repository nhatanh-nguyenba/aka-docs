# Web属性の設定

RCA.Activities.Common.SetWebAttribute

## **説明**

「Web 属性の設定」は、Web 要素の属性の値を設定します。

![image-20220505135026-1.png](/static/img/37bb66_image-20220505135026-1.png)

（\*必須）

## **アクティビティ本体内**

* **画面上に表示**\* - ワークフローの実行中に画面上の UI 要素または領域を選択できるようにします。
* **属性名 (文字列)**\* - 変更する HTML 属性の名前。
* **属性値 (文字列)**\* - 指定された属性に設定される値。このプロパティは文字列変数をサポートしています。

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - このプロパティは、エラーが発生した場合にオートメーションを継続するタイミングを指定します。可能な値は True または False の 2 つだけです。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。
* **Delay After (Int32)** - アクティビティ実行後の遅延時間 (ミリ秒単位)。デフォルトの時間は 300 ミリ秒です。
例: 300
* **Delay Before (Int32)** - アクティビティが操作の実行を開始するまでの遅延時間 (ミリ秒単位)。デフォルトの時間は 200 ミリ秒です。
例: 200

**入力**

* **属性 (文字列)**\* - 変更する HTML 属性の名前。
* **ターゲット (ドロップダウン リスト)** - アクティビティが動作する UI 要素を識別します。
**要素 (UIElement)** - 別のアクティビティから返された UiElement 変数を使用します。このプロパティは、Selector プロパティと一緒に使用することはできません。
**セレクター (文字列)** - アクティビティの実行時に特定の UI 要素を検索するために使用されるテキスト プロパティ。これは実際には、探している GUI 要素とその親の一部の属性を指定する XML フラグメントです。
**タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの実行を待機する時間 (ミリ秒単位) を指定します。デフォルト値は 30000 ミリ秒 (30 秒) です。
**準備完了を待つ (ドロップダウン リスト)** - アクションを実行する前に、ターゲットの準備が整うまで待ちます。次のオプションが使用可能です。 なし - ターゲットの準備が完了するまで待機しません。インタラクティブ - アプリの一部のみがロードされるまで待機します。完了 - アプリ全体がロードされるまで待機します。
* **値 (文字列)**\* - 指定された属性の値。このプロパティは文字列変数をサポートしているため、引用符で囲む必要があります。

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。デフォルトはチェックなしです。
