# スクリーンショットを撮る

RCA.Activities.Common.TakeScreenshot

## **説明**

スクリーンショットの取得を使用すると、ブラウザまたはウィンドウ アプリケーション内の UI 要素のスクリーンショットを取得できます。

![image-20220505133457-1.png](/static/img/08d8cd_image-20220505133457-1.png)

（\*必須）

## **アクティビティ本体内**

* **画面上に表示** - 要素を選択してその存在を確認します。このアクティビティは、その要素の場所を指定する文字列変数 (セレクター) を生成します。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの可能な値があります。
  + **True**: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
  + **False**: プロセスの実行の継続をブロックします。

**入力**

* **ターゲット (折りたたみ可能なリスト):**
  + **要素 (UIElement)** - 別のアクティビティから返された UiEIement 変数を使用します。このプロパティは、Selector プロパティと一緒に使用することはできません。
  + **セレクター (文字列)** - アクティビティの実行時に特定の UI 要素を検索するために使用されるテキスト プロパティ。これは実際には、探している GUI 要素とその親の一部の属性を指定する XML フラグメントです。
  + **TimeoutMS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。例: 30000
  + **準備完了を待つ (ドロップダウン リスト)** - アクションを実行する前に、ターゲットの準備が整うまで待ちます。次のオプションが利用可能です。
    - **なし** : ターゲットの準備が完了するまで待機しません。
    - **インタラクティブ** : アプリの一部だけが読み込まれるまで待ちます。
    - **完了**: アプリ全体がロードされるまで待機します。

**その他**

* **公開 (チェックボックス)** - アクティビティを公開するかどうかをチェックします。このプロパティを使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [3424325] スクリーンショットを撮る

**オプション**

* **Wait before (Int32)** - アクティビティが操作の実行を開始するまでの遅延時間 (ミリ秒単位)。デフォルトの時間は 200 ミリ秒です。

**出力**

* **出力スクリーンショット (画像)** - タイプ = 「画像」の出力スクリーンショット。
