# 入力してください

RCA.Activities.Common.TypeInto

## **説明**

Type Into アクティビティを使用すると、Web ページまたはウィンドウ アプリケーション内の任意のテキスト フィールドにテキストを入力できます。

![image-20220505133529-1.png](/static/img/fcd9f6_image-20220505133529-1.png)

（\*必須）

## **アクティビティ本体内**

* **画面に表示** - 要素を選択してその存在を確認します。このアクティビティは、その要素の場所を指定する文字列変数 (セレクター) を生成します。
* **Text (String)\***- 入力するテキスト。このプロパティは文字列変数をサポートしているため、引用符で囲む必要があります。
例: 「私の名前は」

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの可能な値があります。
**- True** : アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
**- False**: プロセスの実行の継続をブロックします。
* **Delay After (Int32)** - アクティビティ実行後の遅延時間 (ミリ秒単位)。デフォルトの時間は 300 ミリ秒です。
* **Delay Before (Int32)** - アクティビティが操作の実行を開始するまでの遅延時間 (ミリ秒単位)。デフォルトの時間は 200 ミリ秒です。

**入力**

* **ターゲット (折りたたみ可能なリスト)**
**・ 要素 (UIElement)** - 別のアクティビティから返された UiEIement 変数を使用します。このプロパティは、Selector プロパティと一緒に使用することはできません。
**・ セレクター (文字列)** - アクティビティの実行時に特定の UI 要素を検索するために使用されるテキスト プロパティ。これは実際には、探している GUI 要素とその親の一部の属性を指定する XML フラグメントです。
**・ TimeoutMS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
例: 30000
**・ Wait For Ready (ドロップダウン リスト)** - アクションを実行する前に、ターゲットの準備が整うまで待ちます。次のオプションが利用可能です。
**- なし** : ターゲットの準備が完了するまで待機しません。
**- インタラクティブ** : アプリの一部だけが読み込まれるまで待ちます。
**- 完了**: アプリ全体がロードされるまで待機します。
* **Text (String)\***- 入力するテキスト。このプロパティは文字列変数をサポートしているため、引用符で囲む必要があります。
例: 「私の名前は」

**その他**

* **公開 (チェックボックス)** - アクティビティを公開するかどうかをチェックします。このプロパティを使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [3424325] と入力します

**オプション**

* **アクティブ (ブール値)** - コントロールをアクティブにします。
* **無効にした後 (ブール値)** - 選択すると、指定された UI 要素が無効であっても、シミュレートされたタイプが実行されます。
* **入力前にクリック (ブール値)** - 選択すると、テキスト フィールドをクリックしてテキストが書き込まれます。
* **キー間の遅延 (Int32)** - キー間の指定された時間。
* **Deselect After (Int32)** - 一定時間 (ミリ秒) 経過後にテキスト フィールドの選択を解除します。
* **空のフィールド (ブール値)** - 選択すると、テキストが書き込まれる前に、テキスト フィールド内の既存のコンテンツがすべて消去されます。
* **入力方法 (ドロップダウン リスト)** - 入力方法 (デフォルト、クリックとウィンドウ メッセージの送信をシミュレート)。
