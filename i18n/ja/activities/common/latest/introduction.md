# はじめに

共通アクティビティ パッケージには、自動化プロジェクトの作成に使用されるすべてのアクティビティが含まれています。これらのアクティビティにより、ロボットは次のことが可能になります。

* ブラウザーの操作とウィンドウの操作を実行します。
* マウスやキーボードのコマンドの実行、テキストの入力や抽出など、人間の対話をシミュレートします。
* ユーザーが別のウェブページにリダイレクトする際の問題のトラブルシューティングに役立つように、ブラウザ上のいくつかの要素が表示されるまで待ちます。
