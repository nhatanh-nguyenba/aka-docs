# ウィンドウを閉じる

RCA.Activities.Common.CloseWindow

## **説明**

Close Window アクティビティを使用すると、画面上のアプリケーション ウィンドウを閉じることができます。

![image-20230417161136-1.png](/static/img/d5509a_image-20230417161136-1.png)

（\*必須）

## **アクティビティ本体内**

* **画面上に表示** - ウィンドウを選択して、どのアプリケーションを閉じることができます。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの可能な値があります。
**- True** : アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
**- False**: プロセスの実行の継続をブロックします。

**入力**

* **セレクター (文字列)** - ウィンドウの位置を識別する文字列。
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
例: 30000
* **ウィンドウを使用:** オープン ウィンドウまたはウィンドウ スコープからの出力変数を使用して、アクティビティのスコープを設定します (From と同様)。
* **準備完了を待つ:** アクションを実行する前に、ターゲットの準備が整うまで待ちます。次のオプションが使用可能です。 なし - ターゲットの準備が完了するまで待機しません。インタラクティブ - アプリの一部のみがロードされるまで待機します。完了 - アプリ全体がロードされるまで待機します。

**その他**

* **公開 (チェックボックス)** - アクティビティを公開するかどうかをチェックします。このプロパティを使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [3424325] ウィンドウを閉じる
