# ウィンドウの移動

RCA.Activities.Common.MoveWindow

## **説明**

ウィンドウの移動アクティビティを使用すると、選択したウィンドウの位置とサイズを変更できます。

![image-20230417163133-1.png](/static/img/image-20230417163133-1.png)

\* は必須フィールドを示します。

## **プロパティ**

**共通**

* **Continue On Error (Boolean)**: True または False の 2 つの値を持つブール変数。
True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False: エラーが発生した場合、プロセスの実行継続を停止します。

**入力**

* **Height (Int32)**\*: ウィンドウの新しい高さ。このプロパティは、正と負の両方の整数をサポートします。
例:`638`
* **Width (Int32)**\*: ウィンドウの新しい幅。このプロパティは、正と負の両方の整数をサポートします。
例:`468`
* **Window (WindowApp)**\*: ターゲット ウィンドウを識別するために使用される WindowApp インスタンス。
* **X (Int32)**\*: ウィンドウの新しい X 位置。このプロパティは、正と負の両方の整数をサポートします。
例:`20`
* **Y (Int32)**\*: ウィンドウの新しい Y 位置。このプロパティは、正と負の両方の整数をサポートします。
例:`20`

**その他**

* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データ セキュリティ要件を考慮してください。
* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`[3424325] Move Window`
