# アプリケーションを開く

RCA.Activities.Common.OpenApplication

## **説明**

「アプリケーションを開く」アクティビティを使用すると、アプリケーションを開いてその中で複数のアクティビティを実行できます。

![image-20230417163220-1.png](/static/img//image-20230417163220-1.png)

\* は必須フィールドを示します。

## **アクティビティの本文内**

* **画面上のウィンドウを指定**: ウィンドウを選択してアプリケーションを自動的に開くことができます。
* **Do**: アプリケーション内で実行するアクティビティ。

## **プロパティ**

**共通**

* **Continue On Error (Boolean)**: True または False の 2 つの値を持つブール変数。
True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False: エラーが発生した場合、プロセスの実行継続を停止します。
* **TimeoutMS (Int32)**: エラーがスローされるまでにアクティビティが完了するまで待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。
デフォルト値:`30000`(ミリ秒)
例:`30000`

**入力**

* **引数 (文字列)**\*: 起動時にアプリケーションに渡されるパラメータ。このプロパティを使用して、アプリケーションで特定のファイルを開くことができます。 **[ファイル名]** または **[引数]** の 2 つのフィールドのうち 1 つだけを指定する必要があります。
例:`C:\Users\currentuser\Documents\Sample.xlsx`
* **ファイル名 (文字列)**\*: アプリケーションの完全なファイル パス。これは引用する必要があります。 **[ファイル名]** または **[引数]** の 2 つのフィールドのうち 1 つだけを指定する必要があります。
例:`"name.docx"`
* **セレクター (文字列)**: ターゲット ウィンドウを識別する文字列。

**オプション**

* **作業ディレクトリ (文字列)**: 作業ディレクトリへのパス。これは引用する必要があります。
例:`"C:\Users\Admin"`

**その他**

* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データ セキュリティ要件を考慮してください。
* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`[3424325] Open Window`

**出力**

* **アプリケーション ウィンドウ (WindowApp)**\*:`WindowApp`タイプのアクティビティの出力。この変数は、「ウィンドウの接続」や「ウィンドウの使用」などの他のアクティビティで使用できます。出力変数名には空白を含めることはできません。
