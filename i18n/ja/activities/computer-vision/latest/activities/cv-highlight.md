# 履歴書のハイライト

RCA.Activities.ComputerVision.CVHighlight

## **説明**

指定された UI 要素を視覚的に強調表示します。要素は、コンピューター ビジョン ニューラル ネットワークを使用して識別されます。

![image-20220505132226-1.png](/static/img/4fc8ed_image-20220505132226-1.png)

（\*必須）

## **アクティビティ本体内**

* **画面上に表示**\* - ハイライトする要素を選択します。このアクティビティは、その要素の場所を指定する文字列変数 (セレクター) を生成します。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には 2 つの値があります: True または False - True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を続行できるようにします。 - False: プロセスの実行の継続をブロックします。
* **Delay After (Int32)** - アクティビティ実行後の遅延時間 (ミリ秒単位)。
例: 300
* **Delay Before (Int32)** - アクティビティが操作の実行を開始するまでの遅延時間 (ミリ秒単位)。
例: 300
* **TimeOut MS (Int32)** - アクティビティの実行を待機する時間 (ミリ秒単位)。デフォルトの時間は 30000 ミリ秒です。
例: 30000

**入力**

* **記述子 (文字列)** - ** ターゲット**と使用される各** アンカー** (存在する場合) の画面上の座標。これらの値は、*CvDescriptor*独自のオブジェクトに保存されます。 **Indicate On Screen** 機能が実行時に使用されると、*CvDescriptor* がこのフィールドに自動的に生成されます。

**その他**

* **パブリック (ブール値)** - アクティビティを公開するかどうかをチェックします。このプロパティを使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [123123] 履歴書のハイライト

**オプション**

* **色 (ドロップダウン リスト)** - ハイライトの色を指定できます。事前定義された色を含むドロップダウン リストが利用可能です。
* **ハイライト時間 (Int32)**- 指定された UI 要素がハイライトされる時間 (ミリ秒単位)。このフィールドのデフォルト値は 1000 です。
例: 1000

**再利用可能な領域**

* **InputRegion (Rectangle)** - Rectangle 変数に格納されている別の CV アクティビティのターゲットを受け取り、それをこのアクティビティのターゲットとして使用します。これにより、同じターゲットをあるアクティビティから別のアクティビティに渡すことができます。同じ UI 要素上で複数のアクションを簡単に実行できるため、設計時に特に役立ちます。このフィールドに変数が設定されている場合、アクティビティの現在のターゲット要素とアンカー要素は無効になります。
* **OutputRegion (Rectangle)** - このアクティビティのターゲットを Rectangle 変数として保存します。これにより、同じターゲットをあるアクティビティから別のアクティビティに渡すことができます。同じ UI 要素上で複数のアクションを簡単に実行できるため、設計時に特に役立ちます。
