# CV 画面の範囲

RCA.Activities.ComputerVision.CVScreenScope

## **説明**

ウィンドウまたは要素上でコンピューター ビジョンを有効にするコンテナー。 akaBot コンピュータ ビジョン ニューラル ネットワークを初期化し、指定されたウィンドウの分析を実行し、後続のすべてのコンピュータ ビジョン アクティビティのスコープを提供します。

![image-20220505132346-1.png](/static/img/a524fe_image-20220505132346-1.png)

（\*必須）

## **アクティビティの本文**

* **画面上に表示**\* - アクティビティの本文にある [画面上に表示] ボタンを使用して、自動化するアプリケーションを CV Screen Scope アクティビティに表示できます。これを実行すると、指定した画面のキャプチャがアクティビティの本文に表示されます。このアクティビティは、そのフィールドの場所を指定する文字列変数 (セレクター) を生成します。
* **Do** - アプリケーション内で実行するアクティビティ。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。

**入力**

* **CV メソッド (ドロップダウン リスト)** - 2 つのオプションがあります: 要素検出、OCR
**・ElementDetection** - 検出範囲に UI 要素 (画像、アイコン、ボタンなど) を表示します。
**・OCR** - 検出範囲にテキストを表示します。
* **ターゲット.クリッピング領域 (ドロップダウン リスト)** - 2 つのオプションがあります: 領域、null
**・領域** - セレクターのクリッピング四角形を、UiElement を基準にして、左、上、右、下の方向にピクセル単位で定義します。正の数値と負の数値の両方をサポートします。スコープを定義するときにマウスをドラッグした後に座標を自動的に生成します。
**・null** - セレクターの四角形をスコープとして使用します。
* **Target.Selector (String)** - アクティビティの実行時に特定の UI 要素を検索するために使用されるテキスト プロパティ。
* **Target.TimeoutMS (Int32)** - アクティビティの実行を待機する時間 (ミリ秒単位)。

**その他**

* **パブリック (ブール値)** - アクティビティを公開するかどうかをチェックします。このプロパティを使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [123123] CV 画面のスコープ

**オプション**

* **API キー (文字列)** - Computer Vision サーバーへの認証に使用される API キー。このフィールドは文字列または文字列変数のみをサポートします。このプロパティは、ワークフロー内のすべての CV スコープ間で同期されます。
* **サーバー URL (文字列)** - 分析を実行するサーバーの URL。このプロパティは、ワークフロー内のすべての CV スコープ間で同期されます。
* **ローカル サーバーを使用する (ブール値)** - CV がローカル サーバーを使用するかどうかを決定します。
