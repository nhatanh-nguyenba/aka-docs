# エージェントアセットの取得

RCA.Core.Activities.GetAgentAsset

## **説明**

このアクティビティでは、指定されたアセット名を使用して特定の akaBot Center アセットを取得できます。

![image-20220506112627-1.png](/static/img/abb503_image-20220506112627-1.png)

\* は必須フィールドを示します。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)**: エラーが発生した場合でもオートメーションを続行するかどうかを指定します。 True または False の値のみが使用可能です。
True: アクティビティ内でエラーが発生した場合でも、残りのプロセスの実行を続行します。
False (デフォルト): エラーが発生したときにプロセスの実行を停止します。
* **タイムアウト (Int32)**: エラーがスローされるまでにアクティビティが完了するまで待機する最大時間 (ミリ秒単位)。タイムアウトに達すると、アクティビティは実行を停止します。デフォルト値は 30000 (ミリ秒) です。
例: 30000

**入力**

* **アセット名 (文字列)**\*: 取得するアセットの名前。
例: 「資産グループ1」

**その他**

* **プライベート (チェックボックス)**: 選択すると、アクティビティ データは記録されません。機密情報を保護するには、このオプションを使用します。
* **表示名 (文字列)**: アクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例: エージェント資産の取得

**出力**

* **アセット値 (OutArgument)**: 変数に格納されている取得されたアセット。
