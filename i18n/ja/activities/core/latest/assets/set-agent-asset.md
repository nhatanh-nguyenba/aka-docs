# エージェント資産の設定

RCA.Core.Activities.SetAgentAsset

## **説明**

このアクティビティでは、指定されたアセット名を使用して、センターですでに利用可能な指定されたアセット値を更新できます。 Set Agent Asset がトリガーされるとき、このアクティビティを実行するには、エージェントが編集権限を持っている必要があることに注意してください。エージェントが実行されるエージェントの認証コンテキストが使用されます。

![image-20220506112725-1.png](/static/img/6097d3_image-20220506112725-1.png)

（\*必須）

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - このプロパティは、エラーが発生した場合にオートメーションを継続するタイミングを指定します。可能な値は True または False の 2 つだけです。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。

**入力**

* **アセット名 (文字列)**\* - 取得するアセットの名前。
例: 「テスト\_整数」
* **資産値 (文字列)**\* - 資産に設定する新しい値。

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: エージェント資産の設定

**出力**

* **Set Success (Boolean)** – このプロパティはアクションの結果を指定します。可能な値は True または False の 2 つだけです。資産値が正常に更新された場合は True を返します。資産値が正常に更新されなかった場合は、False を返します。
