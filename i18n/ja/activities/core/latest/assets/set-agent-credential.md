# エージェント資格情報の設定

RCA.Core.Activities.SetAgentCredential

## **説明**

このアクティビティを使用すると、指定されたアセット名を使用して、センターですでに使用可能な指定された akaBot Center 資格情報の値を更新できます。エージェントには編集権限が必要であることに注意してください。

![image-20220506112758-1.png](/static/img/86f21c_image-20220506112758-1.png)

（\*必須）

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - このプロパティは、エラーが発生した場合にオートメーションを継続するタイミングを指定します。可能な値は True または False の 2 つだけです。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。

**入力**

* **アセット名 (文字列)**\* - 取得するアセットの名前。
例: 「Test\_Integer」。
* **パスワード (文字列)**\* – 言及された認証情報資産の新しいパスワード。このプロパティは文字列値と文字列変数の両方を受け入れます
例: 「Password@123」または strPassword。
* **ユーザー名 (文字列)**\* – 言及された認証情報資産の新しいユーザー名。このプロパティは、文字列値と文字列変数の両方を受け入れます。
例: 「AgentName」または strAgent。

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: エージェント資格情報を設定します。

**出力**

* **Set Success (Boolean)** – このプロパティはアクションの結果を指定します。可能な値は True または False の 2 つだけです。エージェント資格情報が正常に設定された場合は True を返します。エージェント資格情報が正常に設定されていない場合は、False を返します。
