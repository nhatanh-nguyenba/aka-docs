# クリップボードからデータを取得

RCA.Activities.Core.GetFromClipboard

## **説明**

このアクティビティを使用すると、さらに実行するためにクリップボードからデータを取得できます。

![image-20220505092631-1.jpeg](/static/img/a2e9c6_image-20220505092631-1.jpeg)

（\*必須）

## **プロパティ**

**入力**

* **テキスト形式 (ドロップダウン リスト)**\* – クリップボードに取得するデータの形式。サポートされる形式は、Text、UnicodeText、Rtf、Html、CommaSepreatedValue、Xaml です。

**出力**

* **出力テキスト (文字列)** - 変数に保存されたこのアクティビティから取得したテキストの出力。

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [997394815] ファイルの移動
* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
