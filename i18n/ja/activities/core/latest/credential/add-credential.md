# 認証情報の追加

RCA.Activities.Core.AddCredential

## **説明**

このアクティビティを使用すると、指定した資格情報をカスタム永続タイプで Windows 資格情報マネージャーに保存でき、プロセスが成功したかどうかを示すブール値を返します。

![image-20220505151253-1.png](/static/img/651c75_image-20220505151253-1.png)

（\*必須）

## **プロパティ**

**入力**

* **認証情報の種類** - 追加する認証情報の種類。次のオプションが利用可能です。
**・なし**: 資格情報の種類は適用されません。
**・汎用**: 資格情報を保存します。資格情報は、Windows 資格情報マネージャーの [汎用資格情報] セクションに保存されます。
**・DomainPassword**: Microsoft 認証パッケージに固有のパスワード資格情報。Windows 資格情報マネージャーの Windows 資格情報セクションに保存できます。
**・DomainCertificate**: 証明書のみの資格情報を保存します。資格情報マネージャーの「証明書ベースの資格情報」セクションに保存できます。
**・DomainVisiblePassword**: Microsoft 認証パッケージ (Passport) に固有のパスワード資格情報。
* **PersistanceType** - 指定された認証情報の保存に従うルールを定義します。次のオプションが利用可能です。
**・セッション**: 資格情報は、現在のログオン セッションが存続している間のみ Windows 資格情報マネージャーに保存されます。これらは、同じユーザーの他のログオン セッションには表示されません。このユーザーがログオフして再度ログオンすると、それらは存在しません。
**・ローカル コンピュータ**: 資格情報は、この同じコンピュータ上での後続のすべてのログオン セッションに対して保持されます。これらは、同じマシン上の同じユーザーの他のログオン セッションには表示されますが、他のマシン上のこのユーザーのログオン セッションには表示されません。
**・エンタープライズ**: 資格情報は、同じマシン上での後続のすべてのログオン セッションに対して保持されます。これらは、同じマシン上のこの同じユーザーの他のログオン セッション、および他のコンピュータ上のこのユーザーのログオン セッションに表示されます。
* **認証情報名 (文字列)**\* - Windows 認証情報に保存されている認証情報の名前。
* **ユーザー名 (文字列)**\* - 認証情報のユーザー名。
* **パスワード (文字列)**\* - 認証情報のパスワード。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: 認証情報の追加

**出力**

* **結果 (ブール値)** - 指定された資格情報が Windows 資格情報マネージャーに正常に保存されたかどうかを示すブール変数。
