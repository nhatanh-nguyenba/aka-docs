# 資格情報

**認証情報アクティビティ** サブパッケージには、[Windows Credential Manager](https://docs.microsoft.com/en-us/windows/win32/secauthn/credentials-management)と連携するアクティビティが含まれており、特定の Microsoft 認証パッケージの ** 認証情報** の追加、取得、削除が可能になります。

ロボットは、これらのアクティビティを使用して資格情報を取得したり、単に人間のユーザーに資格情報の導入を促すこともできます。

これらのアクティビティはすべて、[**ツールボックス**] パネルの [**コア > 認証情報**] にあります。
