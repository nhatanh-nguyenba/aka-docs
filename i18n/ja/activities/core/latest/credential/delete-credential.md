# 認証情報の削除

RCA.Activities.Core.DeleteCredential

## **説明**

このアクティビティを使用すると、指定したターゲットの資格情報を Windows 資格情報マネージャーから削除でき、プロセスが成功したかどうかを示すブール値を返します。

![image-20220505151543-1.png](/static/img/3994b3_image-20220505151543-1.png)

（\*必須）

## **プロパティ**

**入力**

* **資格情報名 (文字列)**\* - Windows 資格情報マネージャーの一連の資格情報から資格情報を識別します。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: 認証情報の削除

**出力**

* **結果 (ブール値)** - 指定されたターゲットの資格情報が Windows 資格情報マネージャーから正常に削除されたかどうかを示すブール変数。
