# 安全な認証情報を取得する

RCA.Activities.Core.GetSecureCredential

## **説明**

安全な資格情報の取得アクティビティを使用すると、資格情報と永続性の種類に基づいて、Windows 資格情報マネージャーから指定したターゲットから資格情報を取得できます。これは安全な認証情報であるため、常に安全な文字列のパスワードと文字列のユーザー名を取得します。そして、成功/失敗の結果がユーザー名とパスワードとともに保存されます。

![image-20220505152006-1.png](/static/img/561e25_image-20220505152006-1.png)

（\*必須）

## **プロパティ**

**入力**

* **資格情報の種類**
**・なし**: 資格情報の種類は適用されません。
**・汎用**: 資格情報を保存します。資格情報は、Windows 資格情報マネージャーの [汎用資格情報] セクションに保存されます。
**・DomainPassword**: Microsoft 認証パッケージに固有であり、Windows 資格情報マネージャーの Windows 資格情報セクションに保存できます。
**・DomainCertificate**: 証明書のみの資格情報を保存します。資格情報マネージャーの「証明書ベースの資格情報」セクションに保存できます。
**・DomainVisiblePassword**: Microsoft 認証パッケージ (Passport) に固有のパスワード資格情報。
* **資格情報名(文字列)**\* - Windows 資格情報に保存されている資格情報の名前。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: 「認証機能」

**出力**

* **パスワード (文字列)** - 後で使用するために type = 'String' で変数を宣言できる出力パスワード。
* **ユーザー名 (文字列)** - 後で使用するために type = 'String' で変数を宣言できる出力ユーザー名。
* **結果 (ブール値)** - ブール変数としてのアクティビティの結果: True または False。
