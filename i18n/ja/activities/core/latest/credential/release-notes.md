# リリースノート

## **RCA.Activities.Core バージョン 2.2.0.1**

### **バグ修正**

* **資格情報の削除** アクティビティで Windows 資格情報を削除できない問題を修正しました。
* **認証情報の削除** アクティビティで [認証情報の種類] プロパティが欠落している問題を修正しました。
* **資格情報の削除** アクティビティで、入力が最大長を超えた場合にエラー メッセージが表示されない問題を修正しました。
