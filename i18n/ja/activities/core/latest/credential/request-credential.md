# 認証情報の要求

RCA.Activities.Core.RequestCredential

## **説明**

このアクティビティを使用すると、ユーザーにアプリケーション資格情報の追加を求めるカスタム メッセージとタイトルを含むダイアログ ボックスを表示し、後でアプリケーションにログインするために使用できるユーザー名とパスワードを保存できます。

![image-20220505152134-1.png](/static/img/7febcb_image-20220505152134-1.png)

（\*必須）

## **プロパティ**

**入力**

* **メッセージ (文字列)**\* - ダイアログ ボックスに表示するメッセージ。
例: 「プログラムの認証情報が必要です」
* **タイトル (文字列)** - ダイアログ ボックスのタイトル。
例: 「資格情報の要求」

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: 認証情報のリクエスト

**出力**

* **パスワード (文字列)** - ユーザーが文字列変数として追加したパスワード。
* **ユーザー名 (文字列)** - ユーザーによって文字列変数として追加されたユーザー名。
* **結果 (ブール値)** - ユーザーが [OK] または [キャンセル] をクリックしたかどうかを示すブール変数。 [OK] をクリックすると true が返され、[キャンセル] をクリックすると false が返されます。
