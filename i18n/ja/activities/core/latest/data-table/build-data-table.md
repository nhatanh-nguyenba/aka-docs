# ビルドデータテーブル

RCA.Activities.Core.BuildDataTable

## **説明**

このアクティビティを使用すると、異なる設定で新しいデータ テーブルを作成できます。

[![build data table.jpg](/static/img/e93ec6_build-data-table.jpg)](/bin/download/Activities/Data%20Table/Activities/Build%20Data%20Table/WebHome/build%20data%20table.jpg?rev=1.1)

## **アクティビティ本体内**

**データ テーブル** - クリックして列を追加/削除し、列名/列タイプを更新します

![image-20230419142900-1.png](/static/img/bfa8cf_image-20230419142900-1.png)

## **プロパティ**

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
**例**: ビルド データ テーブル

**出力**

* **データ テーブル (DataTable)** - 生成されたテーブルを含むデータ テーブル変数。
