# フィルタデータテーブル

RCA.Activities.Core.FilterDataTable

## **説明**

このアクティビティを使用すると、フィルター ウィザード ウィンドウで条件を指定して DataTable 変数をフィルターできます。アクティビティでは、ウィザードで指定された論理条件に従って、行または列を保持または削除できます。アクティビティの本文にはフィルター ウィザード ボタンが含まれており、いつでもウィザードにアクセスして設定をカスタマイズできます。

[![filter data table.jpg](/static/img/2e3424_filter-data-table.jpg)](/bin/download/Activities/Data%20Table/Activities/Filter%20DataTable/WebHome/filter%20data%20table.jpg?rev=1.1)

（\*必須）

## **アクティビティ本体内**

**フィルター ウィザード** - 行のフィルターを設定し、出力列を設定します

![image-20230419143358-1.png](/static/img/4cae76_image-20230419143358-1.png)

## **プロパティ**

**入力**

* **データ テーブル (DataTable)**\* - フィルターする DataTable 変数。このフィールドは DataTable 変数のみをサポートします。

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: データテーブルをフィルターする

**オプション**

* **行フィルター モード** - ターゲット行を保持するか削除することでテーブルをフィルターするかを指定します。
* **列の選択モード** – ターゲット列を保持するか削除することでテーブルをフィルターするかを指定します。

**出力**

* **データ テーブル (DataTable)** - アクティビティの出力としてフィルタリングされたデータ テーブル。DataTable 変数に保存されます。 [入力] フィールドと同じ変数を使用すると、最初の変数が上書きされますが、新しい変数を追加すると、最初の変数は影響を受けません。このフィールドは DataTable 変数のみをサポートします。
