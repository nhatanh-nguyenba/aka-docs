# データテーブルの生成

RCA.Activities.Core.GenerateDataTable

## **説明**

構造化テキストから DataTable 変数を生成します。このアクティビティでは、テーブルの生成に応じて列と行の区切り文字を選択できます。

[![generate data table.jpg](/static/img/aca851_generate-data-table.jpg)](/bin/download/Activities/Data%20Table/Activities/Generate%20Data%20Table/WebHome/generate%20data%20table.jpg?rev=1.1)

（\*必須）

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - このプロパティは、エラーが発生した場合にオートメーションを継続するタイミングを指定します。可能な値は True または False の 2 つだけです。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。

**固定書式設定オプション**

* **列サイズ (IEnumerable< Int32 >)** - テーブル内に作成される列のサイズを指定します。このフィールドは IEnumerable 変数のみをサポートします。

**書式設定オプション**

* **列区切り文字 (文字列)**\* - 列区切り文字として使用する文字を指定します。このフィールドは文字列変数のみをサポートします。
例: 「|」を使用する場合区切り文字として「|」を入力します
* **CSV 解析 (ブール値)**\* - 入力データを CSV として解析する場合に選択します。このフィールドはブール値 (True、False) のみをサポートします。デフォルト値は False です。
例: 本当
* **改行区切り文字 (文字列)**\* - 改行区切り文字として使用する文字を指定します。このフィールドは文字列変数のみをサポートします。

**入力**

* **入力 (文字列)**\* - テーブルに変換する非構造化データを追加するフィールド。文字列のみがサポートされます。
例: 「ジョン | フランク | リーバイ」
* **位置** - OCR が使用されている場合、[単語情報を取得] チェック ボックスがオンになっており、このフィールドには IEnumerable が含まれます。

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: データテーブルの生成

**出力**

* **出力データ テーブル (DataTable)** - 生成されたテーブルを含むデータ テーブル変数。

**テーブルオプション**

* **タイプの自動検出 (チェックボックス)** - このオプションを選択すると、akaBot は列または行のタイプを自動的に検出します。
* **列ヘッダーを使用する (チェックボックス)** - このオプションを選択すると、最初の識別列が列ヘッダーとして選択されます。
* **行ヘッダーを使用する (チェックボックス)** - このオプションを選択すると、最初の識別行が行ヘッダーとして選択されます。
