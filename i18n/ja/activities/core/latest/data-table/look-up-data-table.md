# データテーブルのルックアップ

RCA.Activities.Core.LookupDataTable

## **説明**

このアクティビティを使用すると、指定された DataTable 内で指定された値を検索し、見つかった RowIndex を返すことができます。このアクティビティは、RowIndex プロパティで指定された行と、Target Column プロパティ カテゴリで指定された列を含むセルで見つかった値を返すのにも役立ちます。

[![look up data table.jpg](/static/img/864a58_look-up-data-table.jpg)](/bin/download/Activities/Data%20Table/Activities/Look%20Up%20Data%20Table/WebHome/look%20up%20data%20table.jpg?rev=1.1)

（\*必須）

## **プロパティ**

**入力**

* **検索値 (文字列)**\* - 指定した DataTable 変数で検索する値。このフィールドは文字列と文字列変数のみをサポートします。
例: 「WordIWantToSearch」
* **データ テーブル (DataTable)**\* - ルックアップを実行する DataTable 変数。このフィールドは DataTable 変数のみをサポートします。

**ルックアップ列**

* **列インデックス (整数)** - 検索する列のインデックス。このフィールドは Int32 変数のみをサポートします。
例: 1
* **列名 (文字列)** - 検索する列の名前。このフィールドは文字列と文字列変数のみをサポートします。
例: 「列の名前」
* **データ列 (DataColumn)** - 検索する列を含む変数。このフィールドは DataColumn 変数のみをサポートします。

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: データテーブルを検索する

**出力**

* **セル値** - オプション。 RowIndex プロパティで指定された行座標と、ターゲット列プロパティ カテゴリで指定された列座標を持つセルで見つかった値。オブジェクト変数のみ。
* **行インデックス** - 見つかったセルの行インデックス。Int32 変数に格納されます。このフィールドは Int32 変数のみをサポートします。

**ターゲット列**

* **列インデックス** - ここで列インデックスを指定すると、この列と RowIndex プロパティの値の間の座標にあるセルの値が返されます。このフィールドは Int32 変数のみをサポートします。
例: 1
* **列名 (文字列)** - ここで列名を指定すると、この列と RowIndex プロパティの値の間の座標にあるセルの値が返されます。このフィールドは文字列と文字列変数のみをサポートします。
例: 「対象となる列の名前」
* **データ列 (DataColumn)** - ここで DataColumn 変数を指定すると、この列と RowIndex プロパティの値の間の座標にあるセルの値が返されます。このフィールドは DataColumn 変数のみをサポートします。
