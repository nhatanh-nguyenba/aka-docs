# ソートデータテーブル

RCA.Activities.Core.SortDataTables

## **説明**

このアクティビティを使用すると、指定した列の値に基づいてデータ テーブルを昇順または降順で並べ替えることができます。

[![sort data table.jpg](/static/img/d4f5f2_sort-data-table.jpg)](/bin/download/Activities/Data%20Table/Activities/Sort%20Data%20Table/WebHome/sort%20data%20table.jpg?rev=1.1)

（\*必須）

## **プロパティ**

**入力**

* **データ テーブル (DataTable)\***- 並べ替えるデータ テーブル変数。このフィールドは DataTable 変数のみをサポートします

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: ソートデータテーブル

**出力**

* **出力データ テーブル (DataTable)** - 並べ替え後の DataTable 変数。 [入力] フィールドにある変数と同じ変数を配置すると、最初の変数が置き換えられますが、新しい変数を追加すると、最初の変数は影響を受けません。このフィールドは DataTable 変数のみをサポートします。

**ソート列**

* **列インデックス (整数)** - 並べ替える列のインデックス。
例: 3
* **列名 (文字列)** - 並べ替え列の名前。
例: 「ここに列名」。
* **データ列 (DataColumn)** - 並べ替える列を含む変数。このフィールドは DataColumn 変数のみをサポートします。このプロパティ フィールドに変数を設定すると、他の 2 つのプロパティが無効になります。
* **順序 (ドロップダウンリスト)** - 2 つのオプション (昇順または降順) で下にスクロールします。
