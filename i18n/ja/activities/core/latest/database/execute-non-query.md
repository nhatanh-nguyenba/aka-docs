# 非クエリの実行

RCA.Activities.Core.ExecuteNonQuery

## **説明**

このアクティビティを使用すると、データベース上でクエリ以外のステートメントを実行できます。 UPDATE、INSERT、および DELETE ステートメントの場合、戻り値はコマンドの影響を受ける行数です。他のすべてのタイプのステートメントの場合、戻り値は -1 です。

![image-20220506105309-1.png](/static/img/fe1566_image-20220506105309-1.png)

（\*必須）

## **アクティビティ本体内**

* **接続の構成** - 接続文字列、データ ソース、データ プロバイダーを設定します。

![image-20230419153435-1.png](/static/img/3e2525_image-20230419153435-1.png)

* **クエリの編集** - 実行される SQL コマンド。このフィールドは、CommandType プロパティの選択に従って入力する必要があります。
例: 「連絡先リストから \* を削除」

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - このプロパティは、エラーが発生した場合にオートメーションを継続するタイミングを指定します。可能な値は True または False の 2 つだけです。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。

例: 本当

* **TimeoutMS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。 **(このプロパティは、akaBot バージョン 2.0.5.3 以降でのみ使用可能です)**

**接続構成**

* **接続文字列 (文字列)**\* - データベース接続を確立するために使用される接続文字列。
例: 「Server=myServerAddress;Database=myDataBase;UserId=myUsername;Password=myPassword;」
* **既存の DbConnection (DatabaseConnection)** - データベース接続アクティビティおよびデータベース トランザクション アクティビティから取得された、すでに開かれているデータベース接続。
このプロパティと、ConnectionString および ProviderName プロパティは相互に排他的です。このプロパティが設定されている場合、他のプロパティは設定できなくなります。
* **プロバイダ名**\*- データベースへのアクセスに使用されるデータベース プロバイダの名前。
例: 「System.Data.SqlClient」

**入力**

* **パラメータ** - SQL コマンドにバインドされる名前付きパラメータの辞書。バインドは、SQL コマンドで「@parameterName」ステートメントを指定することによって行われます。実行時に、parameterName は辞書の値に置き換えられます。
* **SQL クエリ (文字列)**\*- 実行される SQL コマンド。このフィールドは、CommandType プロパティの選択に従って入力する必要があります。
例: 「連絡先リストから \* を削除」

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: クエリ以外の実行
* **コマンド タイプ** - コマンド文字列の解釈方法を指定します。次のオプションが利用可能です。
・テキスト - SQLパラメータで指定されたクエリを実行します。
・StoredProcedure - SQLパラメータで指定したプロシージャ名を呼び出します。
・TableDirect - テーブルからすべてのレコードを呼び出します。

**出力**

* **影響を受けるレコード (Int32)** - SQL コマンドの実行の結果。 UPDATE、INSERT、および DELETE ステートメントの場合、戻り値はコマンドの影響を受ける行数です。他のすべてのタイプのステートメントの場合、戻り値は -1 です。
