# データテーブルの挿入

RCA.Activities.Core.InsertDataTable

## **説明**

このアクティビティを使用すると、互換性のある DataTable 変数を既存のテーブルに挿入できます。影響を受ける行数を返します。テーブルの制約が満たされない場合、アクティビティは失敗し、書き込まれたすべての行が保存されます。

![image-20220506105511-1.png](/static/img/bcb9f9_image-20220506105511-1.png)

（\*必須）

## **アクティビティ本体内**

* **接続の構成** - 接続文字列、データ ソース、データ プロバイダーを設定します。

![image-20230419153435-1.png](/static/img/3e2525_image-20230419153435-1.png)

* **データ テーブル** - テーブルに挿入される DataTable 変数。 DataTable 列の名前と説明は、データベース テーブルのものと一致する必要があります。
* **テーブル名** - データが挿入される SQL テーブル。
例: 「ターゲットテーブル」

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - このプロパティは、エラーが発生した場合にオートメーションを継続するタイミングを指定します。可能な値は True または False の 2 つだけです。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。

例: 本当

**接続構成**

* **接続文字列 (文字列)**\* - データベース接続を確立するために使用される接続文字列。
例: 「Server=myServerAddress;Database=myDataBase;UserId=myUsername;Password=myPassword;」
* **既存の DbConnection (DatabaseConnection)** - Connect アクティビティまたは Start Transaction アクティビティから取得された、すでに開かれているデータベース接続。このプロパティと、ConnectionString および ProviderName プロパティは相互に排他的です。このプロパティが設定されている場合、他のプロパティは設定できなくなります。
* **プロバイダー名**\* - データベースへのアクセスに使用されるデータベース プロバイダーの名前。
例: 「System.Data.SqlClient」

**入力**

* **データ テーブル (DataTable)**\* - テーブルに挿入される DataTable 変数。 DataTable 列の名前と説明は、データベース テーブルのものと一致する必要があります。
* **テーブル名 (文字列)**\*- データが挿入される SQL テーブル。
例: 「ターゲットテーブル」

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: データテーブルの挿入

**出力**

* **影響を受けるレコード (Int32)** - 影響を受ける行の数を Int32 変数に保存します。
