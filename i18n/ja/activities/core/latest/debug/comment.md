# コメント

RCA.Activities.Core.Comment

## **説明**

このアクティビティを使用すると、ワークフローにコメントすることができます。

![image-20220505092245-1.jpeg](/static/img/e3cbe3_image-20220505092245-1.jpeg)

（\*必須）

## **プロパティ**

**入力**

* **テキスト (文字列)**\* - Text プロパティにコメントを書き込みます。
例: このアクティビティは理解するのが難しいです

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例：[556910226] コメント
