# メッセージボックス

RCA.Activities.Core.MessageBox

## **説明**

このアクティビティを使用すると、指定したテキストとボタンのオプションを含むメッセージ ボックスを表示できます。

![image-20220505092843-1.jpeg](/static/img/25aa1c_image-20220505092843-1.jpeg)

（\*必須）

## **アクティビティ本体内**

* **テキスト (文字列)**\* – メッセージ ボックスに表示されるテキスト。
例：「今日は」

## **プロパティ**

**入力**

* **ボタン (ドロップダウン リスト)** - メッセージ ボックスに表示するボタンを指定します。使用可能なオプションは、「OK」、「OK/キャンセル」、「はい/いいえ」、および「はい/いいえ/キャンセル」です。
* **Captions (String)**- メッセージ ボックス ダイアログのタイトル。
例: 「警告! 注意!」
* **メッセージ ボックスの継続時間 (Int32)** - メッセージ ボックスが閉じるまでの継続時間 (ミリ秒単位)。入力が 500 ミリ秒より小さい場合、継続時間は無限に設定されます。
例: メッセージボックスの長さ = 400
* **テキスト (文字列)**\* – メッセージ ボックスに表示されるテキスト。
例：「今日は」

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [120548615] メッセージボックス
* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **最上位 (チェックボックス)** – 選択すると、メッセージ ボックスが常に最前面に表示されます。デフォルトでチェックされています

**出力**

* **選択されたボタン (文字列)** - Buttons プロパティで指定されたボタンのうち、実行時にクリックされたボタンを指定します。可能なオプションは次のとおりです: [OK]、[はい]、[いいえ]、または [キャンセル]。
