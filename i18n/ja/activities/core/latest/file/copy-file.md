# ファイルのコピー

RCA.Core.Activities.CopyFile

## **説明**

このアクティビティを使用すると、指定した場所から別の場所にファイルをコピーできます。

![image-20220513103651-1.png](/static/img/d5bb12_image-20220513103651-1.png)

（\*必須）

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - このプロパティは、エラーが発生した場合にオートメーションを継続するタイミングを指定します。可能な値は True または False の 2 つだけです。デフォルト値は False です。

**ファイル**

* **宛先 (文字列)**\* - ファイルのコピー先。フォルダー パスを指定すると、ファイルは元の名前でコピーされます。ファイル名を指定できます。
例: "C:\Users\Admin\Downloads\outputfile.xlsx"
* **パス (文字列)**\* - コピーするファイルのフルパス。
例: 「C:\Users\Admin\inputfile.xlsx」

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [351576757] フォルダーを選択

**オプション**

* **上書き (チェックボックス)** - 宛先フォルダー内のファイルをコピーするファイルで上書きするには、このオプションをオンにします。
