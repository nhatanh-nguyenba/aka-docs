# ファイルの移動

RCA.Activities.Core.MoveFile

## **説明**

このアクティビティを使用すると、指定した場所から別の場所にファイルを移動できます。

![image-20220513105249-1.png](/static/img/62cc32_image-20220513105249-1.png)

（\*必須）

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - このプロパティは、エラーが発生した場合にオートメーションを継続するタイミングを指定します。可能な値は True または False の 2 つだけです。デフォルト値は False です

**から**

* **Path (String)**\* - 移動するファイルまたはディレクトリのパス。
例: 「C:\Users\Admin\Downloads」

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [997394815] ファイルの移動
* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。

**オプション**

* **上書き (チェックボックス)** - チェックすると、コピー先のフォルダー内のファイルがコピーされるファイルで上書きされます。

**へ**

* **Destination (String)**\* - ファイルの移動先のパス。
例: 「C:\Users\Admin\Downloads」
