# パスが存在します

RCA.Activities.Core.PathExists

## **説明**

このアクティビティを使用すると、ダイアログを開いて、ファイルまたはフォルダー/ディレクトリの指定されたパスが存在するかどうかを確認できます。

![image-20220505093138-1.jpeg](/static/img/6d4e86_image-20220505093138-1.jpeg)

（\*必須）

## **アクティビティ本体内**

* **パスの種類 (ドロップダウン リスト)** - パスの種類を選択します。ファイルまたはフォルダー。
* **テキスト (文字列)**\* - チェックする必要があるファイルまたはフォルダーのフルパス。
例: 「C:\Users」

## **プロパティ**

**入力**

* **パス (文字列)**\* - チェックする必要があるファイルまたはフォルダーの完全なパス。
例: 「C:\Users」
* **パスの種類 (ドロップダウン リスト)** - パスの種類を選択します。ファイルまたはフォルダー。

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [791440496] パスが存在します

**出力**

* **パスが存在します (ブール値)** - チェックの結果。 True または False のみを返します。
