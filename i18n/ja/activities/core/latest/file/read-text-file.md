# テキストファイルを読む

RCA.Activities.Core.ReadTextFile

## **説明**

このアクティビティを使用すると、指定したファイルからすべての文字を読み取り、それを String 変数に保存できます。

![image-20220513110137-1.png](/static/img/2373a1_image-20220513110137-1.png)

（\*必須）

## **アクティビティ本体内**

* **ファイル名 (文字列)**\* - ファイルのパス。パスがフルでない場合、ファイルはプロジェクト フォルダーに作成されます。
例: "C:\Users\Admin\Desktop\inputfile.xlsx" または "inputfile.xlsx"

## **プロパティ**

**ファイル**

* **Encoding (String)**- 使用されるエンコード タイプ。エンコード タイプが指定されていない場合、アクティビティはファイルのバイト オーダー マークを検索してエンコードを検出します。バイト オーダー マークが検出されない場合は、デフォルトで UTF-8 エンコーディングが選択されます。
* **ファイル名 (文字列)\***- ファイルのパス。パスがフルでない場合、ファイルはプロジェクト フォルダーに読み込まれます。
例: "C:\Users\Admin\Desktop\inputfile.xlsx" または "inputfile.xlsx" (プロジェクト フォルダーに作成されたファイルの場合)

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [674340923] テキストファイルの読み取り

**出力**

* **コンテンツ (文字列)** - 文字列変数に格納されているファイルから抽出されたテキスト。
