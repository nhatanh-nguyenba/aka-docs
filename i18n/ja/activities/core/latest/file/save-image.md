# 画像の保存

RCA.Activities.Core.SaveImage

## **説明**

このアクティビティを使用すると、イメージをディスクに保存できます。

![image-20220513111129-1.png](/static/img/a26eec_image-20220513111129-1.png)

（\*必須）

## **プロパティ**

**入力**

* **ファイル名 (文字列)**\* - 画像を保存する名前を含む完全なファイル パス。
例: 「D:\NewFolder\downloadimage.jpg」
* **画像 (画像)**\* - 変数に格納されているディスクに保存する画像。

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: 画像を保存
