# テキストファイルの書き込み

RCA。 Core.Activities.WriteTextFile

## **説明**

このアクティビティを使用すると、指定したテキストをファイルに書き込むことができます。ファイルがすでに存在する場合、既存のテキストは上書きされます。ファイルが存在しない場合は作成されます。

![image-20220513111351-1.png](/static/img/2fbe16_image-20220513111351-1.png)

（\*必須）

## **アクティビティ本体内**

* **ファイル名 (文字列)\*** - ファイルのパス。パスがフルでない場合は、プロジェクト フォルダ内でファイルが検索されます。
例: "C:\Users\Admin\Desktop\inputfile.xlsx" または "inputfile.xlsx" (プロジェクト フォルダーに作成されたファイルの場合)
* **Text (String)\***- ファイルに書き込まれるテキスト。文字列と文字列変数の両方をサポートします。
例: 「ファイルにテキストを入力」

## **プロパティ**

**ファイル**

* **Encoding (String)** - 使用されるエンコーディングのタイプ。エンコード タイプが指定されていない場合、アクティビティはファイルのバイト オーダー マークを検索してエンコードを検出します。バイト オーダー マークが検出されない場合は、デフォルトで UTF-8 エンコーディングが選択されます。
* **ファイル名 (文字列)**\* - ファイルのパス。 [ファイル名]で指定したファイルに[入力]テキストを書き込みます。ファイルが存在しない場合は、ファイル [FileName] が作成されます。ファイルに値がある場合、[入力] テキストをこのファイル [ファイル名] に上書きします。

**入力**

* **テキスト (文字列)** - ファイルに追加されるテキスト。文字列と文字列変数の両方をサポートします。
例: 「ファイルにテキストを入力」

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [586182511] テキスト ファイルの書き込み
