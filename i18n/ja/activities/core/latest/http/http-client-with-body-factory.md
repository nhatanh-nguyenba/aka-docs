# Body Factory を備えた HTTP クライアント

## **説明**

この Body Factory アクティビティを使用した HTTP クライアントを使用すると、指定した Web API に対して HTTP リクエストを実行できます。最初にこのアクティビティを [デザイナー] パネルにドラッグすると、[HTTP リクエスト ウィザード] ウィンドウが表示され、リクエストを作成してサーバー応答をプレビューする簡単な方法が提供されます。

[![HTTP Client with.jpg](/static/img/63bea1_http-client-with.jpg)](/bin/download/Activities/HTTP/Activities/HTTP%20Client%20with%20Body%20Factory/WebHome/HTTP%20Client%20with.jpg?rev=1.1)

（\*必須）

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。
* **TimeOut MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 6000 (ミリ秒)。
例: 6000

**入力**

* **受け入れられた形式 (AcceptedHeaderType)** - Web サーバーの応答を受信する形式。次のオプションがサポートされています。
・ANY（デフォルト値）
・XML
・JSON。
* **エンドポイント (文字列)**\* - リクエストを行う先の URL。
例: https://akabot.com/
* **メソッド (ドロップダウン リスト)** – API を呼び出すときに使用されるリクエスト メソッド。次の HTTP 動詞がサポートされています:**GET、POST、PUT、DELETE、HEAD、OPTIONS、PATCH**、**MERGE。**
デフォルトでは、**GET** メソッドが選択されています。

**OAuth1**

* **コンシューマ キー (文字列)** - リクエストを行っている API が OAuth1 認証プロトコルに基づいている場合に使用されるコンシューマ キー。
* **コンシューマ シークレット (文字列)** - リクエストを行っている API が OAuth1 認証プロトコルに基づいている場合に使用されるコンシューマ シークレット。
* **OAuth1 トークン (文字列)** - リクエストを行っている API が OAuth1 認証プロトコルに基づいている場合に、コンシューマの承認リクエストが承認された後に使用されるアクセス トークン。
* **OAuth1Token Secret (String)** - リクエストを行っている API が OAuth1 認証プロトコルに基づいている場合に、コンシューマの承認リクエストが承認された後に使用されるシークレット。

**OAuth2**

* **OAuth2Token (String)** - リクエストを行っている API が OAuth2 認証プロトコルに基づいている場合に使用されるアクセス トークン。

**オプション**

* **添付ファイル (文字列)** - リクエストにファイルを添付できます。 [省略記号] ボタンをクリックして [添付ファイル] ウィンドウを開き、必要なファイルをすべて追加します。
* **本文 (文字列)** - リクエストの本文。
* **本文の形式 (文字列)** - 本文の形式。デフォルトでは、アプリケーション/XML が入力されます。
* **Cookies (文字列)** - HTTP リクエスト内の Cookie。
* **ヘッダー (文字列)** - HTTP リクエストにカスタム ヘッダーを含めることができます。
* **パラメータ (文字列)** - リクエストパラメータ。
* **リソース パス (文字列)** - 応答を保存するファイル パス。このフィールドは文字列と文字列変数をサポートします。
**例**: 「 D:\Users\
* **Url Segments\*** - 追加された行がフラグメントであること、または [エンドポイント] フィールドに指定された URL であることを示します。

**出力**

* **結果 (文字列)**\* - HTTP リクエストから受信した応答 (文字列変数)。
* **ステータス コード (Int32)** - HTTP リクエストから整数変数として受信したステータス コード。
例: 200

**プロキシ**

* **HTTP プロキシ** - アクセスしようとしているサイトの IP アドレス。これは文字列なので、引用符で囲む必要があります。
例：「127.0.0.0」
* **プロキシ ユーザー名** - プロキシへのアクセスに使用されるユーザー アカウントのユーザー名。これは文字列なので、引用符で囲む必要があります。
例: 「管理者」
* **プロキシ パスワード** - プロキシへのアクセスに使用されるユーザー アカウントのパスワード。これは文字列なので、引用符で囲む必要があります。
例：「abcd23」

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [35413123123] HTTP リクエスト

**簡単な認証**

* **パスワード (文字列)**\* - Web サービスが簡易認証を使用し、パスワードを入力できることを示します。パスワードの追加も必要です。このプロパティは文字列と文字列変数をサポートします。
例: admin@123
* **ユーザー名 (文字列)**\* - Web サービスが単純な認証を使用し、ユーザー名を入力できることを示します。ユーザー名の追加も必要です。このプロパティは文字列と文字列変数をサポートします。
例: ユーザー
