# 画像をクリック

RCA.Activities.Core.ClickImage

## **説明**

このアクティビティを使用すると、UI 要素内で画像を検索してクリックできます。

![image-20220505092211-1.jpeg](/static/img/54acdd_image-20220505092211-1.jpeg)

（\*必須）

## **アクティビティ本体内**

* **スクリーンショットを撮る** - 画像セレクター。このオプションをクリックすると、スクリーンショットを取得できます。このスクリーンショットを使用してテキストが抽出されます。

## **プロパティ**

**入力**

* **クリック タイプ (ドロップダウン リスト)** - 使用するマウス クリックのタイプを決定します。使用可能なオプションは、左クリック、左ダブルクリック、右クリック、右ダブルクリックです。
* **Delay MS (Int32)** - 実行後の時間。入力値が 1000 未満または空の場合、タイムアウトは 1000 ミリ秒になります。
* **OffsetX (Int32)** - 位置の水平方向の変位 (ピクセル単位)
* **OffsetY (Int32)** - 位置の垂直方向の変位。 (ピクセル単位)
* **画像パス (文字列)** - ディスクから画像をロードし、それを Image 変数に保存できます。
例: 「D:\photo.jpg」

**注意**: 画像パスを入力するか、画像セレクターを使用するかのどちらかのみを選択できます。

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [997446715] 画像をクリック
* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。

**オプション**

* **クリック位置 (ドロップダウン リスト)** - クリックを設定する位置。使用可能なオプションは、左、右、中央です。
* **自信 (ダブル)** - ボットが画像を認識する割合。値が大きいほど、ボットがクリックできる画像の精度が高くなければなりません。
例: 0.7
* **画像の高度な検索を使用する (チェックボックス)** - このツールを使用すると、画像を高度な検索として使用できます: ぼやけた/スケールアウトした画像...
