# 画像が存在します

RCA.Activities.Core.ImageFound

## **説明**

このアクティビティでは、画像が存在するかどうかを確認できます。

![image-20220505093941-1.jpeg](/static/img/3fd041_image-20220505093941-1.jpeg)

（\*必須）

## **アクティビティ本体内**

* **スクリーンショットを撮る** - 後で使用するために画像のスクリーンショットを撮ることができます。

## **プロパティ**

**入力**

* **Delay MS (Int32)** - 指定された時間内にアクティビティが実行されるまで待機します。入力値がない、または入力値 \<= 1000 の場合、待機時間 = 1000ms (1 秒)。それ以外の場合、待ち時間 = 入力値となります。

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [639993019] 画像が存在します

**オプション**

* **自信 (ダブル)** - ボットが画像を認識する割合。デフォルトの比率は 0.7 です。

**出力**

* **結果 (ブール値)** – 画像が存在するかどうかを示すチェックの結果。 True または False のみを返します。
