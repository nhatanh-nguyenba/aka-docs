# はじめに

コア アクティビティ パックには、自動化プロジェクトの作成に使用されるすべての基本アクティビティが含まれています。これらのアクティビティにより、ロボットは次のことが可能になります。

* OCR や画像認識などのテクノロジーを使用して、画像とテキストを実行します。
* マシン上のディレクトリやファイルと直接対話し、人間のユーザーが行うようなあらゆるアクションを実行します。

また、このパックには、論理演算子や式など、自動化プロジェクト自体の作成と実行を支援する他のタイプのアクティビティもあります。
