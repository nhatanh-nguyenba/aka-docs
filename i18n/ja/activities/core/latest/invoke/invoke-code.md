# コードの呼び出し

RCA.Activities.Core.InvokeCode

## **説明**

このアクティビティを使用すると、オプションで入力引数のリストを渡して、VB.NET または C# コードを同期的に呼び出すことができます。このアクティビティは、呼び出し元のワークフローに引数を返すこともできます。

![image-20220505160824-1.png](/static/img/bdc644_image-20220505160824-1.png)

（\*必須）

## **アクティビティ本体内**

* **コードの編集 (ボタン)** – ここをクリックして希望のコードを記述します
* **引数の編集 (ボタン)** – ここをクリックして、さまざまな種類の引数を追加および編集します。

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - このプロパティは、エラーが発生した場合にオートメーションを継続するタイミングを指定します。可能な値は True または False の 2 つだけです。
True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
False (デフォルト) - プロセスの実行の継続をブロックします。

**入力**

* **引数 (文字列)** - 呼び出されるコードに渡すことができるパラメータ
* **コード (文字列)** - 呼び出される VB.Net コード。

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [246443801] コードの呼び出し
