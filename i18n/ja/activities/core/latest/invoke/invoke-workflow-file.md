# ワークフローファイルの呼び出し

RCA.Activities.Core.InvokeWorkflowFile

## **説明**

このアクティビティを使用すると、指定したワークフローを同期的に呼び出し、オプションで入力引数のリストを渡すことができます。アクティビティの本文にある [引数のインポート] ボタンを使用すると、このアクティビティで指定したワークフローから引数をインポートできます。

![image-20220505161118-3.png](/static/img/ac80db_image-20220505161118-3.png)

（\*必須）

## **アクティビティ本体内**

* **ワークフロー パス (文字列)**\* - 呼び出される .xaml ファイルのファイル パス。
例: 「C:\Users\Admin\Documents\akaBot\Invokefile.xaml」
* **引数の編集 (ボタン)** – ここをクリックして、さまざまな種類の引数を追加および編集します。
* **引数のインポート (ボタン)** - このフィールドではさらに多くの引数をインポートできます

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - このプロパティは、エラーが発生した場合にオートメーションを継続するタイミングを指定します。可能な値は True または False の 2 つだけです。デフォルト値は False です。

**入力**

* **引数 (文字列)** - 呼び出されるコードに渡すことができるパラメータ
* **ワークフロー ファイル名 (文字列)**\* - 呼び出される .xaml ファイルのファイル パス。
例: 「C:\Users\Admin\Documents\akaBot\Invokefile.xaml」

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [663505729] ワークフロー ファイルの呼び出し

**オプション**

* **ワークフロー タイムアウト (TimeSpan)** - エラーがスローされるまでにアクティビティの実行を待機する時間。
例: 00:00:05
