# OCRテキストを取得する

RCA.Activities.Core.GetOCRText

## **説明**

このアクティビティを使用すると、OCR 画面スクレイピング方法を使用して UI 要素または画像からテキストを取得できます。このアクティビティは「OCR スコープ」内でのみ有効です。

![image-20220513104515-1.png](/static/img/252a2f_image-20220513104515-1.png)

（\*必須）

## **アクティビティ本体内**

* **スクリーンショットを撮る** – 画像セレクター。このオプションをクリックすると、スクリーンショットを取得できます。このスクリーンショットを使用してテキストが抽出されます。

## **プロパティ**

**入力**

* **Delay MS (Int32)** - 実行後の時間。デフォルト値は 300 ミリ秒です。
* **画像パス (文字列)** – テキストが描画される画像のパス。
* **テンプレート (文字列)** - テンプレートベースのシステムが登場します

**注意**: 画像パスを入力するか、画像セレクターを使用するかのどちらかのみを選択できます。

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [586133740] OCR テキストを取得

**オプション**

* **自信 (ダブル)** - ボットが画像を認識する割合。値が大きいほど、ボットがテキスト コンテンツを取得するために必要な画像の精度が高くなります。デフォルト値は 0.7 です。
* **画像詳細検索を使用する (ブール値)** - このツールを使用すると、画像を事前検索として使用できます: ぼやけた/スケールアウトした画像...

**対象**

* **高さ (Int32)** - 位置の高さ。
* **OffsetX (Int32)** - 位置の水平方向の変位 (ピクセル単位)
* **OffsetY (Int32)** - 位置の垂直方向の変位。 (ピクセル単位)
* **Width (Int32)** – ポジションの幅
