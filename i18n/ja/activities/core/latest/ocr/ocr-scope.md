# OCR スコープ

RCA.Activities.Core.OCRScope

## **説明**

このアクティビティを使用すると、OCR アクティビティを実行するためのスコープを作成できます。

![image-20230417095210-1.png](/static/img/ebba26_image-20230417095210-1.png)

## **アクティビティ本体内**

* **実行** - ツールボックスからさまざまなアクティビティをドラッグして追加できます。

## **プロパティ**

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [765026544] OCR スコープ

**オプション**

* **言語 (ドロップダウン リスト)** - 画像で使用される言語を指定します。利用可能なオプションは、英語、日本語、ベトナム語、繁体字中国語です。
