# ユーザー入力をブロックする

RCA.Activities.Core.BlockUserInput

## **説明**

このアクティビティを使用すると、アクティビティの実行中にマウスとキーボードからのユーザー入力をブロックできます。マウス、キーボード、またはその両方をブロックするように設定し、それらを再度有効にするホットキーの組み合わせを定義できます。

![image-20220505100618-1.jpeg](/static/img/image-20220505100618-1.jpeg)

\* は必須フィールドを示します。

## **アクティビティの本文内**

* **Alt**: 選択すると、akaBot は Alt キーを無効にします。
* **Ctrl**: 選択すると、akaBot は Ctrl キーを無効にします。
* **Shift**: 選択すると、akaBot は Shift キーを無効にします。
* **Win**: 選択すると、akaBot は Windows キーを無効にします。
* **キー**: マウスとキーボードを再アクティブ化するために使用されるキー。使用可能な特殊キーのリストが提供されます。

## **プロパティ**

**共通**

* **Continue On Error (Boolean)**: エラーが発生した場合でもオートメーションを続行するかどうかを指定します。 True または False の値のみがサポートされます。
デフォルト:`False`
* **タイムアウト MS (Int32)**: エラーがスローされるまで待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、実行は停止します。

**入力**

* **ブロック (ドロップダウン リスト)**: ブロックする対象を指定します: すべて、マウス、またはキーボード。

**ブロック解除オプション**

* **キー (文字列)**: マウスとキーボードを再アクティブ化するために使用されるキー。これはアクティビティの本文でも選択できます。
* **KeyModifiers (ドロップダウン リスト)**: 選択したキーと一緒に使用される修飾キー。利用可能なオプション: Alt、Ctrl、Shift、Win。
* **SpecialKey (ブール値)**: キーボード ショートカットで特殊キーが使用されているかどうかを示します。使用可能な特殊キーのリストは、アクティビティ本体の [キー] ドロップダウンに表示されます。

**その他**

* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データのセキュリティを考慮してください。
* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`[132682924] Block User Input`
