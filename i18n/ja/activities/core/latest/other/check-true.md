# 真にチェックを入れる

RCA.Activities.Core.CheckTrue

## **説明**

このアクティビティを使用すると、指定された式が True と評価されるかどうかを確認できます。

![image-20220505095635-1.jpeg](/static/img/image-20220505095635-1.jpeg)

\* は必須フィールドを示します。

## **プロパティ**

**入力**

* **ブール式 (ブール)**\*: 評価されるブール式。変数に格納されます。

**その他**

* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`[997467815] Check True`
* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データのセキュリティを考慮してください。

**出力**

* **結果 (ブール値)**: 評価された式の結果。
