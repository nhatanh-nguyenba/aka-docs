# キューアイテムを一括追加

RCA.Activities.Core.BulkAddQueueItems

## **説明**

このキュー項目の一括追加アクティビティを使用すると、項目のコレクションをデータ テーブルの特定のキューに追加できます。アイテムがキューに追加されると、アイテムのステータスは新規に変更されます。

![image-20220506113338-1.png](/static/img/993a2c_image-20220506113338-1.png)

（\*必須）

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。

例: 本当

* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。

**入力**

* **コミット タイプ (ドロップダウンリスト)**\* - 項目をキューに追加する方法とエラーの処理方法を選択できます。
次の 2 つのオプションがあります。
・All or nothing: エラーが発生しなければ、すべての項目が追加されます。そうしないと、すべての項目が追加されません。
・ProcessAllIndependently: 各項目を個別に追加し、追加できなかったすべてのキュー項目のリストを返します。
デフォルト値は次のとおりです: AllOrNothing
* **キュー アイテム データテーブル (DataTable)** - キューを追加するデータ テーブル。
* **QueueName (String)** - 新しいキューが追加されるキュー。文字列のみがサポートされます。

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: 一括追加キュー

**出力**

* **結果 (DataTable)** - これはキューに追加したデータ テーブルです。
