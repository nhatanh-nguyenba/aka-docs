# キュー項目の削除

RCA.Activities.Core.DeleteQueueItems

## **説明**

このアクティビティを使用すると、特定のキュー内のアイテムを削除できます。注: Studio でこのアクティビティを使用するには、エージェントを akaBot Center に接続する必要があり、キューの権限を編集、作成、削除し、トランザクションの権限を削除する必要があります。

![image-20220506113413-1.png](/static/img/a347ce_image-20220506113413-1.png)

（\*必須）

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。

例: 本当
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。

**入力**

* **キュー アイテム (`IEnumerable<Queue.Item>`)**\* - ユーザーが削除するすべてのアイテムを取得するには、キュー アイテムの取得アクティビティを使用する必要があります。このプロパティの値は、キュー アイテムの取得アクティビティの出力アイテムです。 [キュー アイテム] で指定されたすべてのキュー アイテムのステータスを削除済みに設定します。ただし、キュー アイテムのステータスが進行中である場合は削除済みに変更できません。

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: キュー項目を削除します。
