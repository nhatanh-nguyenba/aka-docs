# トランザクションアイテムの取得

RCA.Activities.Core.GetTransactionItem

## **説明**

このアクティビティを使用すると、キューから項目を取得して処理できるようになり、そのステータスを [進行中] に設定できます。注: ボットにはキューに対する編集権限が必要です。

![image-20220506113519-1.png](/static/img/3b0665_image-20220506113519-1.png)

（\*必須）

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - アクティビティがエラーをスローした場合でもオートメーションを続行するかどうかを指定します。このフィールドはブール値 (True、False) のみをサポートします。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。
**例**: 本当です
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。

**入力**

* **QueueName (文字列)**\* - 新しいキューが追加されるキュー。文字列のみがサポートされます。
例: 「キュー\_koBot01」

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: トランザクションアイテムを取得します。

**出力**

* **トランザクション アイテム (QueueItem)** - 取得されたトランザクション アイテムは、キュー変数に格納されます。

**参考**

* **フィルター戦略** - リファレンスに従ってデータをフィルターできます。次の 2 つのオプションがあります。
・開始: 参照フィールドに示された値で始まる参照を持つ項目を探します。
・等しい：参照フィールドに示された値と同じ参照を持つ項目を検索します。
デフォルト値は StartsWith です。
* **参照 (文字列)** - 参照に基づいて項目をフィルタリングします。
