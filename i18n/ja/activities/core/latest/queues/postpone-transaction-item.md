# 取引アイテムの延期

RCA.Activities.Core.PostponeTransactionItem

## **説明**

このアクティビティを使用すると、トランザクションを処理する必要がある時間パラメータを追加できます。

![image-20220506113548-1.png](/static/img/886660_image-20220506113548-1.png)

（\*必須）

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。

例: 本当です。
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。

**入力**

* **Deadline (DateTime)**\*- キュー アイテムが処理される直前の日付。
* **Postpone (DateTime)**\* - キュー アイテムが処理されるようになる日付。これには、次のような相対的な日付を入力できます。
DateTime.Now.AddHours()、DateTime.Now.AddDays()
* **トランザクション アイテム (QueueItem)**\* - 延期したいアイテム

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: トランザクションアイテムを延期します。
