# ファイル変更トリガー

System.Activities.Core.FileChangeTrigger

## **説明**

このアクティビティを使用すると、ファイルの変更情報を取得できます。これはイベントの監視でのみ使用できます。

![image-20220505160559-1.png](/static/img/a03576_image-20220505160559-1.png)

（\*必須）

## **プロパティ**

**イベント**

* **変更タイプ (チェックボックス)** - 選択する変更のタイプ。使用可能なオプションは次のとおりです: 作成、削除、変更、名前変更、ALl。
* **通知フィルター (ドロップダウン リスト)** - 変更を監視するフィルターの組み合わせ。ファイル名はデフォルトで選択されます。
使用可能なオプションは次のとおりです: FileName、DirectoryName、Attributes、Size、LastWrite、LastAccess、CreationTime、Security。

**ファイル**

* **ファイル名 (文字列)** - 変更を監視するファイルの名前。
例: 「inputfile.xlsx」
* **Path (String)**\* - 変更を監視するファイルのパス
例: 「D:\inputfile.xlsx」

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [250235209] ファイル変更トリガー

**オプション**

* **サブディレクトリを含める (チェックボックス)** - これは、指定した場所のサブディレクトリを含めるかどうかに使用されます。
