#  データサービスの範囲

RCA.Activities.DataService.DataServiceScope

## **説明**

このスコープは、データサービス アクティビティの実行に役立ちます

## **プロパティ**

* **入力**
  + **認証タイプ**: データ サービスでの認証に使用される方法を決定します。使用可能なオプションは 2 つあります: Agent と ManagedToken
  + **データ サービス クライアント:** 以前のデータ サービス スコープのトークンを再利用します
* **トークンの管理**:
  + **認証トークン**: 認証タイプが「管理トークン」に設定されている場合は、「認証トークン」フィールドにトークンを入力する必要があります。このトークンは、ボットが安全な接続を確立し、データ サービスと対話するために必要です。認証トークンを提供すると、ボットは自身を認証し、データ サービスに正常にアクセスできるようになります。
* **その他**
  + **表示名:** アクティビティの表示名
  + **パブリック**: 選択すると、変数と引数の値が詳細レベルで記録されます。
* **出力**
  + **出力データ サービス クライアント:** このアクティビティに続く DataService スコープ アクティビティは、再認証を回避するためにこの出力を使用しようとします。
