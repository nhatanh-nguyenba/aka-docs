# データサービス活動

**データ サービス アクティビティ** は、Studio でデータ サービスを操作するように設計された、事前に構築されたアクティビティのコレクションです。これらのアクティビティにより、ユーザーは自動化ワークフロー内でデータベース、Web サービス、API などのさまざまなデータ サービスと対話できるようになります。 これらのアクティビティにより、ロボットは次のことが可能になります。

* エンティティレコードの作成
* ID によるエンティティ レコードの取得
* エンティティレコードの更新
* エンティティレコードの削除
* エンティティレコードのクエリ
