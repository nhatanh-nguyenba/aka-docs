# 範囲の追加

RCA.Activities.Excel.AppendRange

## **説明**

このアクティビティを使用すると、DataTable 変数のデータを指定した Excel ワークシートの末尾に追加できます。
シートが存在しない場合は、「シート名」フィールドに指定された名前で新しいシートが作成されます。

![image-20220505140101-1.jpeg](/static/img/image-20220505140101-1.jpeg)

\* は必須フィールドを示します。

## **アクティビティの本文内**

* **ワークブックのパス**: 使用する Excel ファイルのフルパス。ファイルがプロジェクト フォルダー内にある場合は、相対パスを使用できます。文字列変数と文字列のみがサポートされます。
例:`"C:\FolderName\ClientList.xlsx"`
* **シート名**: データを追加するシートの名前。デフォルト値は`"Sheet1"`です。文字列変数と文字列のみがサポートされます。
例:`"ClientNames"`
* **データ テーブル**: 追加するデータ。DataTable 変数として提供されます。

## **プロパティ**

**入力**

* **ワークブック パス (文字列)**\*: 使用する Excel ファイルのフル パス。ファイルがプロジェクト フォルダー内にある場合は、相対パスを使用できます。文字列変数と文字列のみがサポートされます。
例:`"C:\FolderName\ClientList.xlsx"`
* **シート名 (文字列)**\*: データを追加するシートの名前。デフォルト値は`"Sheet1"`です。文字列変数と文字列のみがサポートされます。
例:`"ClientNames"`
* **データ テーブル (DataTable)**\*: 追加するデータ。DataTable 変数として提供されます。

**その他**

* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データ セキュリティ要件を考慮してください。
* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`"Append Excel Range"`
