# コラムを読む

RCA.Activities.Excel.ReadColumn

## **説明**

startingCell フィールドで指定されたセルで始まる列から値を読み取り、それを IEnumerable 変数に保存します。

![image-20220505140503-2.png](/static/img/b6579d_image-20220505140503-2.png)

（\*必須）

## **アクティビティ本体内**

* **ワークブック パス (文字列)**\* - 読み取りたい Excel ファイルのパス。
例: 「C:\フォルダー名\SaleReport.xlsx」
* **シート名 (文字列)**\* - 読み取りたいシート。
例: 「シート 1」
* **開始セル (文字列)**\* - 列データの抽出を開始するセル。デフォルトでは「A1」が入力されています。
例**:**「A1」

## **プロパティ**

**入力**

* **シート名 (文字列)**\* - 読み取りたいシート。
例: 「シート 1」
* **開始セル (文字列)**\* - 列データの抽出を開始するセル。デフォルトでは「A1」が入力されています。
例：「A1」
* **ワークブック パス (文字列)**\* - 読み取りたい Excel ファイルのパス。
例: 「C:\フォルダー名\SaleReport.xlsx」

**その他**

* **パブリック (ブール値)** - アクティビティを公開するかどうかをチェックします。このプロパティを使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [482156710] 列を読む

**オプション**

* **形式を保持 (ブール値)** - このチェック ボックスをオンにすると、読み込む列の形式が保持されます。デフォルトでは、チェックボックスはオフになっています。

**出力**

* **結果 (IEnumerable)** - 指定されたスプレッドシート列の情報を IEnumerable 変数に保存します。
