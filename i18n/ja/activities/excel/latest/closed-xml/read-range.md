# 読み取り範囲

RCA.Activities.Excel.ReadRange

## **説明**

このアクティビティを使用すると、Excel 範囲からデータを読み取り、DataTable 変数に保存できます。
範囲が指定されていない場合は、最初のセルから始まるワークシート全体が読み取られます。
このアクティビティでは、ClosedXML ライブラリを使用します。

![image-20220505135434-1.jpeg](/static/img/image-20220505135434-1.jpeg)

\* は必須フィールドを示します。

## **アクティビティの本文内**

* **ワークブック パス (文字列)**\*: 読み取る Excel ファイルのパス。
例:`"C:\FolderName\SaleReport.xlsx"`
* **シート名 (文字列)**\*: 読み取るシート。
例:`"Sheet1"`
* **範囲**: 読み取るデータの範囲。指定しない場合は、ワークシート全体が読み取られます。
例:`"A1:B2"`

## **プロパティ**

**入力**

* **ワークブック パス (文字列)**: 読み取る Excel ファイルのパス。
例:`"C:\FolderName\SaleReport.xlsx"`
* **シート名 (文字列)**: 読み取るシート。
例:`"Sheet1"`
* **範囲**: 読み取るデータの範囲。指定しない場合は、ワークシート全体が読み取られます。
例:`"A1:B2"`

**オプション**

* **ヘッダーの追加 (チェックボックス)**: 選択すると、指定された範囲の列ヘッダーが含まれます。デフォルトでは、このオプションは選択されています (`True`)。
* **形式を保持 (チェックボックス)**: 範囲の元の形式を保持します。デフォルトでは、このオプションは選択されていません。

**その他**

* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データのセキュリティを考慮してください。
* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`Read Range`

**出力**

* **DataTable (DataTable)**: 指定された Excel 範囲から抽出されたデータを DataTable 変数に格納します。
