# セルの書き込み

RCA.Activities.Excel.WriteCell

## **説明**

このアクティビティを使用すると、指定したスプレッドシートのセルまたは範囲に値または数式を書き込むことができます。
シートが存在しない場合は、新しいシートが作成されます。

![image-20220505135743-1.jpeg](/static/img/image-20220505135743-1.jpeg)

\* は必須フィールドを示します。

## **アクティビティの本文内**

* **ワークブックのパス**: 使用する Excel ファイルのパス。
例:`"C:\FolderName\ClientList.xlsx"`
* **シート名**: データを書き込むシート。
例:`"ClientNames"`
* **セルの内容**: セルまたは範囲に書き込む値または数式。文字列変数、汎用変数、および文字列のみがサポートされます。
例:`"Hello world"`または`"=A2&A3"`
* **セル**: 書き込むセル。文字列値のみがサポートされます。
例:`"A20"`

## **プロパティ**

**入力**

* **ワークブック パス (文字列)**\*: 使用する Excel ファイルのパス。
例:`"C:\FolderName\ClientList.xlsx"`
* **シート名 (文字列)**\*: データを書き込むシート。
例:`"ClientNames"`
* **値 (文字列)**: セルまたは範囲に書き込む値または式。文字列変数、汎用変数、および文字列のみがサポートされます。
例:`"Hello world"`または`"=A2&A3"`
* **セル (文字列)**\*: 書き込むセル。文字列値のみがサポートされます。
例:`"A20"`

**その他**

* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データのセキュリティを考慮してください。
* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`Write Cell`
