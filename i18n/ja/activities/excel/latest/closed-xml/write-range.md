# 書き込み範囲

RCA.Activities.Excel.WriteRange

## **説明**

このアクティビティを使用すると、[開始セル] フィールドで指定されたセルから開始して、DataTable 変数のデータをスプレッドシートに書き込むことができます。
開始セルが指定されていない場合、データはセル`A1`から書き込まれます。

![image-20220505135914-1.jpeg](/static/img/image-20220505135914-1.jpeg)

\* は必須フィールドを示します。

## **アクティビティの本文内**

* **ワークブック パス**: 書き込む Excel ファイルのパス。
例:`"C:\FolderName\ClientList.xlsx"`
* **シート名**: データが書き込まれるシートの名前。デフォルト値は`"Sheet1"`です。
例:`"ClientNames"`
* **開始セル**: データの書き込みを開始するセル。文字列変数と文字列のみがサポートされます。
* **データ テーブル**: 指定された範囲に書き込むデータ。DataTable 変数として提供されます。

## **プロパティ**

**入力**

* **ワークブック パス (文字列)**\*: 書き込む Excel ファイルのパス。
例:`"C:\FolderName\ClientList.xlsx"`
* **シート名 (文字列)**\*: データが書き込まれるシートの名前。デフォルト値は`"Sheet1"`です。
例:`"ClientNames"`
* **開始セル (文字列)**\*: データの書き込みを開始するセル。文字列変数と文字列のみがサポートされます。
* **データ テーブル (DataTable)**\*: 指定された範囲に書き込むデータ。DataTable 変数として提供されます。

**オプション**

* **ヘッダーの追加 (チェックボックス)**: 選択すると、データの書き込み時に列ヘッダーが含まれます。デフォルトでは、このオプションは選択されています (`True`)。

**その他**

* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データ セキュリティ要件を考慮してください。
* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`Write Range`
