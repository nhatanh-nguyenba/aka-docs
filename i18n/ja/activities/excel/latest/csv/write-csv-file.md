# CSVファイルの書き込み

RCA.Activities.CSV.WriteCSVFile

## **説明**

このアクティビティを使用すると、指定した DataTable を CSV ファイルに上書きできます。

## **アクティビティ本体内**

* **ファイル パス (文字列)** - ここに CSV のファイル パスを追加できます。文字列のみがサポートされています。
* **データ テーブル (DataTable)** - CSV ファイルに書き込まれる DataTable を指定します。

## **プロパティ**

**入力**

* **データ テーブル (DataTable)**\* - CSV ファイルに書き込まれる DataTable を指定します。

**ファイル**

* **ファイル パス (文字列)\***- CSV ファイルのファイル パス。
例: 「C:\CSVFolder\clientList.csv」

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: 「CSV を書き込む」

**オプション**

* **区切り文字 (DropDownList)** - CSV ファイル内の区切り文字を指定します。オプションは次のとおりです: タブ、カンマ、セミコロン
* **エンコーディング (文字列)** - 使用するエンコーディングのタイプ。使用するエンコーディング タイプを指定するには、[名前] フィールドの値を使用します。文字列変数のみ。
例: 「utf-8」
* **ヘッダーの追加 (チェックボックス)** - DataTable の列名を出力 CSV ファイルに追加するかどうかを指定するブール値。
