# Excel 範囲選択

RCA.Activities.Excel.ExcelSelectRange

## **説明**

このアクティビティを使用すると、Excel スプレッドシートから指定した範囲を選択して実行し、さらにアクションを実行できます。

![image-20220505133120-1.jpeg](/static/img/bc8598_image-20220505133120-1.jpeg)

（\*必須）

## **プロパティ**

**入力**

* **シート名 (文字列)**- 指定した範囲を選択するシートの名前。デフォルトでは、「Sheet1」が入力されます。文字列変数と文字列のみがサポートされます。
* **範囲 (文字列)**\* - 選択される範囲。文字列変数と文字列のみがサポートされます。
例: 「A1:K10」

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: 範囲の選択
