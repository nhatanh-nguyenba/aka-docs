# エクスポートチャート

RCA.Activities.Excel.ExcelExportChart

## **説明**

このアクティビティを使用すると、指定した Excel ファイルからチャートまたはグラフを画像としてエクスポートできます。

![image-20220505134428-1.jpeg](/static/img/019cc6_image-20220505134428-1.jpeg)

（\*必須）

## **プロパティ**

**入力**

* **画像形式 (ドロップダウンリスト)\***- エクスポートしたチャート/グラフを保存するファイルの種類を選択します。 Png、Bmp、Jpgの3種類が入っています。
* **名前には (文字列) が含まれます**\* - チャートの名前。
例: 「チャートセール」
* **保存先のパス (文字列)\***- ファイルを保存するディレクトリ。
例: 「C:\DestinationSaveFolderName\」
* **シート名 (文字列)** - エクスポートするグラフを含むシートの名前。
例: 「SheetHasChartName」

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: チャートのエクスポート

**出力**

* **ファイル (List< String >)** - ファイル名を含む文字列のリスト。
