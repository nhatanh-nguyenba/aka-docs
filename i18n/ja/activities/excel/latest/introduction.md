# エクセルアクティビティ

Excel アクティビティは Microsoft Excel との緊密な統合を提供し、Excel で実行するタスクを自動化する多数のアクティビティが付属しています。これらのアクティビティにより、ロボットは次のことが可能になります。

* セル、列、行、範囲から情報を読み取り、他のスプレッドシートやワークブックに書き込みます。
* マクロを実行したり、数式を抽出したりすることもできます。

また、このパックには、論理演算子や式など、自動化プロジェクト自体の作成と実行を支援する他のタイプのアクティビティもあります。
