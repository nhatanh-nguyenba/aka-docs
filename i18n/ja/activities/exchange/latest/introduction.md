# 交換

Exchange アクティビティ パックには、自動化プロジェクトの作成に使用されるすべてのアクティビティが含まれています。これらのアクティビティにより、ロボットは次のことが可能になります。

* メールを別のフォルダーに移動します。
* Microsoft Exchange Server で電子メールを送受信します。
