# レイアウトコンポーネント

カラムグリッド、パネル、テーブル、タブページなどのレイアウト用コンポーネントについて説明します。

![layout-components.png](/static/img/layout-components.png)

---

## HTML Element

**データ型:** なし（レイアウトコンテナ）

任意の HTML タグをフォーム内にレンダリングできます。

* **HTML Tag** - `div`, `span`, `p`, `h1` など指定。
* **Content** - タグ内に表示するテキストまたは HTML。

---

## Content

リッチテキストやメディアを表示するコンテナです。別のフィールドの変更時に再描画する設定も可能です。

---

## Columns / Panel / Table / Tabs

カラム分割、グループ化、表レイアウト、タブページなど、複雑なフォームレイアウトを作成するためのコンポーネント群です。
