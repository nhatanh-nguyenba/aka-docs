# はじめに

**Form Builder アクティビティ** パッケージには、ワークフロー内で Form Builder のフォームを表示および操作するためのアクティビティが含まれています。

これらのアクティビティにより、ロボットは次のことが行えます：

* JSON スキーマファイルからフォームを表示する。
* フォームフィールドにデータを渡し、送信されたデータを取得する。
* フォームが開いている間にフィールドの値を取得または設定する。
* 表示されたフォームを有効化、無効化、フォーカス設定、リセット、閉じる。

ほとんどの Form Builder 操作では **Display Form** をコンテナアクティビティとして使用してください。
