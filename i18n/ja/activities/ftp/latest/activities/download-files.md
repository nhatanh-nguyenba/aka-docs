# ファイルをダウンロードする

RCA.Activities.FTP.DownloadFiles

## **説明**

このアクティビティでは、FTP サーバーにファイルをダウンロードできます。

![image-20220505152223-1.png](/static/img/fc347d_image-20220505152223-1.png)

（\*必須）

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト): プロセスの実行の継続をブロックします。

例: 本当

**入力**

* **リモート パス (文字列)** - ダウンロードされる FTP サーバー上のファイルのパス。
例: 「/myftpurl.com/wwwroot/myRemoteFolder/」

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: 「FTP ダウンロードファイル」

**オプション**

* **作成 (チェックボックス)** - ローカル マシンに新しいフォルダーを作成してフォルダーを保存するかどうかをチェックします。これはデフォルトではオフになっています。
* **サブフォルダーを含める (チェックボックス)** - すべてのサブフォルダーをダウンロードする場合にチェックを入れます。これはデフォルトではオフになっています。
* **上書き (チェックボックス)** - フォルダー名が重複した場合にフォルダーを上書きするかどうかをチェックします。これはデフォルトではオフになっています。

**出力**

* **ローカル パス (文字列)** - ダウンロードされるファイルのローカル パス。このフィールドは文字列変数のみをサポートします。
例: 「C:/宛先フォルダー」
