# ファイルを列挙する

RCA.Activities.FTP.EnumerateFiles

## **説明**

このアクティビティを使用すると、指定した FTP サーバー パスで見つかったファイルのリストを取得し、それを DataTable 変数の下に保存できます。

![image-20220505152257-1.png](/static/img/e04ad9_image-20220505152257-1.png)

（\*必須）

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト): プロセスの実行の継続をブロックします。

例: 本当

**入力**

* **リモート パス (文字列)**\* - ファイルが列挙される FTP サーバー上のディレクトリのパス。
例: 「/root/TargetFolder/」

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: 「FTP ファイルの列挙」

**オプション**

* **サブフォルダーを含める (チェックボックス)** - すべてのサブフォルダーを列挙する場合にチェックを入れます。これはデフォルトではオフになっています。

**出力**

* **ファイル (IEnumerable)** - 出力を保存するための IEnumerable 変数。
