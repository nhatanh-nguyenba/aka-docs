# ファイルが存在します

RCA.Activities.FTP.FileExists

## **説明**

* このアクティビティでは、ファイルが FTP サーバーに存在するかどうかを確認できます。

![image-20220505152334-1.png](/static/img/382075_image-20220505152334-1.png)

（\*必須）

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト): プロセスの実行の継続をブロックします。

例: 本当

**入力**

* **リモート パス (文字列)\***- 指定されたファイルが存在するかどうかを確認する FTP サーバー上のファイルのパス。
例: 「/root/フォルダー名/FileThatNeedcheck.txt」

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: 「FTP ファイルが存在します」

**出力**

* **Exist (ブール値)** - 変数にはチェック結果 (はい/いいえ) が含まれます
