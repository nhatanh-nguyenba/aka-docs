# はじめに

FTP アクティビティ パックには、自動化プロジェクトの作成に使用されるすべてのアクティビティが含まれています。これらのアクティビティにより、ロボットは次のことが可能になります。

* FTP サーバー上のディレクトリやファイルと直接対話し、人間のユーザーが行うようなあらゆるアクションを実行します。
