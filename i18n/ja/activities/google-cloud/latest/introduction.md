# はじめに

**Google Cloud アクティビティ** パッケージには、Google Cloud への認証と Google Cloud Storage のバケットおよびオブジェクトを操作するためのアクティビティが含まれています。

子アクティビティに Google Cloud のコンテキストを提供するには **GCP Scope** を使用してください。
