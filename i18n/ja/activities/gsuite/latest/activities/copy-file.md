# ファイルのコピー

RCA.Activities.Gsuite.CopyFile

## **説明**

このアクティビティでは、Google ドライブにファイルをコピーできます

![image-20220505155012-1.png](/static/img/c03b32_image-20220505155012-1.png)

（\*必須）

## **アクティビティ本体内**

* **コピーするファイルの ID (文字列)**\* - コピーするファイルの ID。文字列変数と文字列がサポートされています。
* **宛先フォルダーの ID、または「ルート」 (文字列)**\* - ファイルがコピーされるフォルダーの宛先。文字列変数と文字列がサポートされています。

## **プロパティ**

**入力**

* **FileID (String)**\* - コピーするファイルの ID。このプロパティは、文字列変数と文字列形式の値をサポートします。
* **新しいファイル名 (文字列)** - コピーするファイルの新しい名前。このプロパティは、文字列変数と文字列形式の値をサポートします。
* **新しい親 ID (文字列)** - ファイルをコピーする親フォルダーの ID。このプロパティは、文字列変数と文字列形式の値をサポートします。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [185197651] ファイルのコピー

**出力**

* **結果 (文字列)** - コピーされたファイルの ID。このプロパティは文字列変数をサポートします。この変数は、他の Google GSuite アクティビティの入力プロパティ値として使用できます。
