# ファイルの作成権限

RCA.Activities.GSuite.CreateFilePermission

## **説明**

このアクティビティを使用すると、Google ドライブ内の指定されたファイルに対する権限を作成できます。

![image-20220505155108-1.png](/static/img/00cbf7_image-20220505155108-1.png)

（\*必須）

## **アクティビティ本体内**

* **権限を作成するファイルの ID (文字列)**\* - 権限を作成するファイルの ID。このプロパティは、文字列変数と文字列形式の値をサポートします。

## **プロパティ**

**入力**

* **ファイル ID (文字列)**\* - アクセス許可を作成するファイルの ID。このプロパティは、文字列変数と文字列形式の値をサポートします。
* **ドメイン (文字列)** - ファイル権限を付与するドメイン。このプロパティは、文字列変数と文字列形式の値をサポートします。このプロパティは、DeliverType 値が Domain の場合にのみ必要です。
* **GaranteeType (ドロップダウン リスト)** - ファイル権限を付与するエンティティを指定するドロップダウン リスト。使用可能なプロパティ値は次のとおりです。
**・user** - 単一のユーザーに権限を付与します。
**・group** - グループに権限を付与します。
**・ドメイン** - ドメインに権限を付与します。
**・anyone** - 会社のドメインにアクセスできるすべてのユーザーに権限を付与します。
* **ロール (ドロップダウン リスト)** - エンティティに付与するファイル権限ロールを指定するドロップダウン リスト。使用可能なプロパティ値は次のとおりです。
**・リーダー** - エンティティはファイルを読み取り、ダウンロードできます。
**・ライター** - エンティティはファイルを編集できます。
**・所有者** - エンティティはファイルを所有します。
**・コメント者** - エンティティはファイルにコメントできます。
* **UserEmail (文字列)** - ファイル権限を取得しているユーザーまたはグループのメール アドレス。このプロパティは、文字列変数と文字列形式の値をサポートします。このプロパティは、EAN 保証タイプの値がユーザーまたはグループの場合にのみ必要です。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [821884234] ファイル作成権限

**オプション**

* **電子メール メッセージ (文字列)** - ファイルのアクセス許可が更新されたときに送信するメッセージの内容。このプロパティは、文字列変数と文字列形式の値をサポートします。デフォルト値は「新しい権限が作成されました。あなたが受信者です!」です。このプロパティは、** SendNotificationEmail** が選択されている場合にのみ必要です。
例: 「新しい権限が作成されました。あなたが受信者です。」
* **SendNotificationEmail (チェックボックス)** - 選択すると、許可を得て電子メール通知がエンティティに送信されます。
* **UseDomainAdminAccess (チェックボックス)** - 選択すると、要求者がファイルが属するドメインの管理者であればアクセスが許可されます。

**出力**

* **結果 (文字列)** - 付与された権限。このプロパティは、Google Permission 変数をサポートします。
