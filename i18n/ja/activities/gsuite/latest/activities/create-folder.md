# フォルダーの作成

RCA.Activities.GSuite.CreateFolder

## **説明**

このアクティビティにより、Google ドライブに新しいフォルダーを作成できます。

![image-20220505155144-1.png](/static/img/f40f41_image-20220505155144-1.png)

（\*必須）

## **アクティビティ本体内**

* **無題のフォルダー (文字列)**\* - 新しいフォルダーの説明。文字列変数がサポートされています。
例: :「タイル化されていないフォルダー」
* **親フォルダーの ID、または「ルート」（文字列）** - 新しいフォルダーを作成する親フォルダーの ID。

## **プロパティ**

**入力**

* **フォルダーの説明 (文字列)** - 新しいフォルダーの説明。文字列変数がサポートされています。
* **フォルダー名 (文字列)**\* - 新しいフォルダーの名前。文字列変数と文字列がサポートされています。
例: :「タイル化されていないフォルダー」
* **親 ID (文字列)** - 新しいフォルダーを作成する親フォルダーの ID。このプロパティは、文字列変数と文字列形式の値をサポートします。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。2
例: [91512312] フォルダーの作成

**出力**

* **結果 (文字列)** - 新しいフォルダーの ID。文字列変数がサポートされています。
