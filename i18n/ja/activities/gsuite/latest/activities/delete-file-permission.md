# ファイルの削除権限

RCA.Activities.GSuite.DeleteFilePermission

## **説明**

このアクティビティにより、ファイルに対する削除権限が許可されます。

![image-20220505155423-1.png](/static/img/d19794_image-20220505155423-1.png)

（\*必須）

## **アクティビティ本体内**

* **削除権限を持つファイルの ID (文字列)**\* - 削除権限を要求されたファイルの ID。文字列変数と文字列がサポートされています。
* **削除する権限の ID (文字列)**\* - 削除する権限の ID。

## **プロパティ**

**入力**

* **ファイル ID (文字列)\***- アクセス許可を削除するファイルの ID。このプロパティは、文字列変数と文字列形式の値をサポートします。
* **PermissionID (String)**\* - 削除する権限の ID。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [345963453] ファイルの削除権限

**オプション**

* **ドメイン管理者アクセスを使用する (チェックボックス)** - 選択すると、要求者がファイルが属するドメインの管理者であればアクセスが許可されます。
