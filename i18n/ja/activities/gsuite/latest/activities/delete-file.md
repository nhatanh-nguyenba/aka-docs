# ファイルの削除

RCA.Activities.GSuite.DeleteFile

## **説明**

このアクティビティを使用すると、Google ドライブにあるファイルを削除できます。

![image-20220505155255-1.png](/static/img/17f323_image-20220505155255-1.png)

（\*必須）

## **アクティビティ本体内**

* **削除するファイルの ID (文字列)**\* - 削除するファイルの ID。このプロパティは、文字列変数と文字列形式の値をサポートします。

## **プロパティ**

**入力**

* **FileID (String)**\* - 削除するファイルの ID。このプロパティは、文字列変数と文字列形式の値をサポートします。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
**例**: [123125436] ファイルを削除
