# ダウンロードファイル

RCA.Activities.GSuite.DownloadFile

## **説明**

このアクティビティは、Google ドライブがファイルをダウンロードするように指定されています。

![image-20220505155459-1.png](/static/img/de3c0d_image-20220505155459-1.png)

（\*必須）

## **アクティビティ本体内**

* **ダウンロードするファイルの ID (文字列)**\* - ダウンロードするファイルの ID。このプロパティは、文字列変数と文字列形式の値をサポートします。

## **プロパティ**

**入力**

* **FileID (String)**\* - ダウンロードするファイルの ID。このプロパティは、文字列変数と文字列形式の値をサポートします。
* **LocalPath (文字列)** - ファイル名と拡張子を含む、ファイルをダウンロードする完全なパス。このプロパティは、文字列変数と文字列形式の値をサポートします。
例: 「D:\file.txt\」

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [610085514] ダウンロードファイル
