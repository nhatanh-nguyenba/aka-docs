# ファイルフォルダーの検索

RCA.Activities.GSuite.FindFileFolder

## **説明**

このアクティビティでは、Google ドライブ上にあるファイルまたはフォルダーを検索できます。次に、名前、サイズ、ID などのファイル情報を返します。

![image-20220505155537-1.png](/static/img/af53b1_image-20220505155537-1.png)

（\*必須）

## **アクティビティ本体内**

* **オブジェクトの検索 (文字列)**\* - オブジェクトは名前、サイズ、または ID です。文字列変数と文字列がサポートされています。
例: 「name = '{0} を null にすることはできません。」

## **プロパティ**

**入力**

* **MaxResults (Int32)** - 返すファイルの最大数を指定します。文字列変数と文字列がサポートされています。デフォルト: 50

例: 50
* **SearchExpression (String)** - 返したいファイルの最大数を指定します。文字列変数と文字列がサポートされています。

例: 「name = '{0} を null にすることはできません。」

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [2144546456] ファイルフォルダーの検索

**出力**

* **FeelingLucky (文字列)** - 最初の結果のファイル ID。このプロパティは文字列変数をサポートします
* **結果 (ファイル)** - 検索結果に一致するファイル。このプロパティは、Google Files 配列変数をサポートします。
