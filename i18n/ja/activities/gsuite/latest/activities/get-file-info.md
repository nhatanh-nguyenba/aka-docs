# ファイル情報の取得

RCA.Activities.GSuite.GetFileInfo

## **説明**

このアクティビティを使用すると、ファイルの名前、サイズ、タイプなどのファイルに関する情報を返すことができます。

![image-20220505155834-1.png](/static/img/7c4f84_image-20220505155834-1.png)

（\*必須）

## **アクティビティ本体内**

* **情報を取得するファイルの ID (文字列)**\* - 情報を取得するファイルの ID。文字列変数と文字列がサポートされています。

## **プロパティ**

**入力**

* **FileID (String)**\* - 情報を取得するファイルの ID。このプロパティは、文字列変数と文字列形式の値をサポートします。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
**例**: [913123123] ファイル情報の取得

**出力**

* **結果 (ファイル)** - ファイルの詳細。このプロパティは Google の「ファイル」変数をサポートします
