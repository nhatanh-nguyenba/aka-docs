# ファイル権限の取得

RCA.Activities.GSuite.GetFilePermission

## **説明**

このアクティビティにより、リクエストに応じて、指定されたファイルに対するアクセス許可を取得できます。

![image-20220505160123-1.png](/static/img/4706f3_image-20220505160123-1.png)

（\*必須）

## **アクティビティ本体内**

* **アクセス許可を取得するファイルの ID (文字列)**\* - アクセス許可の取得を要求されたファイルの ID。文字列変数と文字列がサポートされています。

## **プロパティ**

**入力**

* **ファイル ID (文字列)**\* - 許可を取得するために要求されたファイルの ID。文字列変数と文字列がサポートされています。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [65623834] ファイル権限の取得

**オプション**

* **ドメイン管理者アクセスを使用する (チェックボックス)** - 選択すると、要求者がファイルが属するドメインの管理者であればアクセスが許可されます。

**出力**

* **結果 (文字列)** - 指定されたファイルの権限。このプロパティは Google Permission 配列変数をサポートします
