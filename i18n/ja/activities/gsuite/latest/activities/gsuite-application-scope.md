# GSuiteの適用範囲

RCA.Activities.GSuite.GsuiteApplicationScope

## **説明**

このアクティビティを使用すると、GSuite アクティビティの接続クライアントと認証を処理できます。 GSuite アクティビティは、アプリケーション スコープで指定されたアカウントで動作します。複数の認証タイプを使用できます。

![image-20220505160344-1.png](/static/img/a63f7e_image-20220505160344-1.png)

（\*必須）

## **プロパティ**

**API キー**

* **ApiKey (文字列)** - アカウント API キー。 このプロパティは、文字列変数と文字列形式の値をサポートします。

**入力**

* **認証タイプ (ドロップダウン リスト)** - GSuite サービスへのアクセスに使用する認証のタイプを指定するドロップダウン リスト。使用可能なプロパティ値は、** ApiKey、OAuthClientID**、** ServiceAccountKey** です。選択した認証タイプに応じて、追加のプロパティ フィールドが必要になる場合があります。
* **サービス (GoogleServices)** - 利用可能なサービス: Gmail やドライブなど。活動目的に合わせて適切なものをお選びください。

**OAuth クライアント ID (\*AuthenticationType 値が OAuthClientID の場合のみ)**

* **ClientID (String)**\* - ユーザー アカウントの認証情報 ID。文字列変数と文字列がサポートされています。
* **ClientSecret (文字列)**\* - ユーザー アカウントの認証情報。文字列変数と文字列がサポートされています。
* **ユーザー (文字列)** - 承認されたユーザーに割り当てられた識別子。識別子ごとに一意の応答トークンが生成され、将来の使用に備えて保存できます。文字列変数と文字列がサポートされています。
・このプロパティは、選択した AuthenticationType が OAuthClientID の場合にのみ適用されます。

**サービス アカウント キー (\*AuthenticationType 値が ServiceAccountKey の場合のみ)**

* **HasDomainwideAccount (チェックボックス)** - チェックすると、アクティビティによってサービス アカウントへのドメイン全体のアクセスが許可されます。
* **KeyPath (文字列)** - サービス アカウント キー ファイルへのパス。このプロパティは、文字列変数と文字列形式の値をサポートします。
* **KeyType (ドロップダウン リスト)** - JSON と P12 のドロップダウン リスト
* **パスワード (文字列)** - KeyType が P12 の場合、パスワードが必要です。文字列変数と文字列がサポートされています
* **サービス電子メール アカウント (文字列)** - KeyType で P12 を選択すると、電子メール アドレスが自動的に生成されます。
* **ユーザーの電子メール (文字列)** - HasDomainwideAccess がチェックされている場合、サービス アカウントはドメインのユーザーになりすます。このフィールドを使用して、偽装ユーザーの電子メール アドレスを入力します。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
**例**: [4523423424] GSuite の適用範囲
