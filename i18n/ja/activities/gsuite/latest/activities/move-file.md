# ファイルの移動

RCA.Activities.Gsuite.MoveFile

## **説明**

このアクティビティでは、Google ドライブ上のファイルを移動できます

![image-20220505160420-1.png](/static/img/13f180_image-20220505160420-1.png)

（\*必須）

## **アクティビティ本体内**

* **移動するファイルの ID (文字列)**\* - 移動するファイルの ID。文字列変数と文字列がサポートされています。
* **宛先フォルダーの ID、または「ルート」 (文字列)**\* - ファイルの移動先フォルダーの宛先。文字列変数と文字列がサポートされています。

## **プロパティ**

**入力**

* **FileID (String)**\* - 移動するファイルの ID。このプロパティは、文字列変数と文字列形式の値をサポートします。
* **新しいファイル名 (文字列)** - 移動するファイルの新しい名前。このプロパティは、文字列変数と文字列形式の値をサポートします。
* **新しい親 ID (文字列)** - 新しい親フォルダーの ID。このプロパティは、文字列変数と文字列形式の値をサポートします。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [345197651] ファイルの移動
