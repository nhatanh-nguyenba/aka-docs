# ファイル権限の更新

RCA.Activities.GSuite.UpdateFilePermission

## **説明**

このアクティビティを使用すると、指定したファイルのアクセス許可を更新できます。

![image-20220505160539-1.png](/static/img/ce3b36_image-20220505160539-1.png)

（\*必須）

## **アクティビティ本体内**

* **更新する権限を持つファイルの ID (文字列)**\* - 権限を更新するために要求されたファイルの ID。文字列変数と文字列がサポートされています。
* **更新する権限の ID (文字列)**\* - 更新する権限の ID。

## **プロパティ**

**入力**

* **ファイル ID (文字列)**\* - 許可を取得するために要求されたファイルの ID。文字列変数と文字列がサポートされています。
* **権限 ID (文字列)**\* - 更新する権限の ID。
* **ExpirationTime (DateTime)** - ファイルのアクセス許可を期限切れにする時刻。このプロパティは、< Nullable >DateTime 変数と< Nullable >DateTime 形式の値をサポートします。値が指定されていない場合、権限は有効期限なしで更新されます。
* **ロール (ドロップダウン リスト)** - エンティティに付与するロールを指定するドロップダウン リスト。使用可能なプロパティ値は次のとおりです。
**・リーダー** - エンティティはファイルを読み取り、ダウンロードできます。
**・ライター** - エンティティはファイルを編集できます。
**・所有者** - エンティティはファイルを所有します。
**・コメント者** - エンティティはファイルにコメントできます。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [65623834] ファイル権限の更新

**オプション**

* **ドメイン管理者アクセスを使用する (チェックボックス)** - 選択すると、要求者がファイルが属するドメインの管理者であればアクセスが許可されます。
* **RemoveExpiration (チェックボックス)** - 選択すると、以前の権限の有効期限が削除されます。

**出力**

* **結果 (文字列)** - 更新された権限。このプロパティは Google Permission 変数をサポートします
