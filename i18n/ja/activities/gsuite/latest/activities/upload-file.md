# ファイルをアップロードする

RCA.Activities.GSuite.UploadFile

## **説明**

このアクティビティは、Google ドライブにファイルをアップロードするために指定されています。

![image-20220505160610-1.png](/static/img/3bd52f_image-20220505160610-1.png)

（\*必須）

## **アクティビティ本体内**

* **ロード ファイル パス (文字列)**\* - ドキュメントがアップロードされるファイル パス。
例: 「D:\file.txt」

## **プロパティ**

**入力**

* **ローカル ファイル パス (文字列)**\* - ドキュメントがアップロードされるファイル パス。
例: 「D:\file.txt」
* **親 ID (文字列)**\* - ファイルをアップロードする親フォルダーの ID。このプロパティは、文字列変数と文字列形式の値をサポートします。

**その他**

* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [3275687978] ファイルをアップロードする

**オプション**

* **既存の更新 (チェックボックス)** - 選択すると、宛先の場所に既に存在するソース ファイルと同じ名前のファイルは、オートメーションの実行時に置き換えられます。

**出力**

* **結果 (文字列)** - アップロードされたファイルの ID。文字列変数がサポートされています。
