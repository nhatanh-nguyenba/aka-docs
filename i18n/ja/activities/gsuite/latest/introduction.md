# はじめに

GSuite アクティビティ パックには、自動化プロジェクトの作成に使用されるすべてのアクティビティが含まれています。これらのアクティビティにより、ロボットは次のことが可能になります。

* Google Workspace 上のディレクトリやファイルを直接操作し、人間のユーザーが行うようなあらゆるアクションを実行します。
* Gmail でメールを送受信します。
