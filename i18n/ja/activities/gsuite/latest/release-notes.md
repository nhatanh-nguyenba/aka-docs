# リリースノート

**RCA.Activities.Gsuite バージョン 2.2.1.0**

このリリースでは、さらに新しいアクティビティが追加されます。

### **1.  シーツのサブパッケージ**

* 列の追加削除: 指定したシートから空白の列を追加するか、既存の列を削除します。
* 行の追加削除: 指定したシートから空の行を追加するか、既存の行を削除します。
* 新しいシートの追加: 指定した Google スプレッドシート内に新しいシートを作成します
* 行を追加: 既存のスプレッドシートに行を追加します。
* 範囲の自動入力: Google スプレッドシートのユーザー インターフェースから自動入力機能を実行します。
* 範囲のクリア: Microsoft Graph 範囲のクリア API を使用して、指定したシートから範囲の値、書式、塗りつぶし、境界線などをクリアします。
* 範囲をコピーして貼り付け: 指定したセルの範囲をコピーして、指定したシートに貼り付けます。シートからコピーして別のシートに貼り付けることも、同じシート内でコピーして貼り付けることもできます。
* シートをコピー: 指定したワークブックからシートをコピーし、別の指定したワークブックに貼り付けます。他のワークブックが指定されていない場合は、ソース ファイルにコピーされます。
* 範囲の削除: Microsoft Graph 範囲のクリア (**Shift** が *None* の場合) または範囲の削除 (**Shift** が *Up* または *Left* の場合) API を使用して、指定されたブック (**Workbook** ) ワークシート (**SheetName** ) からセル データの範囲 (**Range** ) を削除します。
* シートの削除: 指定したスプレッドシートから指定したシートを削除します。
* スプレッドシートをダウンロード: Google スプレッドシート ファイルのコピーをダウンロードします。
* セルの色の取得: セルの背景色を抽出し、色変数として保存します。
* シートの取得: Microsoft Graph ワークシート取得 API を使用して、指定されたワークブック (**Workbook** ) からワークシートの配列 (**Sheets** ) を取得して返します。
* セルの読み取り: Excel セルの値を読み取り、変数に保存します。 **Excel アプリケーション スコープ** または **Excel ファイルを使用** アクティビティ内でのみ使用できます。
* 列の読み取り:**StartingCell** プロパティ フィールドで指定されたセルで始まる列から値を読み取り、`IEnumerable<Object>`変数に保存します。
* 範囲の読み取り: 範囲の書き込みなど、他のアクティビティで使用するために、指定した Excel ワークブックのセル範囲から情報を読み取り、保存します。
* 行の読み取り:StartingCell フィールドで指定されたセルで始まる行から値を読み取り、それを`IEnumerable<object>`変数に保存します。
* シートの名前を変更: Excel ファイル内のシートの名前を変更します。
* セルに書き込む: セルにテキストまたは数式を入力します。このアクティビティは、親の [Excel ファイルを使用] アクティビティで選択された Excel ファイルまたはプロジェクト ノートブックで使用できます。
* 範囲の書き込み: DataTable 変数を Excel ワークブックの指定されたセル範囲に書き込みます。

### **2.  ドライブサブパッケージ**

新しいスプレッドシートを作成: Google ドライブに新しいスプレッドシートを作成します。
