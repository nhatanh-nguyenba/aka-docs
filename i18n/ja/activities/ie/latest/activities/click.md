# クリック

RCA.Activities.IE.Click

## **説明**

Click アクティビティは、クリック アクションを Web ページに送信するために使用されます。

![image-20220506110125-1.png](/static/img/a909be_image-20220506110125-1.png)

（\*必須）

**アクティビティ本体内**

* **ターゲット要素を選択** - 項目を選択するフィールドを選択します。このアクティビティは、そのフィールドの場所を指定する文字列変数 (セレクター) を生成します。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト): プロセスの実行の継続をブロックします。
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの実行を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
例: 30000

**入力**

* **マウス ボタン (ドロップダウン リスト)** – クリックするアクションを選択します
・LEFT - マウスの左ボタンでクリックする動作を設定します。
・RIGHT – マウスの右ボタンでクリックする動作を設定します。
・MIDDLE – マウスの中ボタンをクリックする動作を設定します。
* **セレクター (文字列)\***- IE ブラウザーでクリックされる要素を識別する文字列。

**オプション**

* **From Elements (IEElement)** - 目的の要素を検索するときに参照される要素。 IEElement 変数タイプのみが受け入れられます。
* **仮想クリック (チェックボックス)** - 仮想クリックが希望のクリック方法であるかどうかを確認します。

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例：[215266456]をクリック

**対象**

* **スクロールの高さ (数値)** - 要素を取得する前にスクロールします。垂直方向にスクロールするピクセル数に等しい数値を入力します。負の値は上にスクロールし、正の値は下にスクロールします。
例: 20
* **Scroll IntoView (チェックボックス)** - 選択すると、画面上の目的の要素まで画面が自動的にスクロールします。
* **スクロール幅 (数値)** - 要素を取得する前にスクロールします。水平方向にスクロールするピクセル数に等しい数値を入力します。負の値は左方向にスクロールし、正の値は右方向にスクロールします。
例: 20
