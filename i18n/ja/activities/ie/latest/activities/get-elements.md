# 要素の取得

RCA.Activities.IE.GetElements

## **説明**

Internet Explorer 上の UI 要素を抽出します。アクティビティは、インターネット カテゴリの [ブラウザを開く] アクティビティ内でのみ有効です。

![image-20220506110422-1.png](/static/img/4b0c8e_image-20220506110422-1.png)

（\*必須）

**アクティビティの本文**

* **ターゲット要素を選択**\* - 項目を取得するフィールドを選択します。このアクティビティは、そのフィールドの場所を指定する文字列変数 (セレクター) を生成します。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト): プロセスの実行の継続をブロックします。

**入力**

* **From Element (IEElement)** - Internet Explorer で類似の要素を検索するために使用される入力要素。 IE 要素タイプの変数のみを受け入れます。
* **最大結果 (Int32)** - このアクティビティから返される要素の最大量。デフォルト: 1。
例: 1
* **セレクター (文字列)\***- 指定された要素を識別する文字列。
* **タイムアウト MS (Int32)** - アクティビティが完了するまで待機する時間 (ミリ秒単位) を指定します。デフォルト: 30000
例: 30000

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [2424234] 要素を取得します。

**出力**

* **要素 (IEElement)** - アクティビティから返される出力 UI 要素。要素の格納には、IEElement 変数タイプの配列のみがサポートされています。
