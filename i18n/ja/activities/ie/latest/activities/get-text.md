# テキストの取得

RCA.Activities.IE.GetText

## **説明**

Get Text アクティビティは、Web ページ上で抽出し、それを String 変数に保存します。

![image-20220506110459-1.png](/static/img/ebb950_image-20220506110459-1.png)

（\*必須）

**アクティビティの本文**

* **ターゲット要素を選択**\* - テキストを取得するフィールドを選択します。このアクティビティは、そのフィールドの場所を指定する文字列変数 (セレクター) を生成します。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト): プロセスの実行の継続をブロックします。
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの実行を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
例: 30000

**入力**

* **セレクター (文字列)**\* - 指定されたフィールドを識別する文字列。

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [71577779] テキストを取得

**オプション**

* **From Element (IEElement)** - 属性の値を取得する要素を識別する変数。 IEElement タイプのみをサポートします。

**出力**

* **テキスト (文字列)**\* - Internet Explorer から取得する要素

**対象**

* **スクロールの高さ (数値)** - 要素を取得する前にスクロールします。垂直方向にスクロールするピクセル数に等しい数値を入力します。負の値は上にスクロールし、正の値は下にスクロールします。
例: 20
* **Scroll Into View (チェックボックス)** - 選択すると、画面上の目的の要素まで画面が自動的にスクロールします。
* **スクロール幅 (数値)** - 要素を取得する前にスクロールします。水平方向にスクロールするピクセル数に等しい数値を入力します。負の値は左方向にスクロールし、正の値は右方向にスクロールします。例: 20
