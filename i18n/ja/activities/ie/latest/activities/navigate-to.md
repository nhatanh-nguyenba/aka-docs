# ナビゲート先

RCA.Activities.IE.NavigateTo

## **説明**

Navigate To アクティビティを使用すると、現在のブラウザーで別の URL に移動できます。

![image-20220506110824-1.png](/static/img/a8d4dd_image-20220506110824-1.png)

（\*必須）

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト): プロセスの実行の継続をブロックします。

**入力**

* **Url (String)\*** - ブラウザーで移動する URL。 URL は引用符で囲む必要があります。
例: https://akabot.com/

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [294859345] に移動します。
