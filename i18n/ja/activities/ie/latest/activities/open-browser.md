# ブラウザを開く

RCA.Activities.IE.OpenBrowser

# 

## **説明**

IE の [ブラウザを開く] アクティビティを使用すると、IE ブラウザを起動し、その中で複数のアクティビティを実行することで、指定した URL に移動できます。

![image-20220506110901-1.png](/static/img/204732_image-20220506110901-1.png)

（\*必須）

**アクティビティ本体内**

* **URL (文字列)** - 指定した URL をコンテナに挿入します。
例：「[https://google.com/”](https://google.com/%E2%80%9D)」
* **Do** - 指定されたブラウザと対話するアクティビティを追加します。

## **プロパティ**

**入力**

* **入力ブラウザ (ブラウザ)**\* - アクティビティがアクティブになる指定されたブラウザを挿入します。
* **URL (文字列)**\* - ブラウザが開く URL を指定します。
例：「[https://google.com/”](https://google.com/%E2%80%9D)」

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [95659056] ブラウザを開く

**オプション**

* **最大化 (チェックボックス)** - このオプションを選択すると、IE ブラウザが最大化します。
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。
例: 30000

**出力**

* **IEBrowser (ブラウザ)** - ブラウザを開いて対話するアクティビティの出力。要求されたすべての情報はブラウザ セッションとして保存できます。ブラウザ変数タイプのみがサポートされます。
