# 項目の選択

RCA.Activities.IE.SelectItem

## **説明**

[項目の選択] アクティビティを使用すると、IE ブラウザのコンボ ボックスまたはドロップダウン リストから項目を選択できます。

![image-20220506111011-1.png](/static/img/1f6857_image-20220506111011-1.png)

**アクティビティの本文**

* **ターゲット要素を選択**\* - 項目を選択するフィールドを選択します。このアクティビティは、そのフィールドの場所を指定する文字列変数 (セレクター) を生成します。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト): プロセスの実行の継続をブロックします。
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの実行を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
例: 30000

**入力**

* **値の選択 (文字列)\*** - 値は、使用する選択の種類に基づきます。引用符で囲む必要があります。
* **セレクター (文字列)**\* - 指定されたフィールドを識別する文字列。

**オプション**

* **From Elements (IEElement)** - 目的の要素を検索するときに参照される要素。 IE要素のみ
* **インデックスによる選択 (チェックボックス)** - 値ではなくインデックスによってリストから項目を選択します。

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例：[282223123] 項目を選択

**対象**

* **スクロール高さ (Int32)**- 要素を取得する前にスクロールします。垂直方向にスクロールするピクセル数に等しい数値を入力します。負の値は上にスクロールし、正の値は下にスクロールします。
例: 20
* **Scroll IntoView (チェックボックス)** - 選択すると、画面上の目的の要素まで画面が自動的にスクロールします。
* **スクロール幅 (Int32)**- 要素を取得する前にスクロールします。水平方向にスクロールするピクセル数に等しい数値を入力します。負の値は左方向にスクロールし、正の値は右方向にスクロールします。
例: 20
