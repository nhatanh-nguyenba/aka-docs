# Web属性の設定

RCA.Activities.IE.SetWebAttribute

## **説明**

指定された Web 属性の値を設定します。

![image-20220506111217-1.png](/static/img/f12f3d_image-20220506111217-1.png)  
（\*必須）

**アクティビティの本文**

* **属性名 (文字列)**\* - 値を設定する属性の名前。このプロパティは String 変数をサポートしているため、引用符で囲む必要があります。
* **属性値 (文字列)**\* - 指定された属性の値。このプロパティは String 変数をサポートしているため、引用符で囲む必要があります。
* **ターゲット要素を選択** - アイテムの属性を設定するフィールドを選択します。このアクティビティは、そのフィールドの場所を指定する文字列変数 (セレクター) を生成します。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト): プロセスの実行の継続をブロックします。
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの実行を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
例: 30000

**入力**

* **属性名 (文字列)**\* - 値を設定する属性の名前。このプロパティは String 変数をサポートしているため、引用符で囲む必要があります。
* **属性値 (文字列)**\* - 指定された属性の値。このプロパティは String 変数をサポートしているため、引用符で囲む必要があります。
* **セレクター (文字列)**\* - 指定された要素を識別する文字列。

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
**例**: [78923423] Web 属性の設定

**オプション**

* **From Element (IEElement)** - 属性の値を取得する要素を識別する変数。 IEElement タイプのみをサポートします。

**対象**

* **スクロールの高さ (Int32)** - 要素を取得する前にスクロールします。垂直方向にスクロールするピクセル数に等しい数値を入力します。負の値は上にスクロールし、正の値は下にスクロールします。
**例**: 20
* **Scroll Into View (チェックボックス)**- 選択すると、画面上の目的の要素まで画面が自動的にスクロールします。
* **スクロール幅 (Int32)**- 要素を取得する前にスクロールします。水平方向にスクロールするピクセル数に等しい数値を入力します。負の値は左方向にスクロールし、正の値は右方向にスクロールします。
**例**: 20
