# 入力してください

RCA.Activities.IE.TypeInto

## **説明**

Type Into アクティビティを使用すると、IE ブラウザのフィールドにテキストを入力できます。

![image-20220506111327-1.png](/static/img/d6966f_image-20220506111327-1.png)

（\*必須）

**アクティビティの本文**

* **ターゲット要素を選択**\* - 項目を選択するフィールドを選択します。このアクティビティは、そのフィールドの場所を指定する文字列変数 (セレクター) を生成します。

## **プロパティ**

**入力前**

* **クリック(IE) (チェックボックス)** - 選択すると、このアクティビティの前にクリック アクションが実行されます。
* **空のフィールド (チェックボックス)** - 選択すると、入力する前にフィールドがクリアされます。
* **フォーカスの設定 (チェックボックス)** - 選択すると、入力する前にフィールドにフォーカスが置かれます。

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト): プロセスの実行の継続をブロックします。
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの実行を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
**例**: 30000

**入力**

* **セレクター (文字列)**\* - IE ブラウザー上で存在を確認する要素を識別する文字列。
* **テキスト (文字列)**\* - 入力する要素
例: 「アカボット」

**オプション**

* **キー間の遅延 (Int32)** - 入力時に各キーの間に時間 (ミリ秒単位) の遅延を設定します。
例: 30000
* **From Elements (IEElement)** - 目的の要素を検索するときに参照される要素。 IEElement 変数タイプのみが受け入れられます。
* **仮想タイプ (チェックボックス)** - 選択すると、仮想タイプ メソッドが実行されます。

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
**例**: [44155135] と入力してください

**対象**

* **スクロールの高さ (Int32)** - 要素を取得する前にスクロールします。垂直方向にスクロールするピクセル数に等しい数値を入力します。負の値は上にスクロールし、正の値は下にスクロールします。
例: 20
* **Scroll IntoView (チェックボックス)** - 選択すると、画面上の目的の要素まで画面が自動的にスクロールします。
* **スクロール幅 (Int32)**- 要素を取得する前にスクロールします。水平方向にスクロールするピクセル数に等しい数値を入力します。負の値は左方向にスクロールし、正の値は右方向にスクロールします。
例: 20
