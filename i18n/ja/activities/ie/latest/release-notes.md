# リリースノート

## **RCA.Activities.IE バージョン 2.1.2.3**

### **修正されたバグ**

* IE 5 と互換性のある Microsoft Edge 環境で、「テキストを取得」アクティビティを使用すると、空の文字列が取得されていました。現在、さまざまなコンポーネント (テキスト ボックス、ボタン、ラベルなど) のテキスト コンテンツを正しく取得します。
* IE 5 と互換性のある Microsoft Edge 環境では、さまざまなレベルの複数のフレームセットで構造化されたウェブサイト上の要素を示すときに、インスペクター インターフェースの「ハイライト」機能を使用すると、コンポーネントの実際の位置と比較してハイライトされた位置が不正確に表示されていました。さらに、「クリック」または「入力」アクティビティを実行すると、実際のコンポーネントの位置からオフセットされた位置を指すため、正確に機能しませんでした。この問題は、コンポーネントの位置を正確にターゲットできるように修正されました。
