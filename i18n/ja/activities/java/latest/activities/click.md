# クリック

RCA.Activities.Java.Click

## **説明**

このアクティビティを使用すると、指定した要素をクリックできます。

[![https://files.readme.io/4d8c2a8-Screenshot_2021-06-03_102748.jpg](/static/img/037630_click.jpg)](/bin/download/Activities/Java/Activities/Click/WebHome/click.jpg?rev=1.1)
（\*必須）

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの実行を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。

**入力**

* **クリック タイプ (DropdownList)** - クリック イベントをシミュレートするときに使用されるマウス クリックのタイプ (LEFTCLICK、LEFTDOUBLECLICK、RIGHTCLICK、RIGHTDOUBLECLICK.) を指定します。デフォルトでは、「左クリック」が選択されています。
* **From (JavaElement)** - 要素を取得するファイル
* **入力要素(JavaElement)\***- 取得する要素。 [セレクター]を指定しない場合は[入力要素]が指定されますが、[入力要素]と[セレクター]を両方入力することはできません。
* **セレクター (文字列) \***- ここにセレクターを追加できます。文字列のみがサポートされます。 [入力要素]を指定しない場合はセレクターが指定されますが、[入力要素]と[セレクター]を両方入力することはできません。
例:

テキスト
`<wnd app="..." ></wnd><java role="..." ></java>`

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: 「Java」をクリックします。
* **OffsetX (Int32)** - [位置] フィールドで選択されたオプションに応じたカーソル位置の水平方向の変位。
* **OffsetY (Int32)** - [位置] フィールドで選択されたオプションに応じたカーソル位置の垂直方向の変位。

**オプション**

* **アニメーション マウス (チェックボックス)** - これを選択すると、アニメーション マウスが画面上に表示されます。
* **仮想クリック (チェックボックス)** - これを選択すると、akaBot がアクションを実行するときにクリックが表示されます。
