# 入力してください

RCA.Activities.Java.TypeInto

## **説明**

この Type Into アクティビティを使用すると、指定した Java ファイルに入力できます。

[![type into.jpg](/static/img/ea68b0_type-into.jpg)](/bin/download/Activities/Java/Activities/Type%20Into/WebHome/type%20into.jpg?rev=1.1)

（\*必須）

**アクティビティの本文**

* **ターゲット要素を選択** - 項目を選択するフィールドを選択します。このアクティビティは、そのフィールドの場所を指定する文字列変数 (セレクター) を生成します。

## **プロパティ**

**入力前**

* **クリック (チェックボックス)** - テキストを入力する前にクリックすることを選択する場合は、チェックを入れます。
* **空のフィールド (チェックボックス)** - テキストを入力する前にフィールドを空にしたい場合は、チェックを入れます。
* **フォーカスの設定 (チェックボックス)** - テキストを入力する前にクリックするようにフォーカスを設定する場合は、チェックを入れます。

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。
* **タイムアウト MS (Int32)** - エラーがスローされるまでにアクティビティの実行を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。

**入力**

* **入力要素 (JavaElement)**\* - 取得する要素。
* **From (JavaElement)** - 要素の取得元のファイル。
* **セレクター (文字列) \***- ここにセレクターを追加できます。文字列のみがサポートされます。

**例**:

テキスト
`<wnd app="..." ></wnd><java role="..." ></java>`

* **テキスト (文字列)** - 入力するテキスト。

**その他**

* **公開 (チェックボックス)** - 公開したい場合にチェックを入れます。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
**例**: Java に入力

**オプション**

* **キー間の遅延 (数値)** - キーストローク間の遅延 (ミリ秒)、最小値は 10、最大値は 1000 です。
* **仮想タイプ (チェックボックス)**: イベントを使用して要素に仮想的に入力する場合にチェックされます。

**出力**

* **成功 (ブール値)** - すべてのアクションが成功したかどうかを示します。
