# リリースノート

## **RCA.Activities.Java バージョン 2.3.0.0**

### **修正されたバグ**

* [GetText] セレクターにテキストが含まれている場合、すべてのデータを表示しません。
* [クリック] クリックタイプ=左クリック、アニメーションマウス=チェックを選択するとマウスポイントを移動できません。
* [クリック] このアクティビティでエラー メッセージが表示される前にアプリが閉じられます。
* [GetText] [セレクター]に不正な形式のデータを入力した場合、「エラー時に続行」プロパティの動作が正しくありません。
* [GetText] Javaアプリケーションでテキストを取得できません。
