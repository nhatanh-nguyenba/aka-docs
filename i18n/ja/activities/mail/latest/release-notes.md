# リリースノート

## **RCA.Activities.Mail バージョン 2.1.1.0**

### **バグ修正**

* メール フォルダー パスにサブフォルダーが存在しない場合にメール メッセージを返す **Outlook メール メッセージの取得** アクティビティの問題を修正しました。
* **Exchange メール メッセージの取得** アクティビティで、代理送信でメールを取得できない問題を修正しました。
* 「To」/「Cc」/「Bcc」リストにセミコロンで区切られたメール アドレスが含まれているメールを送信できない場合、**Outlook メールの送信**、** SMTP メール メッセージの送信** アクティビティでの問題を修正しました。
* **Outlook メール メッセージの取得** アクティビティで、出力 MailMessage の送信者が常に null になる問題を修正しました。
* **SMTP メール メッセージの送信** アクティビティで、件名と本文が空で、MailMessage が一部のメール メッセージを参照している場合に、電子メールが送信者の送信済みフォルダーに保存されない問題を修正しました。
* **SMTP メール メッセージの送信** アクティビティで、件名または本文が空ではなく、MailMessage が何らかのメール メッセージを参照している場合に、予期される受信者にメールが送信されない問題を修正しました。
* **IMAP メール メッセージを取得** アクティビティで常に値 = 0 のメールが返される問題を修正しました。
