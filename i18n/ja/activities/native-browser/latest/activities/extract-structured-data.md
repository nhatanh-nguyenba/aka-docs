# 構造化データの抽出 (NB)

RCA.Activities.NativeBrowser.ExtractStructuredData

## **説明**

「構造化データの抽出」を使用すると、指定した Web ページから構造化データを抽出できます。

![image-20220505145843-1.png](/static/img/98c685_image-20220505145843-1.png)

（\*必須）

## **アクティビティ本体内**

* **ターゲット要素を選択**\* - 存在を確認する要素を選択します。このアクティビティは、その要素の場所を指定する文字列変数 (セレクター) を生成します。

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - このプロパティは、エラーが発生した場合にオートメーションを継続するタイミングを指定します。可能な値は True または False の 2 つだけです。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。
* **Delay After (Int32)** - アクティビティ実行後の遅延時間 (ミリ秒単位)。デフォルトの時間は 300 ミリ秒です。
例: 300
* **Delay Before (Int32)** - アクティビティが操作の実行を開始するまでの遅延時間 (ミリ秒単位)。デフォルトの時間は 300 ミリ秒です。
例: 300

**入力**

* **HTML (文字列) の抽出** - このプロパティを使用すると、抽出する必要がある構造データを含むページ ソースを入力できます。 [HTML 抽出] フィールドと (ネイティブ) ブラウザの両方が入力されている場合、ネイティブ ブラウザからのデータが優先されます。ロボットは、Open Browser または Attach Browser アクティビティの出力よりも、HTML の抽出からの入力を優先します。
* **ブラウザ (ネイティブ ブラウザ)** - 抽出先の既存のブラウザ変数。
* **セレクター (文字列)**\* - アクティビティの実行時に特定の UI 要素を検索するために使用されるテキスト プロパティ。これは実際には、探している GUI 要素とその親の一部の属性を指定する XML フラグメントです。

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。デフォルトはチェックなしです。

**オプション**

* **ページ間の遅延 MS (Int32)** – 次のページが読み込まれるまでの待機時間 (ミリ秒単位)。デフォルト値は 1000 ミリ秒です。ページの読み込み時間が長い場合は、この値を大きくする必要があります。
例: 1000
* **結果の最大数 (Int32)** – 抽出される結果の最大数。値が 0 の場合、識別されたすべての要素が出力に追加されます。デフォルト値は 100 です。
例: 100
* **次のページのタイムアウト MS (Int32)** - 次のページが読み込まれるまでの待機時間 (ミリ秒単位)。タイムアウトが経過すると、ページはそれ以上読み込まれなくなります。デフォルトの時間は 30000 ミリ秒です。入力値が 0 以下の場合、タイムアウトは 30000 (ミリ秒) になります。
例: 30000

**出力**

* **結果 (データ テーブル)** - タイプ = 「DataTable」の出力データ。出力名には空白を使用できません。
