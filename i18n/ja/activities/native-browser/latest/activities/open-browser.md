# ブラウザを開く (注意)

RCA.Activities.NativeBrowser.OpenBrowser

## **説明**

Open Browser (NB) アクティビティを使用すると、ブラウザを起動し、その中で複数のアクティビティを実行することで、指定された URL に移動できます。

![image-20220505150847-1.png](/static/img/c5eb05_image-20220505150847-1.png)

（\*必須）

## **アクティビティ本体内**

* **挿入される URL (文字列)**\* - ブラウザーで開きたい URL。 URL は引用符で囲む必要があります。
例： 「[https://akabot.com/”](https://akabot.com/%E2%80%9D)」
* **Do** - ブラウザ内で実行するアクティビティ。

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - このプロパティは、エラーが発生した場合にオートメーションを継続するタイミングを指定します。可能な値は True または False の 2 つだけです。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。
* **完了時または失敗時にブラウザを終了する (チェックボックス)** - 実行の終了時または失敗時にブラウザを閉じるかどうかを指定します。デフォルト値は true です。これにより、このアクティビティによって開かれたウィンドウ ブラウザのみが閉じられます。

**入力**

* **ブラウザの種類 (ドロップダウン リスト)** - このアクティビティで使用するブラウザの選択。 Chrome、Firefox、Edge の 3 つの選択肢があります。
* **タイムアウト MS (Int32)** - アクティビティの実行を待機する時間 (ミリ秒単位)。タイムアウトが経過すると、ページはそれ以上読み込まれなくなります。デフォルトの時間は 30000 ミリ秒です。
例: 30000
* **URL (文字列)**\* – ブラウザを開いたときにこのリンク先 URL を開始します。
例： 「[https://akabot.com/”](https://akabot.com/%E2%80%9D)」

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。デフォルトはチェックなしです。

**オプション**

* **最大化 (チェックボックス)** - ブラウザのサイズを最大化します。このボックスはデフォルトではオフになっています。
* **プライベート モード (チェックボックス)** - プライベート/シークレット セッションを開くかどうかをチェックします。デフォルト値は False です
* **Zoom Factor (Int32)** - ページの拡大率を変更できます。デフォルト値は 100 です。値が設定されていない場合、ページ拡大率の変更は無視されます。

**出力**

* **出力ブラウザ (NativeBrowser)** - タイプ = 「ブラウザ」のアクティビティの出力。ブラウザセッションに関するすべての情報を保存します。
