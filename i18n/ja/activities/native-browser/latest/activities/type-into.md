# 入力してください (注)

RCA.Activities.NativeBrowser.TypeInto

## **説明**

Type Into アクティビティを使用すると、Web ページ内の任意のテキスト フィールドにテキストを入力できます。

![image-20220505151551-1.png](/static/img/image-20220505151551-1.png)

\* は必須フィールドを示します。

## **アクティビティの本文内**

* **ターゲット要素を選択**\*: ターゲット UI 要素を選択します。このアクションにより、要素を識別するための **Selector** 文字列が生成されます。
* **テキスト (文字列)**\*: ターゲット フィールドに入力するテキスト。引用符で囲む必要があります。
例:`"My name is"`

## **プロパティ**

**入力前**

* **空のフィールド (チェックボックス)**: 入力する前にターゲット フィールドをクリアするかどうかを指定します。
オン: フィールドをクリアします。
チェックなし: 既存のコンテンツを保持します。
* **フォーカスの設定 (チェックボックス)**: 選択すると、入力する前にフィールドにフォーカスを設定します。デフォルトではチェックが入っていません。

**共通**

* **エラー時に続行 (ブール値)**: エラーが発生したときにオートメーションを続行するかどうかを指定します。
True: アクティビティ内でエラーが発生しても実行を継続します。
False (デフォルト): エラーが発生したときにプロセスを停止します。
* **Delay After (Int32)**: アクティビティ実行後の遅延時間 (ミリ秒単位)。デフォルトは`300`です。
例:`300`
* **Delay Before (Int32)**: アクティビティが開始されるまでの遅延時間 (ミリ秒単位)。デフォルトは`300`です。
例:`300`
* **タイムアウト MS (Int32)**: アクティビティの実行を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、実行は停止します。デフォルトは`30000`です。
例:`30000`

**入力**

* **セレクター (文字列)**\*: ターゲット UI 要素を識別するために使用される XML 文字列。
* **表示を待機 (チェックボックス)**: 選択すると、ターゲット要素が表示されるまで待機してから実行します。このオプションはデフォルトで選択されています。
* **テキスト (文字列)**\*: ターゲット フィールドに入力するテキスト。引用符で囲む必要があります。
例:`"My name is"`

**オプション**

* **キー間の遅延 (Int32)**: 各キーストローク間の遅延 (ミリ秒単位)。
例:`200`
* **入力をシミュレート (チェックボックス)**: 選択すると、ターゲット アプリケーションのテクノロジを使用して入力をシミュレートします。この方法は高速であり、バックグラウンドで実行できます。
選択しない場合、入力はハードウェア ドライバーを使用して実行されますが、速度が遅くなり、バックグラウンドで実行できません。
デフォルトでは、このオプションが選択されています。

**その他**

* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データ セキュリティ要件を考慮してください。デフォルトではチェックが入っていません。
