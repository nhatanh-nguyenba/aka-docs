# 待機ページ URL (注)

RCA.Activities.NativeBrowser.WaitPageUrl

## **説明**

「ページ URL を待機」アクティビティは、ページ URL が事前定義された条件に一致するかどうかを検証します。

![image-20220505151834-1.png](/static/img/image-20220505151834-1.png)

\* は必須フィールドを示します。

## **アクティビティの本文内**

* **比較演算子 (ドロップダウン リスト)**: ページ URL を評価する条件を定義します。
**EQUALS** – ページ URL は事前定義された値と正確に一致します。
**CONTAINS** – ページ URL には事前定義された値が含まれています。
**MATCHES** – ページ URL は事前定義された正規表現と一致します。
* **値 (文字列)**\*: ページ URL の検証に使用される事前定義された値。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)**: エラーが発生したときにオートメーションを続行するかどうかを指定します。
True: アクティビティ内でエラーが発生しても実行を継続します。
False (デフォルト): エラーが発生したときにプロセスを停止します。
* **Delay After (Int32)**: アクティビティ実行後の遅延時間 (ミリ秒単位)。デフォルトは`300`です。
例:`300`
* **Delay Before (Int32)**: アクティビティが開始されるまでの遅延時間 (ミリ秒単位)。デフォルトは`300`です。
例:`300`
* **タイムアウト MS (Int32)**: アクティビティの実行を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、実行は停止します。デフォルトは`30000`です。
例:`30000`

**入力**

* **比較演算子 (ドロップダウン リスト)**: ページ URL を評価する条件を定義します。
**EQUALS** – ページ URL は事前定義された値と正確に一致します。
**CONTAINS** – ページ URL には事前定義された値が含まれています。
**MATCHES** – ページ URL は事前定義された正規表現と一致します。
* **値 (文字列)**\*: ページ URL の検証に使用される事前定義された値。

**その他**

* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データ セキュリティ要件を考慮してください。デフォルトではチェックが入っていません。

**出力**

* **結果 (ブール値)**: ページ URL が指定された条件を満たすかどうかを示します。
True: ページ URL が条件を満たします。
False: ページ URL が条件を満たしていません。
