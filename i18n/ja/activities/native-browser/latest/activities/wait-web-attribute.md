# 待機 Web 属性 (NB)

RCA.Activities.NativeBrowser.WaitWebAttribute

## **説明**

待機 Web 属性は、指定された Web 属性の値が有効になるまで待機します。

![image-20220505151917-1.png](/static/img/ee8d45_image-20220505151917-1.png)

（\*必須）

## **アクティビティ本体内**

* **ターゲット要素を選択**\* - ブラウザ上のフィールドを選択して項目を選択します。このアクティビティは、そのフィールドの場所を指定する文字列変数 (セレクター) を生成します。
* **属性名 (文字列)**\* - 選択される属性の式。これは文字列なので、引用符で囲む必要があります。
* **属性値 (文字列)**\* - 指定された属性の値。このプロパティは文字列変数をサポートしているため、引用符で囲む必要があります。

## **プロパティ**

**共通**

* **Continue On Error (Boolean)** - このプロパティは、エラーが発生した場合にオートメーションを継続するタイミングを指定します。可能な値は True または False の 2 つだけです。 True - アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト) - プロセスの実行の継続をブロックします。
* **Delay After (Int32)** - アクティビティ実行後の遅延時間 (ミリ秒単位)。デフォルトの時間は 300 ミリ秒です。
例: 300
* **Delay Before (Int32)** - アクティビティが操作の実行を開始するまでの遅延時間 (ミリ秒単位)。デフォルトの時間は 300 ミリ秒です。
例: 300
* **タイムアウト MS (Int32)** - アクティビティの実行を待機する時間 (ミリ秒単位)。タイムアウトが経過すると、ページはそれ以上読み込まれなくなります。デフォルトの時間は 30000 ミリ秒です。
例: 30000

**入力**

* **属性名 (文字列)**\* - 選択される属性の式。これは文字列なので、引用符で囲む必要があります。
例: 「属性名」
* **属性値 (文字列)**\* - 指定された属性の値。このプロパティは文字列変数をサポートしているため、引用符で囲む必要があります。
例: 「属性値」
* **セレクター (文字列)**\* - アクティビティの実行時に特定の UI 要素を検索するために使用されるテキスト プロパティ。これは実際には、探している GUI 要素とその親の一部の属性を指定する XML フラグメントです。
* **表示を待機 (チェックボックス)** - アクティビティを実行する前にターゲットが表示されるまでオートメーションを待機させる場合は、このボックスをオンにします。これはデフォルトでチェックされています

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。デフォルトはチェックなしです。
