# はじめに

ネイティブ ブラウザ アクティビティ パックには、Web オートメーション プロジェクトの作成に使用されるすべてのアクティビティが含まれています。これらのアクティビティにより、ロボットは次のことが可能になります。

* ブラウザー操作を実行します。
* マウスやキーボードのコマンドの実行、テキストの入力や抽出など、人間の対話をシミュレートします。

以下のように、ネイティブ ブラウザ パッケージのバージョンに対応する、サポートされている適切なマニフェスト バージョンを選択してください。

|  |  |
| --- | --- |
| **ネイティブ ブラウザ パッケージのバージョン** | ** サポートされているマニフェスト** |
| < 5.x.x.x | マニフェスト V2 ***注意してください:*** マニフェスト V2 は 2024 年に非推奨となり、Chrome によって削除されます。将来的にワークフローを自動化するには、Manifest V3 を使用することをお勧めします。 |
| >= 5.x.x.x | マニフェスト V3 |
