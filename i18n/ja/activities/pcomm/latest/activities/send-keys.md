# Send Keys

RCA.Activities.PCOMM.SendKeys

## **説明**

現在のカーソル位置または特定の座標において、テキスト文字列の入力をシミュレートし、オプションで端末にニーモニックコマンドキーを追加します。

![send-keys](/static/img/send-keys.png)

（\*は必須）

## **プロパティ**

**入力**

* **Text (文字列)*** - 端末に入力するテキスト文字列。
* **Reset Screen (ブール値)** - 選択した場合、テキストを入力する前にエミュレーター画面をクリアまたはリセットします。

**オプション**

* **Row (Int32)** - 入力を開始するエミュレーター画面上の行座標。空白のままにすると、現在のカーソル位置から入力を開始します。
* **Column (Int32)** - 入力を開始するエミュレーター画面上の列座標。空白のままにすると、現在のカーソル位置から入力を開始します。
* **Command Key (MnemonicKey)** - テキストの入力後に自動的に実行するオプションの端末制御キー（例: `enter`、`clear`、`tab` など）。デフォルトは `nullchar`（キーは送信されません）です。

**その他**

* **Public (チェックボックス)** - 選択した場合、このアクティビティの実行詳細と変数をログに記録します。機密情報がログに記録される可能性があるため、このオプションを有効にする前にデータ セキュリティ要件を考慮してください。
* **Display Name (文字列)** - デザイナー パネルに表示されるこのアクティビティの名前。ワークフローをより適切に整理および構造化するために、この名前を編集できます。  
  例: [3424325] Send Keys
