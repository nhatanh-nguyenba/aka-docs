# Set Text

RCA.Activities.PCOMM.SetText

## **説明**

PCOMM 画面上の指定された Row（行）と Column（列）の座標に、指定されたテキスト文字列を直接入力します。

![set-text](/static/img/set-text.png)

（\*は必須）

## **プロパティ**

**入力**

* **Text (文字列)*** - エミュレーター画面に入力するテキスト文字列。
* **Reset Screen (ブール値)** - 選択した場合、テキストを入力する前にエミュレーター画面をクリアまたはリセットします。

**オプション**

* **Row (Int32)** - テキストを入力するエミュレーター画面上の行座標。空白のままにすると、テキストは現在のカーソル位置に入力されます。
* **Column (Int32)** - テキストを入力するエミュレーター画面上の列座標。空白のままにすると、テキストは現在のカーソル位置に入力されます。

**その他**

* **Public (チェックボックス)** - 選択した場合、このアクティビティの実行詳細と変数をログに記録します。機密情報がログに記録される可能性があるため、このオプションを有効にする前にデータ セキュリティ要件を考慮してください。
* **Display Name (文字列)** - デザイナー パネルに表示されるこのアクティビティの名前。ワークフローをより適切に整理および構造化するために、この名前を編集できます。  
  例: [3424325] Set Text
