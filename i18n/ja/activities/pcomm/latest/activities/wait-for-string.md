# Wait For String

RCA.Activities.PCOMM.WaitForString

## **説明**

タイムアウト時間内に、エミュレーター画面上の指定された座標に特定のテキスト文字列が表示されるまで、ワークフローの実行を一時停止します。

![wait-for-string](/static/img/wait-for-string.png)

（\*は必須）

## **プロパティ**

**入力**

* **Reset Screen (ブール値)** - 選択した場合、待機を開始する前にエミュレーター画面をクリアまたはリセットします。
* **Column (Int32)** - テキストをチェックするエミュレーター画面上の列座標。空白のままにすると、画面全体がチェックされます。
* **Ignore Case (ブール値)** - 選択した場合、テキスト文字列を検索するときに大文字と小文字の違いを無視します。
* **Row (Int32)** - テキストをチェックするエミュレーター画面上の行座標。空白のままにすると、画面全体がチェックされます。
* **Timeout (Int32)** - エラーがスローされるまでにテキスト文字列が表示されるのを待機する最大時間（ミリ秒単位）。デフォルトは `10000`（10秒）です。
* **Wait For Input Ready (ブール値)** - 選択した場合、文字列を確認する前に端末入力が準備完了状態になるのを待ちます。
* **Wait String (文字列)*** - エミュレーター画面上で待機する予期されるテキスト文字列。

**その他**

* **Public (チェックボックス)** - 選択した場合、このアクティビティの実行詳細と変数をログに記録します。機密情報がログに記録される可能性があるため、このオプションを有効にする前にデータ セキュリティ要件を考慮してください。
* **Display Name (文字列)** - デザイナー パネルに表示されるこのアクティビティの名前。ワークフローをより適切に整理および構造化するために、この名前を編集できます。  
  例: [3424325] Wait For String
