# はじめに

**PCOMM** (IBM Personal Communications) アクティビティ パッケージを使用すると、端末エミュレーション セッション上のタスクを自動化できます。IBM Personal Communications と直接インターフェースすることにより、これらのアクティビティを使用して、メインフレーム システムへの接続、タイピングのシミュレート、画面のナビゲート、テキストの取得、または端末状態の検証を行うことができます。

このパッケージは、手動のユーザー操作や視覚的なスクレイピングに依存せずに、レガシー ホスト システム (IBM 3270 や 5250 など) を自動化するために不可欠です。
