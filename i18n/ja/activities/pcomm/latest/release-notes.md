# リリースノート

## v3.1.0

**機能と機能向上**

* **ユーザー ドキュメントの改善**: アクティビティのプロパティ説明をよりユーザーフレンドリーにし、標準的なワークフロー設計の用語に合わせました。
* **プロパティ ラベルの整合**: ドキュメント内のプロパティ ラベルを更新および標準化し、akaBot Studio のプロパティ パネルのレイアウトに正確に一致させました（`Rows`/`Cols` を `Row`/`Column` に、`Text` を `Wait String` に変更するなど）。
* **ビジュアル ワークフロー**: プロセス設計を容易にするため、主要な端末アクティビティのスクリーンショットと視覚的支援を追加しました。

## v2.1.0

* **初回リリース**: Connection Scope、Send Keys、Move Cursor、Send Command Key、Get Text、Get Text Rect、Set Text、Set Text Rect、Start/Stop Communication、Wait For、および Wait For String を含む、IBM Personal Communications (PCOMM) のコア端末エミュレーション アクティビティをサポートしました。
