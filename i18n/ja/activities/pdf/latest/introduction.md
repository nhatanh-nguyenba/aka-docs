# RCA.Activities.PDF

RCA.Activities.PDF は PDF テキストの読み取り、OCR を使用した読み取り、PDF ページを画像として書き出す、PDF ファイルの結合と分割、PDF ファイルの暗号化／復号、ページ数の取得、および PDF ドキュメントのプロパティ読み取りのアクティビティを提供します。

## アクティビティ

* PDF ファイルの結合
* PDF ファイルの分割
* PDF ページ数の取得
* PDF ページを画像として書き出す
* PDF の復号
* PDF の暗号化
* PDF プロパティの取得
* PDF テキストを読み取る
* OCR で PDF を読み取る
