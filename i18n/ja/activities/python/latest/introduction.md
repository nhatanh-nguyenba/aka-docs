# はじめに

このパックの目的は、任意のワークフローで Python スクリプトとメソッドを製品から直接呼び出せるようにすることです。呼び出されたコードに入力引数を渡したり、アクティビティによって生成された出力データを取得したりすることもできます。

Python スコープ コンテナー アクティビティは、マシンにインストールされている Python 環境に接続し、そのアクティビティ内で残りのアクティビティを使用できるようにします。
PythonObject 変数は、Python データを処理および操作するために特別に設計されています。 Python スクリプトの読み込みアクティビティを使用すると、Python スクリプトを含むファイルのコードを PythonObject に簡単に割り当てることができます。また、Python オブジェクトの取得アクティビティを使用してコードを .NET データ型に変換することもできます。
この強力な変数を使用すると、これらのタスクごとに専用のアクティビティを使用して、スクリプトを実行したり、.NET 型を抽出したり、Python メソッドを呼び出したりすることができます。
