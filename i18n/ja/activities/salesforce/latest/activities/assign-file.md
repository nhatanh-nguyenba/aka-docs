# ファイルの割り当て

RCA.Activities.Salesforce.AssignFile

このアクティビティを使用してレコードにファイルを添付し、カスタマイズされた共有と表示を可能にすることができます。これは操作ステータスをレポートし、Salesforce アプリケーション スコープ内でのみ機能します。

**プロパティ**

**入力**

* **Id:** このフィールドは、Salesforce オブジェクト レコードに添付するファイル ID を指定するための文字列値と文字列変数を受け入れます。
* **レコード ID:** ファイルを添付する Salesforce オブジェクト レコード ID を指定するための文字列値と文字列変数を受け入れます。
* **共有タイプ:** このフィールドは文字列値と文字列変数を受け入れます。ファイルの共有タイプを決定するには、Salesforce でサポートされている有効な値が必要です。 3 つのオプションが利用可能:
  + V：閲覧者
  + C: 協力者
  + I: 内部
* **可視性:** ファイルにアクセスできるユーザー グループを指定します。 3 つのオプションが利用可能:
  + すべてのユーザー: すべてのユーザーがファイルにアクセスできます。
  + 内部ユーザー: 社内のユーザーがファイルにアクセスできます。
  + 共有ユーザー: 共有ユーザーはファイルにアクセスできます。

**その他**

* **DisplayName** - アクティビティの表示名。
* **パブリック** - 選択すると、変数と引数の値が詳細レベルで記録されます。

**オプション**

* **SalesforceStatus:** アクションが完了すると Salesforce から返されるステータス。SalesforceStatus 変数に保存されます。
