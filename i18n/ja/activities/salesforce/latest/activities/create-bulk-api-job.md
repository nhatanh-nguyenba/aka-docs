# 一括APIジョブの作成

RCA.Activities.Salesforce.CreateBulkAPIJob

このアクティビティを使用して、Salesforce で一括 API ジョブを作成できます。

**プロパティ**

**入力**

* **列区切り文字** - ジョブ データを含む .csv ファイル内の列の区切り文字として使用される記号を指定します。デフォルトでは、このプロパティは **COMMA** に設定されています。次のオプションが利用可能です。
  + バッククオート
  + カレット
  + コンマ
  + パイプ
  + セミコロン
  + タブ
* **外部 ID フィールド名** - 更新/挿入されるオブジェクトの外部 ID。この ID は Upsert 操作に必要です。フィールド値は、ジョブ データを含む .csv ファイルにも存在する必要があります。
* **ファイル パス** – ジョブ データを含む .csv ファイルへのフルパス。文字列変数のみがサポートされています。
* **行末 -** ジョブ データのデータ行の終わりを示す行末を指定します。デフォルトでは、このプロパティは **LF** に設定されています。以下の 2 つのオプションが利用可能です。
  + LF - 改行文字。
  + CRLF - 復帰文字とその後に続く改行文字。
* **オブジェクト –** 操作を実行する Salesforce オブジェクト。文字列変数のみがサポートされています。
* **操作** - 実行する操作の種類を指定します。 「挿入」がデフォルト値です。次のオプションが利用可能です。
  + 挿入
  + アップデート
  + アップサート
  + 削除

**その他**

* **パブリック** - 選択すると、変数と引数の値が詳細レベルで記録されます。
* **DisplayName** - アクティビティの表示名。

**オプション**

* **ID** - 一括 API ジョブの ID。文字列変数のみがサポートされています。
* **Salesforce ステータス** – アクションが完了すると Salesforce から返されるステータス。SalesforceStatus 変数に保存されます。
