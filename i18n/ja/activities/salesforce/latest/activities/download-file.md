# ダウンロードファイル

RCA.Salesforce.Salesforce.Activities.DownloadFile

このアクティビティにより、一意の ID に基づいて Salesforce からファイルをダウンロードできるようになります。これは、Salesforce アプリケーション スコープ アクティビティ内でのみ動作するように設計されています。

**アクティビティ本体内**

* **FolderName:** ファイルがダウンロードされるローカル フォルダーへの完全なパスを設定します。ローカル マシン上のファイルに移動して、構成ウィザードを使用して構成することもできます。文字列値と文字列変数のみがサポートされます。

**プロパティ**

**入力**

* **FileName:** ダウンロードしたファイルの任意の名前を指定します。空のままにすると、Salesforce の元のファイル名が保持されます。ファイル拡張子は変更されません。文字列値と文字列変数のみがサポートされます。
* **FolderName:** ファイルがダウンロードされるローカル フォルダーへの完全なパスを設定します。ローカル マシン上のファイルに移動して、構成ウィザードを使用して構成することもできます。文字列値と文字列変数のみがサポートされます。
* **Id:** 一意の ID を使用して、ダウンロードするファイルを識別します。文字列値と文字列変数のみがサポートされます。

**その他**

* **DisplayName** - アクティビティの表示名。
* **パブリック** - 選択すると、変数と引数の値が詳細レベルで記録されます。

**出力**

* **説明:** Salesforce から取得したファイルの説明を提供します。文字列値と文字列変数のみがサポートされます。

* **SalesforceStatus:** アクションが完了すると Salesforce から返されるステータス。SalesforceStatus 変数に保存されます。
* **タイトル:** Salesforce からダウンロードされたファイルのタイトル。文字列値と文字列変数のみがサポートされます。
