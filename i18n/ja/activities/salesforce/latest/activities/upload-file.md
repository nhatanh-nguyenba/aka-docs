# ファイルをアップロードする

RCA.Activities.Salesforce.UploadFile

このアクティビティは Salesforce アプリケーション スコープ アクティビティ内で排他的に動作し、Salesforce 内の新しいファイルまたは既存ファイルの新しいバージョンのアップロードを容易にします。

**プロパティ**

**入力**

* **説明:** アップロードするファイルの説明。文字列値と文字列変数のみがサポートされます。
* **ファイル パス:** アップロードするファイルの完全パスを指定します。構成ウィザードを使用して、ローカル マシン上のファイルを見つけることもできます。文字列値と文字列変数のみがサポートされています
* **Id:** 指定した場合、新しいバージョンで更新する Salesforce 上の既存のファイルの ID を表します。これはオプションであり、空のままにすると新しいファイルがアップロードされます。文字列値と文字列変数のみがサポートされます。
* **タイトル:** アップロードするファイルのタイトルを指定します。文字列値と文字列変数のみがサポートされます。

**その他**

* **DisplayName:** アクティビティに対して表示される名前。
* **パブリック:** 選択すると、変数と引数の値が詳細レベルで記録されます。

**出力**

* **出力 ID:** このフィールドには、Salesforce で作成されたファイルの ID が含まれます。 InputId プロパティで ID が指定されている場合、このフィールドはその ID を反映します。新しいオブジェクトがアップロードされると、新しく生成された ID がここに表示されます。文字列値と文字列変数のみがサポートされます。
* **SalesforceStatus:** アクションが完了すると Salesforce から返されるステータス。SalesforceStatus 変数に保存されます。
* **バージョン ID:** このフィールドには、新しく作成されたファイルのバージョンの ID が保持されます。文字列値と文字列変数のみがサポートされます。
