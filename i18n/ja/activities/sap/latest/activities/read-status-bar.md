# ステータスバーの読み取り

RCA.Activities.Common.ReadStatusBar

このアクティビティを使用して、SAP GUI ウィンドウの下部にあるステータス バーに表示されるメッセージを読み取ることができます。

**アクティビティ本体内**

* **画面上に表示ボタン –** ステータス バー要素を画面上に直接表示できます。指定すると、セレクターの値が対応するフィールドに表示されます。

**プロパティ**

**共通**

* **ContinueOnError** - アクティビティでエラーが発生した場合でも自動化を続行するかどうかを指定します。ブール値 (True、False) のみがサポートされます。デフォルト値は False です。その結果、値が空白または False でエラーがスローされた場合、プロジェクトの実行は停止します。値が True に設定されている場合、プロジェクトはエラーに関係なく実行を続けます。

**注:** このアクティビティが ** Try Catch** に含まれており、** ContinueOnError** プロパティの値が True の場合、プロジェクトの実行時にエラーは捕捉されません。

**入力**

* **Target.CvElement –** アクティビティの実行時にコンピューター ビジョンを使用して特定の要素を検索するために使用します。
* **Target.CvType** – コンピューター ビジョンを使用して識別された要素コントロールのタイプを示します。
* **Target.Element** – アクティビティの実行時に UIFramework を使用して特定の要素を検索するために使用しますが、このアクティビティはコンピューター ビジョン メソッドをサポートしていません。
* **Target.FuzzySelector** – アクティビティの実行時にあいまい検索を使用して特定の要素を検索するために使用しますが、このアクティビティはコンピューター ビジョン メソッドをサポートしていません。
* **Target.SearchSteps** – 要素の識別に使用するセレクターのタイプ。このアクティビティは、Strict Selector と Fuzzy Selector の 2 種類のセレクター タイプのみをサポートしており、他のタイプはサポートされていません。
* **Target.Selector –** アクティビティの実行時に特定の UI 要素を検索するために使用します。これは実際には、探している GUI 要素の属性とその親の一部を指定する XML フラグメントです。
* **Target.Timeout MS –** エラーをスローする前にログインの成功を待機する時間 (ミリ秒単位) を指定します。デフォルト値は 30000 ミリ秒です。
* **Target.Wait For Ready -** アクションを実行する前に、ターゲット要素が準備完了になるのを待ちます。 3 つの利用可能なオプション:
  + なし: アクションを実行する前に、ターゲット UI 要素以外のものが存在するのを待機しません。
  + インタラクティブ: 実際にアクションを実行する前に、ターゲット アプリ内のすべての UI 要素が存在し、インタラクティブになるまで待機します。
  + Complete: 実際にアクションを実行する前に、ターゲット アプリ内のすべての UI 要素が存在するのを待ちます。

**その他**

* **DisplayName** - アクティビティの表示名。
* **パブリック** - 選択すると、変数と引数の値が詳細レベルで記録されます。

**出力**

* **メッセージ データ** – メッセージのステータスのデータを変数に保存するために使用されます。文字列配列変数のみがサポートされています。
* **メッセージ ID** – メッセージの ID を変数に保存するために使用されます。文字列変数のみがサポートされています。
* **メッセージ番号** – メッセージの番号を変数に保存するために使用されます。文字列変数のみがサポートされています。
* **メッセージ テキスト** – メッセージ テキスト全体を変数に保存するために使用されます。文字列変数のみがサポートされています。
* **メッセージ タイプ** – メッセージ タイプを変数に保存するために使用されます。文字列変数のみがサポートされています。サポートされているメッセージのタイプは次の 3 種類です。
  + **S** - 成功
  + **W** - 警告
  + **E** - エラー
