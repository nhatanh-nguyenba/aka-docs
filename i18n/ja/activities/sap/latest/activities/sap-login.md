# SAP ログイン

**SAP ログイン**

RCA.Activities.Common.SAPLogin

このアクティビティを使用して、SAP システムに直接ログインできます。

![1714793840785-992.png](/static/img/1d2608_1714793840785-992.png)

**アクティビティの本文:**

* **クライアント** - ログインする SAP クライアント番号。テキストは引用符で囲む必要があります。
* **ユーザー名 -** SAP にログインするためのユーザー名。テキストは引用符で囲む必要があります。
* **パスワード** – SAP にログインするためのパスワード。このパスワードを使用するには、チェックイン オプション「安全です」を削除する必要があります。テキストは引用符で囲む必要があります。
* **安全なパスワード** – [パスワード]フィールドに書き込まれる安全なテキスト。このテキストを使用するには、「安全です」オプションをチェックする必要があります。
* **言語** - SAP が画面、メニュー、フィールドの表示に使用する言語。テキストは引用符で囲む必要があります。
* **複数のログオン オプション** – 同じアカウントに同時に複数のログオンがある場合は、以下の 3 つのオプションのいずれかを選択して続行方法を指定する必要があります。
  + シングル - このログオンを続行し、他のログオンを終了します。
  + 複数 - 他のログオンを終了せずに、このログオンを続行します。
  + 終了 - このログオンを終了します。

**プロパティ**

**共通**

* **ContinueOnError** - アクティビティでエラーが発生した場合でも自動化を続行するかどうかを指定します。ブール値 (True、False) のみがサポートされます。デフォルト値は False です。その結果、値が空白または False でエラーがスローされた場合、プロジェクトの実行は停止します。値が True に設定されている場合、プロジェクトはエラーに関係なく実行を続けます。

**注:** このアクティビティが ** Try Catch** に含まれており、** ContinueOnError** プロパティの値が True の場合、プロジェクトの実行時にエラーは捕捉されません。

* **タイムアウト MS** - ログインが成功するまで待機する時間 (ミリ秒単位) を指定します。この時間を経過するとエラーがスローされます。デフォルト値は 5000 ミリ秒です。

**入力**

* **クライアント** - ログインする SAP クライアント番号。テキストは引用符で囲む必要があります。
* **言語** - SAP が画面、メニュー、フィールドの表示に使用する言語。テキストは引用符で囲む必要があります。
* **パスワード** – SAP にログインするためのパスワード。このパスワードを使用するには、チェックイン オプション「安全です」を削除する必要があります。テキストは引用符で囲む必要があります。
* **安全なパスワード** – [パスワード]フィールドに書き込まれる安全なテキスト。このテキストを使用するには、「安全です」オプションをチェックする必要があります。
* **ユーザー名 -** SAP にログインするためのユーザー名。テキストは引用符で囲む必要があります。

**その他**

* **パブリック** - 選択すると、変数と引数の値が詳細レベルで記録されます。
* **DisplayName** - アクティビティの表示名。

**オプション**

* **IsSecure** - 選択すると、** Secure** ** Password** フィールドが SAP のパスワード フィールドへの入力に使用され、それ以外の場合は ** Password** フィールドが使用されます。
* **複数のログオン オプション** – 同じアカウントに同時に複数のログオンがある場合は、以下の 3 つのオプションのいずれかを選択して続行方法を指定する必要があります。
  + シングル - このログオンを続行し、他のログオンを終了します。
  + 複数 - 他のログオンを終了せずに、このログオンを続行します。
  + 終了 - このログオンを終了します。
