# SAP ログオン

**SAP ログオン**

RCA.Activities.Common.SAPLogon

このアクティビティを使用して、SAP システムに直接ログオンできます。

![1714791370040-272.png](/static/img/5443fe_1714791370040-272.png)

**アクティビティの本文:**

* **SAP ログオン パス** - コンピュータ上の SAP ログオン プログラムへのパス。 テキストは引用符で囲む必要があります。
* **接続名** - SAP システムへのログオンに使用される SAP ログオン ウィンドウに表示される SAP 接続名。テキストは引用符で囲む必要があります。

**プロパティ**

**共通**

* **ContinueOnError** - アクティビティでエラーが発生した場合でも自動化を続行するかどうかを指定します。ブール値 (True、False) のみがサポートされます。デフォルト値は False です。その結果、値が空白または False でエラーがスローされた場合、プロジェクトの実行は停止します。値が True に設定されている場合、プロジェクトはエラーに関係なく実行を続けます。

**注:** このアクティビティが ** Try Catch** に含まれており、** ContinueOnError** プロパティの値が True の場合、プロジェクトの実行時にエラーは捕捉されません。

**入力**

* **接続名** - SAP システムへのログオンに使用される SAP ログオン ウィンドウの正確な SAP 接続名。
* **NumberOfRetries** - アクティビティが SAP へのログオンを試行する回数 (デフォルトでは 5)。
* **RetryInterval** - ログオンを再試行する間隔の時間 (ミリ秒単位)。デフォルトの時間は 5000 ミリ秒です。
* **SAP ログオン パス** - コンピュータ上の SAP ログオン プログラムへのパス。

**その他**

* **パブリック** - 選択すると、変数と引数の値が詳細レベルで記録されます。
* **DisplayName** - アクティビティの表示名。

**出力**

* **SAP ログイン ウィンドウ** - Window 変数で新しく開かれた SAP ログイン ウィンドウ。
