# メニュー項目の選択

**SAP 選択メニュー項目**

RCA.Activities.Common.Select メニュー項目

このアクティビティを使用して、SAP アプリケーションのメニュー項目を選択できます。 SAP GUI ウィンドウからメニューを指定すると、使用可能なすべてのメニュー項目のリストがアクティビティに表示され、選択できます。

**アクティビティ本体内**

* **画面上に表示ボタン –** 画面上に要素を直接表示できます。指定すると、セレクターの値が対応するフィールドに表示されます。
* **メニュー** – SAP GUI ウィンドウのメニューを指定します。 SAP GUI ウィンドウからメニューを指定すると、使用可能なすべてのメニュー項目のリストがアクティビティに表示され、選択できます。

**プロパティ**

**共通**

* **ContinueOnError** - アクティビティでエラーが発生した場合でも自動化を続行するかどうかを指定します。ブール値 (True、False) のみがサポートされます。デフォルト値は False です。その結果、値が空白または False でエラーがスローされた場合、プロジェクトの実行は停止します。値が True に設定されている場合、プロジェクトはエラーに関係なく実行を続けます。

**注:** このアクティビティが ** Try Catch** に含まれており、** ContinueOnError** プロパティの値が True の場合、プロジェクトの実行時にエラーは捕捉されません。

* **DelayAfter** – アクティビティの実行後の遅延時間をミリ秒単位で指定します。デフォルトの時間は 300 ミリ秒に設定されています。
* **DelayBefore** - アクティビティに先立ち、操作が開始されるまでの期間 (ミリ秒単位)。デフォルト時間は 200 ミリ秒です。

**入力**

* **メニュー** – SAP GUI ウィンドウのメニューを指定します。 SAP GUI ウィンドウからメニューを指定すると、使用可能なすべてのメニュー項目のリストがアクティビティに表示され、選択できます。
* **Target.CvElement –** アクティビティの実行時にコンピューター ビジョンを使用して特定のメニュー項目を検索するために使用しますが、このアクティビティはコンピューター ビジョン メソッドをサポートしていません。
* **Target.CvType** – コンピューター ビジョンを使用して識別されたメニュー項目のタイプを示しますが、このアクティビティはコンピューター ビジョン メソッドをサポートしていません。
* **Target.Element** – アクティビティの実行時に UIFramework を使用して特定のメニュー項目を検索するために使用します。
* **Target.FuzzySelector** – アクティビティの実行時にあいまい検索を使用して特定のメニュー項目を検索するために使用します。
* **Target.SearchSteps** – メニュー項目の識別に使用するセレクターのタイプ。このアクティビティは、Strict Selector と Fuzzy Selector の 2 種類のセレクター タイプのみをサポートしており、他のタイプはサポートされていません。
* **Target.Selector –** アクティビティの実行時に特定のメニュー項目を検索するために使用します。これは実際には、探している GUI 要素の属性とその親の一部を指定する XML フラグメントです。
* **Target.Timeout MS –** エラーをスローする前にログインの成功を待機する時間 (ミリ秒単位) を指定します。デフォルト値は 30000 ミリ秒です。
* **Target.Wait For Ready -** アクションを実行する前に、ターゲット メニュー項目が準備完了になるのを待ちます。 3 つの利用可能なオプション:
  + なし: アクションを実行する前に、ターゲット ボタン以外の何も待機しません。
  + インタラクティブ: 実際にアクションを実行する前に、ターゲット アプリ内のすべての UI 要素が存在し、インタラクティブになるまで待機します。
  + Complete: 実際にアクションを実行する前に、ターゲット アプリ内のすべての UI 要素が存在するのを待ちます。

**その他**

* **DisplayName** - アクティビティの表示名。
* **パブリック** - 選択すると、変数と引数の値が詳細レベルで記録されます。

**オプション**

* **AlterIfDisabled** - 値がある場合、指定されたボタンが無効であっても、選択された項目に対するアクションが実行されます。
