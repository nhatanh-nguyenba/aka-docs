# クリアコレクション

`System.Activities.Statements.ClearCollection<System.Int32>`

## **説明**

このアクティビティでは、すべてのアイテムの指定されたコレクションをクリアできます。

![image-20220505152501-1.png](/static/img/9d9c08_image-20220505152501-1.png)  
（\*必須）

## **プロパティ**

**その他**

* **コレクション (ICollection&lt;T&gt;)**\* - ここにクリアしたいコレクションを入力できます。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例：「クリアコレクション」
* **公開 (チェックボックス)** - 公開する場合にチェックを入れます。使用する前に、データ セキュリティ要件を必ず考慮してください。
* **TypeArgument (ドロップダウン リスト)** - コレクションの型引数とクリアする項目。使用するコレクションのデータ型に基づいて、さまざまなオプションを選択できます。
**注意:** TypeArgument が設定されると、コレクション プロパティの変数タイプはそれに応じて自動的に更新され、その変数タイプのみを入力として受け入れます。
