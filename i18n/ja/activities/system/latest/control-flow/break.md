# 休憩

RCA.Activities.Core.Break

## **説明**

このアクティビティを使用すると、現在のワークフローの実行を中断できます。これは「For each」ループ内でのみ使用できます。

![image-20220505092118-1.jpeg](/static/img/98a2d1_image-20220505092118-1.jpeg)

## **プロパティ**

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [997394815] ファイルの移動
* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
