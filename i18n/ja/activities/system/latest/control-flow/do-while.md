# しながら行う

System.Activities.Statements.DoWhile

## **説明**

このアクティビティを使用すると、条件が満たされる限り一連のアクティビティを実行するループを作成できます。
**Body** 内のアクティビティが最初に実行され、次に条件が評価されます。ループは、条件が満たされなくなるまで継続します。

![Core_DoWhile](/static/img/do-while.png)

\* は必須フィールドを示します。

## **アクティビティの本文内**

* **本体**: 各反復で実行されるアクティビティを追加できるコンテナー。
* **条件 (ブール値)**\*: 本体の各実行後、次の反復前に評価される条件。
例:`var_input < 15`

## **プロパティ**

**その他**

* **条件 (ブール値)**\*: ループが継続するかどうかを決定するために各反復後に評価される条件。
例:`var_input < 15`
* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`[402352067] Do While`
* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データのセキュリティを考慮してください。
