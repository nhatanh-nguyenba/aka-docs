# それぞれについて

RCA.Activities.Core.ForEach

## **説明**

このアクティビティを使用すると、配列、リスト、DataTable、またはその他の種類のコレクションを反復処理できます。

![image-20220513134113-1.png](/static/img/if.png)

\* は必須フィールドを示します。

## **プロパティ**

**入力**

* **値 (IEnumerable)**\*: 反復処理するコレクション。コレクション内の各アイテムは個別に処理されます。変数または直接値をサポートします。

**その他**

* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`[389861710] For Each <T>`
* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データのセキュリティを考慮してください。
* **型引数 (ドロップダウン リスト)**\*: コレクション内の項目のデータ型を指定します。デフォルト値は`Object`です。

**出力**

* **現在のインデックス (Int32)**: コレクション内の現在位置を示す 0 から始まるインデックス。
