# もし

System.Activities.Statements.If

## **説明**

このアクティビティを使用すると、条件 (True/False) に基づいて簡単な決定を実行できます。
これは、条件 (**IF** ) と 2 つの分岐で構成されます。
- **THEN**: 条件が True の場合に実行されます
- **ELSE**: 条件が False の場合に実行されます。

![image-20220513134317-1.png](/static/img/if.png)

\* は必須フィールドを示します。

## **アクティビティの本文内**

* **条件 (ブール値)**\*: 評価する条件。
例:`varA = varB`
* **その後**: 条件が True の場合に実行されるアクティビティ。
* **Else**: 条件が False の場合に実行されるアクティビティ。

## **プロパティ**

**その他**

* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`[392071310] If`
* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データのセキュリティを考慮してください。
* **条件 (ブール値)**\*: 2 つの分岐のうちの 1 つを実行する前に評価する条件。
例:`varA = varB`
