# パラレル

System.Activities.Statements.Parallel

## **説明**

このアクティビティを使用すると、複数の子アクティビティを同時に非同期で実行できます。
アクティビティが実行されると、子アクティビティごとに個別のジョブが作成されます。

子プロセスが呼び出されても見つからない場合は、プロセス名とともにエラーが記録されます。
子プロセスの開始後にエラーが発生した場合、親プロセスは影響を受けずに実行を継続します。

\* は必須フィールドを示します。

## **プロパティ**

**その他**

* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データのセキュリティを考慮してください。
* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`Parallel`
* **完了条件 (ブール値)**\*: アクティビティがいつ完了するかを決定する条件を指定します。
例:`varA = varB`
