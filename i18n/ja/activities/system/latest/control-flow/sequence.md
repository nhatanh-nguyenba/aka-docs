# シーケンス

System.Activities.Statements.Sequence

## **説明**

シーケンスは、事前定義された順序で順次実行される一連のアクティビティを編成するために使用されるコンテナです。

![image-20230417141912-1.png](/static/img/sequence.png)

\* は必須フィールドを示します。

## **プロパティ**

**その他**

* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データのセキュリティを考慮してください。
* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`[552074308] Sequence`
