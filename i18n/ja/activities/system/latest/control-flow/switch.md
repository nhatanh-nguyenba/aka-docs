# スイッチ

System.Activities.Statements.Switch

## **説明**

指定された式に基づいて、複数の選択肢から 1 つのオプションを選択できます。

たとえば、2 つの Int32 変数 **A** と **B** を比較すると、次の 3 つの結果が考えられます。
- A > B
- B > A
- A = B

Switch アクティビティを使用して、結果ごとに異なるアクションを定義できます。
通常、式は整数ですが、プロパティで型を変更できます。

![Core_Switch](/static/img/switch.png)

\* は必須フィールドを示します。

## **アクティビティの本文内**

* **式**\*: どのケースを実行するかを決定するために使用される値または条件。
* **ケース / デフォルト**: さまざまなケースと対応するアクションを定義します。
式に一致する大文字と小文字がない場合は、**デフォルト** が実行されます。

## **プロパティ**

**その他**

* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`[406493348] Switch`
* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データのセキュリティを考慮してください。
* **式**\*: どのケースを実行するかを決定するために使用される値または条件。
* **型引数 (ドロップダウン リスト)**: 式のデータ型を指定します。
