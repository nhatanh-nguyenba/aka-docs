# その間

System.Activities.Statements.While

## **説明**

指定された条件が True の間、一連のアクティビティを繰り返し実行できます。
このアクティビティは、条件に基づいてアクションを複数回実行する必要がある場合に便利です。

![image-20220513134736-1.png](/static/img/while.png)

\* は必須フィールドを示します。

## **アクティビティの本文内**

* **条件 (ブール値)**\*: 各反復の前に評価される条件。
例:`counter < 15`
* **Body**\*: 各反復で実行されるアクティビティを追加できるコンテナー。

## **プロパティ**

**その他**

* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`[410342694] While`
* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データのセキュリティを考慮してください。
* **条件 (ブール値)**\*: ループの実行を制御する条件。
例:`var_counter < 15`– ボディ内のアクティビティは、条件が False になるまで繰り返されます。
