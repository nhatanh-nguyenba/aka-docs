# 再スロー

System.Activities.Statements.Rethrow

## **説明**

Rethrow アクティビティを使用すると、最初にポップアウトしなかった場合に、検索している例外の種類を再スローできます。このアクティビティは、Try Catch アクティビティの Catch ブロックで使用できます。

![https://files.readme.io/490ac01-Screenshot_2021-06-01_165846.jpg](/static/img/bcc71f_490ac01-screenshot_2021-06-01_165846.jpg)

## **プロパティ**

**その他**

* **公開** - 公開するかどうかをチェックします。使用する前に、データのセキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
