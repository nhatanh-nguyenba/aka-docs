# 投げる

System.Activities.Core.Throw

## **説明**

Throw アクティビティを使用すると、例外の種類を入力すると、例外をスローして検索できます。このアクティビティは、Try Catch アクティビティの Catch ブロックと Try ブロックで使用できます。

![image-20220506112428-1.png](/static/img/41b49d_image-20220506112428-1.png)  
（\*必須）

## **プロパティ**

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例：投げる。
* **例外をスローする (例外)**\* - スローする例外のタイプ。
例: new\_Exception(「何か問題が発生しました」)。
