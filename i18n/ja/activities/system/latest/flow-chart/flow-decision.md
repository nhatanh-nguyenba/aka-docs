# フローの決定

System.Activities.Statements.FlowDecision

## **説明**

このアクティビティは、設定した条件に応じて 2 つの分岐のうち 1 つを実行します。通常は True と False の 2 つの分岐がありますが、プロパティ パネルでタイトルを変更できます。これはフローチャート内でのみ使用でき、If アクティビティと同等です。

## **プロパティ**

**その他**

* **条件 (ブール値)**\*- 2 つの分岐のうちの 1 つの前に実行される条件。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [169542635] フロー決定
* **True Label (String)**\* - 条件が満たされた場合の説明を提供できます。このラベルでは、文字列を引用符で囲む必要はありません。
例: 正しい
* **False Label (String)**\* - 条件が満たされない場合の説明を提供できます。このラベルでは、文字列を引用符で囲む必要はありません。
例: 間違っています
