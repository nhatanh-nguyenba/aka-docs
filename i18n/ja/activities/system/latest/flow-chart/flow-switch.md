# フロースイッチ

System.Activities.Statements.FlowSwitch

## **説明**

このアクティビティを使用すると、制御フローを 3 つ以上の分岐に分割し、各分岐を指定された条件に基づいて実行できます。

## **プロパティ**

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [387438076] フロースイッチ
* **式**\*- akaBot に実行させるワークフローの新しいブランチに対して評価される特定の条件。
デフォルトでは、サポートされる変数は Object です。変数の型を変更するには、TypeArgument プロパティで対応するオプションを選択してください。
* **引数の型 (ドロップダウン リスト)** - 式プロパティに追加できるさまざまな種類のステートメントを選択できます。
