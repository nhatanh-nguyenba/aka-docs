# 書き込み行

System.Activities.Statements.WriteLine

## **説明**

文字列値を出力パネルに出力できるようにします2

![image-20220506113102-1.png](/static/img/37c8c9_image-20220506113102-1.png)

（\*必須）

**アクティビティ本体内**

* **テキスト (文字列)**\* - [出力] パネルに表示するメッセージ。テキスト変数と文字列変数の両方を受け入れます。

例: 「この変数を表示」 + intName.toString。

## **プロパティ**

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: 書き込み行
* **テキスト (文字列)**\* - [出力] パネルに表示するメッセージ。
* **テキスト ライター (TextWriter)** - このボックスはオプションです。 Text プロパティのコンテンツを、デフォルトの出力ペインとは別のターゲットに送信します。
