# 州

System.Activities.Statements.State

## **説明**

このアクティビティはステート マシンで使用され、入口、出口、遷移の 3 つのセクションが含まれています。ダブルクリックしてアクティビティを追加します。

![image-20220506144248-1.png](/static/img/edecc8_image-20220506144248-1.png)

（\*必須）

## **アクティビティ本体内**

* **エントリ**\* – 選択した州のエントリ アクティビティを追加する領域。
* **Exit**\* - 選択した状態の終了アクティビティを追加する領域。
* **トランジション\*** - 選択したステートにリンクされたすべてのトランジションを表示する領域。

## **プロパティ**

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [571047923] 状態
