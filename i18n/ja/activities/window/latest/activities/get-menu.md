# メニューを取得

RCA.Activities.Windows.GetMenu

## **説明**

Get Menu アクティビティを使用すると、指定したウィンドウ要素のメニューを選択できます。

![image-20220505170004-1.png](/static/img/1d8327_image-20220505170004-1.png)

（\*必須）

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの可能な値があります。
  - True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
  - False: プロセスの実行の継続をブロックします。

**入力**

* **コンテキスト メニュー名 (文字列)**- メニューの名前。デフォルト値は「メニュー」です。
例: メニュー
* **Is Context Menu (Boolean)** - メニューがコンテキスト メニューであるかどうかを指定します。
* **ウィンドウを使用する (WindowApp)**\* - メニューを取得するウィンドウ。このプロパティでは、WindowApp 変数を使用できます。

**その他**

* **公開 (チェックボックス)** - アクティビティを公開する場合にチェックを入れます。このプロパティを使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例:[23232] メニューの取得

**出力**

* **出力メニュー (WindowApp)\***- 出力メニュー。このプロパティは、type = 'Menu' の変数をサポートします。
