# プロパティの取得

RCA.Activities.Windows.GetProperty

## **説明**

Get Property アクティビティを使用すると、ウィンドウ要素の指定されたプロパティの値を取得できます。

![image-20220505171020-1.png](/static/img/61bf81_image-20220505171020-1.png)

（\*必須）

## **アクティビティ本体内**

* **ターゲット要素を選択**\* - プロパティの値を取得する要素を選択します。このアクティビティは、その要素の場所を指定する文字列変数 (セレクター) を生成します。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの可能な値があります。
  - True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
  - False: プロセスの実行の継続をブロックします。
* **TimeOut MS (Int32)** - アクティビティの実行を待機する時間 (ミリ秒単位)。デフォルトの時間は 30000 ミリ秒です。
例: 30000
* **Delay After (Int32)** - アクティビティ実行後の遅延時間 (ミリ秒単位)。
例: 300
* **Delay Before (Int32)** - アクティビティが操作の実行を開始するまでの遅延時間 (ミリ秒単位)。
例: 300

**入力**

* **プロパティ名 (文字列)**\* - 値を取得するプロパティの名前。
例: 見通し
* **セレクター (文字列)**\* - ウィンドウの位置を識別する文字列。

**その他**

* **公開 (チェックボックス)** - アクティビティを公開する場合にチェックを入れます。このプロパティを使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [123123] プロパティの取得

**オプション**

* **オートメーション ベース (ドロップダウン リスト)** - オートメーション ライブラリのバージョンを指定します。このオプションは、Process 引数と一緒に使用します。
* **要素 (AutomationElement)** - セレクター文字列の代わりに、type = ‘AutomationElement’ の変数を入力として使用します。
* **From (UiElement)** - セレクター文字列で要素を検索する場合、ルートへの UiElement 変数 (入力されていない場合は、デスクトップのルートから検索します)。
* **ウィンドウを使用する (WindowApp)** - オープン ウィンドウまたはウィンドウ スコープからの出力変数を使用して、アクティビティのスコープを設定します (From と同様)。

**出力**

* **出力値 (文字列)** - プロパティ値の変数名は type = 'String' で返されます。出力名には空白を使用できません。

**待ってください**

* **無効化を待機 (チェックボックス)** - 要素が操作できなくなるまで待機します。
* **有効化を待機 (チェックボックス)** - 要素が対話できるようになるまで待機します。
* **消滅を待つ (チェックボックス)** - 要素が消滅するのを待ちます。
