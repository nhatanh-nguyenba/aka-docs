# スクロール

RCA.Activities.Windows.Scroll

## **説明**

スクロールアクティビティを使用すると、上下にスクロールできます

![image-20220505170247-1.png](/static/img/408093_image-20220505170247-1.png)

（\*必須）

## **アクティビティ本体内**

* **ターゲット要素を選択**\* - 項目を選択するフィールドを選択します。このアクティビティは、そのフィールドの場所を指定する文字列変数 (セレクター) を生成します。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの可能な値があります。
  - True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
  - False: プロセスの実行の継続をブロックします。
* **TimeOut MS (Int32)** - アクティビティの実行を待機する時間 (ミリ秒単位)。デフォルトの時間は 30000 ミリ秒です。
例: 30000
* **Delay After (Int32)** - アクティビティ実行後の遅延時間 (ミリ秒単位)。
例: 300
* **Delay Before (Int32)** - アクティビティが操作の実行を開始するまでの遅延時間 (ミリ秒単位)。
例: 300

**入力**

* **スクロール タイプ (ドロップダウン リスト)**- スクロールのタイプを指定します
**・By\_Amount** - ウィンドウ内のドキュメントを指定された量だけスクロールします。
**・By\_Percent** - 指定されたパーセントでウィンドウ内のドキュメントをスクロールします。
* **セレクター (文字列)** - ウィンドウの位置を識別する文字列。

**N 量**

* **水平方向の量 (ドロップダウン リスト)** - 選択するドロップリスト。 N マウントが選択されている場合、N パーセントは 0 になることに注意してください。
* **垂直量 (ドロップダウン リスト)** - 選択するドロップリスト。 N マウントが選択されている場合、N パーセントは 0 になることに注意してください。

**N パーセント**

* **水平パーセント (ダブル)**\* - N パーセントを選択した場合、N 金額は NoAmount になることに注意してください。
例: 10.5
* **垂直パーセント (ダブル)\***- N マウントが選択されている場合、N 量は NoAmount になることに注意してください。
例: 10.5

**オプション**

* **オートメーション ベース (ドロップダウン リスト)** - オートメーション ライブラリのバージョンを指定します。このオプションは、Process 引数と一緒に使用します。
* **要素 (AutomationElement)** - セレクター文字列の代わりに、type = ‘AutomationElement’ の変数を入力として使用します。
* **From (UiElement)** - セレクター文字列で要素を検索する場合、ルートへの UiElement 変数 (入力されていない場合は、デスクトップのルートから検索します)。
* **ウィンドウを使用する (WindowApp)** - オープン ウィンドウまたはウィンドウ スコープからの出力変数を使用して、アクティビティのスコープを設定します (From と同様)。

**その他**

* **公開 (チェックボックス)** - アクティビティを公開するかどうかをチェックします。このプロパティを使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [57483..] スクロール

**待ってください**

* **無効になるまで待機 (チェックボックス)** - 要素が操作できなくなるまで待機します。
* **有効化を待機 (チェックボックス)** - 要素が対話できるようになるまで待機します。
* **消滅を待つ (チェックボックス)** - 要素が消滅するのを待ちます。
