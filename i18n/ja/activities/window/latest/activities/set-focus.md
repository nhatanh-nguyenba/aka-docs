# フォーカスを設定する

RCA.Activities.Windows.SetFocus

## **説明**

Set Focus アクティビティを使用すると、指定したウィンドウ要素にキーボード フォーカスを設定できます。

![image-20220505171141-1.png](/static/img/43aa65_image-20220505171141-1.png)

（\*必須）

## **アクティビティ本体内**

* **ターゲット要素を選択\***- フォーカスを設定する要素を選択します。このアクティビティは、その要素の場所を指定する文字列変数 (セレクター) を生成します。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には 2 つの値があります: True または False - True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を続行できるようにします。 - False: プロセスの実行の継続をブロックします。
* **TimeOut MS (Int32)** - アクティビティの実行を待機する時間 (ミリ秒単位)。デフォルトの時間は 30000 ミリ秒です。
例: 30000
* **Delay After (Int32)** - アクティビティ実行後の遅延時間 (ミリ秒単位)。
例: 300
* **Delay Before (Int32)** - アクティビティが操作の実行を開始するまでの遅延時間 (ミリ秒単位)。
例: 300

**入力**

* **セレクター (文字列)**\* - ウィンドウの位置を識別する文字列。

**その他**

* **公開 (チェックボックス)** - アクティビティを公開するかどうかをチェックします。このプロパティを使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [131231] フォーカスを設定する

**オプション**

* **オートメーション ベース (ドロップダウン リスト)** - オートメーション ライブラリのバージョンを指定します。このオプションは、Process 引数 (UIA2 または UIA3) と併用します。
* **要素 (AutomationElement)** - セレクター文字列の代わりに、type = ‘AutomationElement’ の変数を入力として使用します。
* **From (UiElement)** - セレクター文字列で要素を検索する場合、ルートへの UiElement 変数 (入力されていない場合は、デスクトップのルートから検索します)。
* **ウィンドウを使用する (WindowApp)** - オープン ウィンドウまたはウィンドウ スコープからの出力変数を使用して、アクティビティのスコープを設定します (From と同様)。

**待ってください**

* **無効化を待機 (チェックボックス)** - 要素が操作できなくなるまで待機します。
* **有効化を待機 (チェックボックス)** - 要素が対話できるようになるまで待機します。
* **消滅を待つ (チェックボックス)** - 要素が消滅するのを待ちます。
