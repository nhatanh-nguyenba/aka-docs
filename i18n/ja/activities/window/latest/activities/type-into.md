# 入力してください

RCA.Activities.Windows.TypeInto

## **説明**

Type Into アクティビティを使用すると、指定されたウィンドウのテキスト フィールドに入力できます。

![image-20220505170410-1.png](/static/img/491ac1_image-20220505170410-1.png)

（\*必須）

## **アクティビティ本体内**

* **ターゲット要素を選択**\* - 入力するテキスト フィールドを選択します。このアクティビティは、その要素の場所を指定する文字列変数 (セレクター) を生成します。
* **テキスト (文字列)**\* - 入力するテキスト。このプロパティは文字列変数をサポートしているため、引用符で囲む必要があります。
例: キーボード

## **プロパティ**

**入力前**

* **クリック (ブール値)** - 選択すると、テキストが書き込まれる前にテキスト フィールドがクリックされます。
* **ダブルクリック (ブール値)** - 選択すると、テキストが書き込まれる前にテキスト フィールドがダブルクリックされます。
* **空のフィールド (ブール値)** - 選択すると、テキストが書き込まれる前に、テキスト フィールド内の既存のコンテンツがすべて消去されます。
* **フォーカスの設定 (ブール値)** - キーボードのフォーカスをテキスト フィールドに設定します。

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの可能な値があります。
  - True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。
  - False: プロセスの実行の継続をブロックします。
* **TimeOut MS (Int32)** - エラーがスローされるまでにアクティビティの完了を待機する最大時間 (ミリ秒単位)。タイムアウトが経過すると、アクティビティは終了します。デフォルト値: 30000 (ミリ秒)。
例: 30000
* **Delay After (Int32)** - アクティビティ実行後の遅延時間 (ミリ秒単位)。
例: 300
* **Delay Before (Int32)** - アクティビティが操作の実行を開始するまでの遅延時間 (ミリ秒単位)。
例: 300

**入力**

* **セレクター (文字列)**\* - ウィンドウの位置を識別する文字列。
* **テキスト (文字列)**\* - 入力するテキスト。
例: キーボード

**その他**

* **公開 (チェックボックス)** - アクティビティを公開する場合にチェックを入れます。このプロパティを使用する前に、データ セキュリティ要件を必ず考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: [586302] と入力します

**オプション**

* **オートメーション ベース (ドロップダウン リスト)** - オートメーション ライブラリのバージョンを指定します。このオプションは、Process 引数 (UIA2 または UIA3) と併用します。
* **キー間の遅延 (数値)** - キー間の指定された時間。
* **要素 (AutomationElement)** - セレクター文字列の代わりに、type = ‘AutomationElement’ の変数を入力として使用します。
* **From (UiElement)** - セレクター文字列で要素を検索する場合、ルートへの UiElement 変数 (入力されていない場合は、デスクトップのルートから検索します)。
* **ウィンドウ メッセージを送信 (チェックボックス)** - ウィンドウ メッセージを送信するかどうかをチェックします。
* **ウィンドウを使用する (WindowApp)** - アクティビティの範囲を指定するために、Open Window または Window Scope からの出力変数を使用します (From と同様)。

**待って**

* **無効化を待機 (チェックボックス)** - 要素が操作できなくなるまで待機します。
* **有効化を待機 (チェックボックス)** - 要素が操作できるようになるまで待機します。
* **消滅を待つ (チェックボックス)** - 要素が消滅するのを待ちます。
