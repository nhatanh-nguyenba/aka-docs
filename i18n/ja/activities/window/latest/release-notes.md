# リリースノート

## **RCA.Activities.Windows バージョン 2.1.2.1**

### **バグ修正**

* **Show Window** アクティビティで、Element: 誤った ShowWindow の前に HideWindow をチェックする問題を修正しました。
* すべてのプロパティに有効な値を入力すると、**スクリーンショットの取得** アクティビティが機能しない問題を修正しました。
* 属性「名前」が正しく機能しないセレクターを持つ **すべて** アクティビティの問題を修正しました
* 入力 [Duration] が負の整数である場合の **ハイライト** アクティビティの問題を修正しました。
* **アクティブ ウィンドウの取得** アクティビティでアクティブ ウィンドウを取得できない問題を修正しました。
* アプリのサイズを変更できない場合に移動できない **ウィンドウの移動** アクティビティの問題を修正しました。
