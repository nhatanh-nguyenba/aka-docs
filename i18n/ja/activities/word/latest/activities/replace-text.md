# テキストを置換する

RCA.Activities.Word.ReplaceText

## **説明**

このアクティビティを使用すると、文字列 (テキスト形式) が 1 回または複数回出現したときに、出現するページごとに文字列を置換できます。このアクティビティは、Word Application Scope アクティビティでのみ使用されます。

![image-20220506111907-1.png](/static/img/47c77c_image-20220506111907-1.png)

**アクティビティ本体内**

* **(文字列)を検索** - 置換するテキストまたはキーワードを入力します。文字列変数と文字列のみがサポートされます。
* **(文字列) で置換** - 置換するテキストまたはキーワードを入力します。文字列変数と文字列のみがサポートされます。

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト): プロセスの実行の継続をブロックします。

**入力**

* **置換 (文字列)** - 検索された文字列を置換するテキストまたはキーワードを入力します。文字列変数と文字列のみがサポートされます。
* **検索 (文字列)** - 置換するテキストまたはキーワードを入力します。文字列変数と文字列のみがサポートされます。

**その他**

* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: テキストを置換
* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。

**結果**

* **Found (Boolean)** - Found 情報は、置換されるテキストまたはキーワードが見つかったかどうかを示します。
