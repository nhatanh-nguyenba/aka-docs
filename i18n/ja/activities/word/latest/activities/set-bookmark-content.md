# ブックマークの内容を設定する

RCA.Activities.Word.SetBookmarkContent

## **説明**

このアクティビティを使用すると、ドキュメント内の指定したブックマークの場所にテキストまたはコンテンツを追加できます。
このアクティビティは、**Word アプリケーション スコープ** アクティビティ内でのみ使用できます。

![image-20220506111935-1.png](/static/img/18e34e_image-20220506111935-1.png)

## **アクティビティの本文内**

* **ブックマーク名 (文字列)**: テキストが書き込まれる既存のブックマークの名前。 String および String 変数のみをサポートします。
例:`"Description"`
* **ブックマーク テキストの内容 (文字列)**: 指定されたブックマークに書き込まれるテキスト。 String および String 変数のみをサポートします。
例:`"Title"`

## **プロパティ**

**共通**

* **Continue On Error (Boolean)**: True または False の 2 つの値を持つブール変数。
True: アクティビティ内でエラーが発生しても実行を継続します。
False (デフォルト): プロセスの実行継続を停止します。

**入力**

* **フォーム フィールド (チェックボックス)**: ブックマークがフォーム フィールドの場合は、このオプションを選択します。デフォルトでは、このオプションはオフになっています。
* **ブックマーク名 (文字列)**: テキストが書き込まれる既存のブックマークの名前。 String および String 変数のみをサポートします。
例:`"Description"`
* **ブックマーク テキストの内容 (文字列)**: 指定されたブックマークに書き込まれるテキスト。 String および String 変数のみをサポートします。

**その他**

* **パブリック (チェックボックス)**: 選択すると、アクティビティ データが記録されます。このプロパティを使用する前に、データ セキュリティ要件を考慮してください。
* **表示名 (文字列)**: このアクティビティの名前。これを編集して、ワークフローをより適切に整理および構造化できます。
例:`Set Bookmark Content`
