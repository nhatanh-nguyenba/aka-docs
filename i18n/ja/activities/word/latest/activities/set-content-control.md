# コンテンツコントロールを設定する

RCA.Activities.Word.SetContentControl

## **説明**

このアクティビティを使用すると、Word 文書内の特定のコンテンツ コントロールにコンテンツを追加できます。現在、CheckBox(Boolean)、TextBox(String)、コンボボックス(int)をサポートしています。アクティビティを選択すると、コンテンツ コントロールで許可される値のタイプを尋ねられます。

![image-20230421110354-1.png](/static/img/735f2b_image-20230421110354-1.png)

## **プロパティ**

**共通**

* **エラー時に続行 (ブール値)** - ブール変数には、True または False の 2 つの値があります。 True: アクティビティ内でエラーが発生した場合でも、プロセスの残りの部分が実行を継続できるようにします。 False (デフォルト): プロセスの実行の継続をブロックします。

**入力**

* **タグで検索 (文字列)** – このオプションを使用すると、タグでコンテンツ コントロールを検索できます。
* **タイトル (文字列) で検索** - このオプションを使用すると、タイトルでコンテンツ コントロールを検索できます。
* **見つかったすべてのコントロールを設定 (ブール値)** – 上記のプロパティで同じタイトルまたはタグの意味を持つすべてのコンテンツ コントロールに値を追加する場合は、このオプションをオンにします。これはデフォルトではオフになっています。
* **値 (オブジェクト)\*** – コンテンツ コントロールに追加する値

**その他**

* **パブリック (チェックボックス)** - チェックすると、このアクティビティのデータがログに表示されます。使用する前にデータのセキュリティを考慮してください。
* **表示名 (文字列)** - このアクティビティの名前。アクティビティの名前を編集して、コードをより適切に整理および構造化できます。
例: コンテンツコントロールの設定

**出力**

* **結果 (ブール値)** – アクティビティの結果。 True または False の 2 つの値のみを返します。 True - 値は正常に追加されました。 False – 値は正常に追加されませんでした。
