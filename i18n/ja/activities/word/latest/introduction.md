# はじめに

**Word** **パッケージ** には、.docx ファイルを操作できるようにする複数のアクティビティが含まれています。これらを使用して、ドキュメントへの画像の追加、ドキュメント内のテキストの読み取り、追加または置換、ブックマークの追加、および PDF へのエクスポートが可能です。
