# ハードウェアとソフトウェアの要件 - akaBot Docs

## **ハードウェア要件**

akaBot Studio と Agent を使用するには、コンピューターが以下のハードウェア要件を満たす必要があります。

|  | 前提条件 | おすすめ |
| --- | --- | --- |
| **CPU** | 2 x 1.8GHz 64ビット | 4 x 2.4GHz 64ビット |
| **RAM** | 4 GB の利用可能なシステム メモリ | 8 GB の利用可能なシステム メモリ |

## **ソフトウェア要件**

akaBot Studio と Agent を使用するには、コンピュータが以下のソフトウェア要件を満たす必要があります。

|  | サポートされているバージョン |
| --- | --- |
| **オペレーティング システム** | Windows 10 |
|  | Windows10N |
|  | Windows 11 |
|  | Windowsサーバー2016 |
|  | Windowsサーバー2019 |
|  | Windowsサーバー2022 |
|  | Windowsサーバー2025 |
| **.NET フレームワーク** | 最小: 4.5.2 |
