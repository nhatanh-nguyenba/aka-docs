# ライセンスのアクティベーション

## **始める前に**

**以下から適切なアクティベーション方法を選択してください:**

1. **標準 (またはスタンドアロン) ライセンス:**
   * akaBot Agent コンピュータ **(デバイス A)** に **アクティブなインターネット接続**がある場合は、**オンライン アクティベーション** 方法を使用して、便利で迅速なアクティベーションを行うことができます。
   * akaBot Agent コンピュータがインターネットに接続できない場合は、**オフライン アクティベーション** 方法を使用してください。正常にアクティブ化するには、次のことを確認する必要があります。
     1. インターネットに接続された別のデバイス **(デバイス B)** を使用して、akaBot ライセンス管理システムからライセンス ファイルを取得します。
     2. ライセンス ファイルをデバイス B からデバイス A に転送するには、デバイス A とデバイス B を接続します。
2. **エンタープライズ (またはネットワーク) ライセンス**
**独自の** **akaBot Center** を通じてアクティブ化できます。 akaBot Agent コンピュータが akaBot Center にアクティブに接続されていることを確認してください。
注: ネットワークがプロキシ サーバー経由でインターネットまたは akaBot Center に接続する必要がある場合は、アクティブ化する前にプロキシ情報があることを確認してください。

## 

## **オンラインアクティベーション**

**ステップ 1** - エージェントを起動します。 Windows のインストールでは、オプション「LMS (ライセンス管理システム)」を選択し、「次へ」ボタンを押します。
![image-20230718223155-1.png](/static/img/cd14b4_image-20230718223155-1.png)

**ステップ 2** – オンライン アクティベーション オプションを選択し、[次へ] ボタンを押します。
![image-20230718223155-2.png](/static/img/73e6b2_image-20230718223155-2.png)

**ステップ 3** – ライセンス情報を入力し、[次へ] ボタンを押します。

1 -**ライセンスサーバー**:[https://aegis.akabot.com/](https://aegis.akabot.com/)

2 -**ライセンス キー**: ライセンス電子メールに記載されているライセンス キーを入力してください。

3 -**ネットワーク オプション (オプション)**: プロキシ サーバーを使用する場合は、***ネットワーク オプション*** の下にプロキシ情報を入力してください。

![image-20230718223155-3.png](/static/img/1f4322_image-20230718223155-3.png)

**ステップ 4** - 「**アクティブ化」** ボタンを押します。 **アクティベーションが成功**すると、akaBot Agent に以下のメッセージが表示されます。

![image-20230718223155-4.png](/static/img/f7c766_image-20230718223155-4.png)

以下のエラーメッセージが表示された場合は、ネットワーク接続を確認してください。

![image-20230718223155-5.png](/static/img/e5aad4_image-20230718223155-5.png)

## **オフラインアクティベーション**

**akaBot Agent コンピュータ (デバイス A):**

**ステップ 1** - エージェントを起動します。 Windows のインストールでは、オプション「LMS (ライセンス管理システム)」を選択し、「次へ」ボタンを押します。

![image-20230718223155-6.png](/static/img/5911a7_image-20230718223155-6.png)

**ステップ 2** – オフラインアクティベーションオプションを選択し、「次へ」ボタンを押します。

![image-20230718223155-7.png](/static/img/ed8824_image-20230718223155-7.png)

**ステップ 3 –** ライセンス メールにライセンス キーを入力し、[次へ] ボタンを押します。

![image-20230718223155-8.png](/static/img/46649e_image-20230718223155-8.png)

**ステップ 4 - 以下の**アクティベーション リクエスト**をコピーし、[次へ] をクリックします。

![image-20230718223155-9.png](/static/img/272a55_image-20230718223155-9.png)
![image-20230718223155-10.png](/static/img/c1d57c_image-20230718223155-10.png)

**ステップ 5 –** アクティベーション要求をインターネット アクセス デバイス (デバイス B) にコピーします。

**デバイス B:**

**ステップ 6** -[https://aegis.akabot.io/#/activation](https://aegis.akabot.io/#/activation)に移動します。
AkaBot Agent コンピュータ (デバイス A) からコピーした**アクティベーション リクエスト**を貼り付け**、キャプチャを選択し、**ファイルをダウンロード**してアクティベーション レスポンス ファイル (activate.txt という名前) をダウンロードします。

![image-20230718223155-11.png](/static/img/ab2866_image-20230718223155-11.png)

**ステップ 7** – アクティベーション応答ファイルをデバイス B からデバイス A にコピーします。

**akaBot Agent コンピュータ (デバイス A):**

**ステップ 8 –** アクティベーション レスポンス ファイルを参照するか、ファイル内のアクティベーション レスポンスをアクティベーション ボックスにコピーして、「アクティベート」ボタンを押します。

![image-20230718223155-12.png](/static/img/2a654d_image-20230718223155-12.png)

**アクティベーションが成功**すると、akaBot Agent に以下のメッセージが表示されます。

![image-20230718223155-13.png](/static/img/9790b8_image-20230718223155-13.png)

## **akaBot センターを通じてアクティブ化します**

**ステップ 1** - エージェントを起動します。 Windowsのインストールでは、オプション「akaBot Center」を選択し、「次へ」ボタンを押します。

![image-20230718223155-14.png](/static/img/d49ed9_image-20230718223155-14.png)

**ステップ 2** – ライセンス情報を入力します。

1. **センター URL:** akaBot センターの URL
2. **ライセンス キー**: ライセンス電子メールに記載されているライセンス キーを入力してください。
3. **ネットワーク オプション (オプション)**: プロキシ サーバーを使用する場合は、***ネットワーク オプション*** の下にプロキシ情報を入力してください。
4. 「有効化」ボタンを押します

![image-20230718223155-15.png](/static/img/2583d0_image-20230718223155-15.png)

**アクティベーションが成功**すると、akaBot Agent に次のメッセージが表示されます。

![image-20230718223155-16.png](/static/img/568bd1_image-20230718223155-16.png)

以下のエラーメッセージが表示された場合は、ネットワーク接続を確認してください。

![image-20230718223155-17.png](/static/img/6661dd_image-20230718223155-17.png)
