# 有人 - 無人エージェント

## 内容

| 基準 | 出席しました | 無人 |
| --- | --- | --- |
| **強さ** | - 個々のユーザーが小規模で反復的なタスクを処理できるように支援します。<br/>- 人間と協力して、長時間実行されるプロセスを完了します。 | 人間の介入を必要とせずに、長時間のプロセスや自動化を実行する。 |
| **操作方法** | プロセスは、ユーザーまたは特定のユーザー イベントによってトリガーされます。 | プロセスのトリガーをユーザーに依存しません。これらにはタスクが割り当てられ、akaBot Center を通じて実行されるようにスケジュールされます。 |
| **エージェント実行のマシン要件** | エージェントは、ユーザーが日常のタスクを実行するマシンと同じマシンにインストールする必要があります。| エージェントは、akaBot Center に接続されている任意のマシンにインストールできます。 |
| **あなたのビジネスにこのタイプのエージェントが必要になるのはどのような場合ですか?** | 日常の小さなタスクやプロセスの特定の部分を支援するために、エンドユーザー (人事担当者、コールセンター オペレーター、またはデータ検証スペシャリスト) のパーソナル アシスタントが必要な場合。 | 会社が、複数のユーザーまたは部門全体の反復的または長時間にわたるタスクを合理化するために、バックグラウンドで大規模に動作する多数のエージェントを必要とする場合。 |
