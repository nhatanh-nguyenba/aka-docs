## **1. akaBot センターをアクティブ化**

**ステップ 1** - ブラウザを開きます。 **センター URL** または **localhost:8080** を使用して、akaBot Center に移動します。ログインに成功したら、左側のメニュー バーで [**管理**] を選択し、[**ライセンス**] オプションを選択して、[**ライセンスの追加**] をクリックします。

![1698997596149-505.png](/static/img/f952ee_1698997596149-505.png)

**ステップ 2:**

* ライセンス電子メールから受け取ったライセンス キーを [ライセンス キー] フィールドに入力します。
* [**生成**] をクリックします。

![1695699235091-617.png](/static/img/ed9bd2_1695699235091-617.png)

**ステップ 3:** 以下のアクティベーション リクエストをコピーし、リンクを開きます: **https:aegis.akabot.com/#/activation**

![1695699317528-675.png](/static/img/6a3a1a_1695699317528-675.png)​

**ステップ 4**: ボックスに貼り付けます -> **ファイルをダウンロード** をクリックします

![1695699508304-280.png](/static/img/1ca059_1695699508304-280.png)

**ステップ 5**: センターの [**アクティブ化] タブ**にダウンロードしたファイルをアップロードし、[**アクティブ化**] をクリックします。

![1695699642889-537.png](/static/img/4e9f2d_1695699642889-537.png)

akaBot Center をアクティブ化するためのすべての手順が完了しました。

**ステップ 6**: akaBot Center が正常にアクティブ化されると、akaBot Center は Studio と Agent をアクティブ化するためのキーを自動的に生成します。 akaBot Studio と Agent をアクティブ化する前に、これをコピーしてください。

![1701916021894-635.png](/static/img/7fbac6_1701916021894-635.png)

## **2.  akaBot Studio をアクティブ化**

**独自の****akaBot センター**をお持ちの場合は、akaBot Studio コンピュータが akaBot センターにアクティブに接続されていることを確認してください。
ネットワークでプロキシ サーバー経由でインターネットまたは akaBot Center に接続する必要がある場合は、アクティブ化する前にプロキシ情報があることを確認してください。

akaBot Center をアクティブ化した後、以下の手順に従って akaBot Studio をアクティブ化してください。

**ステップ 1** - スタジオを開始します。 Windowsのインストールでは、オプション「akaBot Center」を選択し、「次へ」ボタンを押します。

![image-20230717142905-14.png](/static/img/4b6c4b_image-20230717142905-14.png)

**ステップ 2** – ライセンス情報を入力します。

1. **センター URL:** akaBot センターの URL
2. **ライセンス キー**: akaBot Center をアクティブ化した後に自動的に生成されるライセンス キーを入力してください。 (**ステップ 6、パート 1** - akaBot Center をアクティブ化します)
3. **ネットワーク オプション (オプション)**: プロキシ サーバーを使用する場合は、***ネットワーク オプション*** の下にプロキシ情報を入力してください。
4. 「有効化」ボタンを押します

![image-20230717142905-15.png](/static/img/db1837_image-20230717142905-15.png)

**アクティベーションが成功**すると、akaBot Studio に次のメッセージが表示されます。

![image-20230717142905-16.png](/static/img/7232c3_image-20230717142905-16.png)

以下のエラーメッセージが表示された場合は、ネットワーク接続を確認してください。

![image-20230717142628-5.png](/static/img/a2a943_image-20230717142628-5.png)

## **3.  akaBot エージェントをアクティブ化**

**独自の****akaBot センター**をお持ちの場合は、akaBot Studio コンピュータが akaBot センターにアクティブに接続されていることを確認してください。
ネットワークでプロキシ サーバー経由でインターネットまたは akaBot Center に接続する必要がある場合は、アクティブ化する前にプロキシ情報があることを確認してください。

### **3.1. akaBot エージェントをアクティブ化**

**ステップ 1** - エージェントを起動します。 Windowsのインストールでは、オプション「akaBot Center」を選択し、「次へ」ボタンを押します。

![image-20230718223155-14.png](/static/img/d49ed9_image-20230718223155-14.png)

**ステップ 2** – ライセンス情報を入力します。

1. **センター URL:** akaBot センターの URL
2. **ライセンス キー**: akaBot Center をアクティブ化した後に自動的に生成されるライセンス キーを入力してください。 (**ステップ 6、パート 1** - akaBot Center をアクティブ化します)
3. **ネットワーク オプション (オプション)**: プロキシ サーバーを使用する場合は、***ネットワーク オプション*** の下にプロキシ情報を入力してください。
4. 「有効化」ボタンを押します

![image-20230718223155-15.png](/static/img/2583d0_image-20230718223155-15.png)

**アクティベーションが成功**すると、akaBot Agent に次のメッセージが表示されます。

![image-20230718223155-16.png](/static/img/568bd1_image-20230718223155-16.png)

以下のエラーメッセージが表示された場合は、ネットワーク接続を確認してください。

![image-20230718223155-17.png](/static/img/6661dd_image-20230718223155-17.png)

### **3.2. akaBot Agent を akaBot Center に接続**

アクティベーション後、akaBot Agent を akaBot Center に接続して、アクションを開始できるようにする必要があります。

akaBot Agent をアクティブ化した後、以下の手順に従って akaBot Center に接続してください。

**ステップ 1** - ブラウザを開きます。 **センター URL** または **localhost:8080** を使用して、akaBot Center に移動します。ログインに成功したら、左側のメニュー バーで [**エージェント**] を選択し、[**エージェント**] オプションを選択して、[**新規作成**] をクリックします。

![1698998855183-732.png](/static/img/a2f411_1698998855183-732.png)

**ステップ 2**: クリックすると、新しいエージェントを作成/編集するためのプロパティを入力できるポップアップが表示されます。以下のフォームに必要な情報をすべて入力する必要があります。

![image-20221031155928-12.png](/static/img/941015_image-20221031155928-12.png)

**ステップ 3**: akaBot Agent を再度開き、![1698999942255-125.png](/static/img/79e71c_1698999942255-125.png)ボタンを選択します

![1698999960723-227.png](/static/img/6c9b56_1698999960723-227.png)

**ステップ 4**: クリックすると、プロパティを入力できるポップアップが表示されます。

![1699000021013-374.png](/static/img/4f6c9e_1699000021013-374.png)

* マシン名: 現在のコンピューターのコンピューター名。マシン名は自動的に事前入力されます。
* エージェント キー: キーは、ステップ 2 でエージェントを作成したときに akaBot Center で自動生成されました (ステップ 2 でコピーしました。まだコピーしていない場合は、心配しないでください。akaBot Center でエージェントを開いて表示し、コピーしてください)。
* センター URL: akaBot センター URL。

**ステップ 5:** – [接続] をクリックして、akaBot Center への接続を実行します。

接続に成功すると、akaBot Center のステータスが「接続済み」に変わります。

![1699000551168-543.png](/static/img/8791dd_1699000551168-543.png)

ボットが既に akaBot Center に接続されている場合は、[**切断**] ボタンをクリックしていつでも切断できます。
