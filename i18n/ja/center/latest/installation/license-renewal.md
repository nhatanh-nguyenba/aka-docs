# ライセンスの更新

このドキュメントは、**akaBot Center** でライセンスを更新する方法を段階的に説明することを目的としています。

## **更新**

**ステップ 1:**

* Web ブラウザ経由で akaBot Center にアクセスします。
* [ライセンス] オプションを選択します。

![1695699090252-386.png](/static/img/1a43fc_1695699090252-386.png)

**ステップ 2:** 更新するライセンスの [**ライセンスを更新**] オプションをクリックします。

![1695699157651-425.png](/static/img/66a519_1695699157651-425.png)

**ステップ 3:** 以前に受け取ったライセンス キーを (ライセンス電子メールに) 挿入し、[生成] をクリックします。

![1695699235091-617.png](/static/img/ed9bd2_1695699235091-617.png)

**ステップ 4:** 以下の **アクティベーション リクエスト**をコピーし、リンクを開きます: **https:aegis.akabot.com/#/activation**

![1695699317528-675.png](/static/img/1695699317528-675.png)

**ステップ 5**: ボックスに貼り付けます -> **ファイルをダウンロード** をクリックします

![1695699508304-280.png](/static/img/1ca059_1695699508304-280.png)

**ステップ 6**: akaBot センターに戻り、センターの **[アクティブ化] タブ**にダウンロードしたファイルをアップロードし、[**アクティブ化**] をクリックします。

![1695699642889-537.png](/static/img/4e9f2d_1695699642889-537.png)

ライセンスを更新するためのすべての手順が完了しました。すべての手順を実行しても成功しない場合は、電子メール アドレス [[support@akaBot.com]] までお気軽にご連絡ください。

**注意**: 自動ライセンス更新機能は、akaBot Agent バージョン 2.2.0.2 以降でのみ利用可能になります。これは、ユーザーが akaBot Center でライセンスを更新すると、エージェントが有効期限、エージェントの最大数などの情報を自動的に更新することを意味します。古いエージェント バージョン (例: akaBot Agent バージョン 2.2.0.1) の場合、ユーザーはライセンスの有効期限が切れたときに再アクティブ化する必要があります。
