akaBot Center ユーザー インターフェイスは、明確さと運用効率を目的に設計された Web ベースのアプリケーションです。レイアウトは、永続的な左側のナビゲーション サイドバー、上部のヘッダー バー、および選択したモジュールに基づいて変化するメイン コンテンツ領域を中心に構成されています。この一貫した構造により、ユーザーはコンテキストを失うことなく、実行中のジョブの監視、レポートの確認、エージェントの管理など、さまざまな操作機能をすばやく切り替えることができます。

上部のヘッダー バーには、現在のユーザーの名前とプロファイル オプション、システム通知、およびクイック アクセス設定が表示されます。通知ベルのアイコンは、ジョブの失敗、エージェントの切断、ライセンスの有効期限切れの警告などの重要なイベントに対してリアルタイムのアラートを提供します。通知をクリックすると、ユーザーは関連する詳細ページに直接移動できるため、運用上の問題の診断と対応に必要な時間が短縮されます。

メイン コンテンツ領域は、アクティブなモジュールに応じてレイアウトを調整します。リスト ビューは、並べ替え可能でフィルター可能なテーブルを使用して、ジョブ、エージェント、パッケージなどのレコードを表示します。詳細ビューには、構成フォーム、実行ログ、ステータス インジケーターが表示されます。ホーム画面などのダッシュボード ビューでは、チャートやカウンターを備えたウィジェット ベースのレイアウトを使用して、一目でわかる操作概要を提供します。

## UI レイアウトの概要

| いいえ | パネル | 説明 |
| --- | --- | --- |
| 1 | **機能メニュー** | akaBot Center の主な機能をすべて表示します |
| 2 | **クイックナビゲーション** | ページ、パッケージ、エージェント、エージェント グループ、ワークフロー、アセット、キュー、スケジュールなどに簡単に移動できます。 |
| 3 | **メイン画面** | 各機能の詳細を表示します |
| 4 | **クイック作成** | タスク/スケジュール/ワークフロー/トリガー/エージェントを素早く作成 |
| 5 | **組織単位選択メニュー** | アクティブな組織単位を変更する |
| 6 | **言語** | Akabot Center の表示言語を変更します。英語、韓国語、日本語、ベトナム語、台湾語をサポートします。 |
| 7 | **ユーザーメニュー** | 次のオプションとページにアクセスします。 ***設定**: このページでは、名、姓、電子メール、表示言語などのユーザーの詳細を変更できます。* **パスワード**: このページでは、Center へのログインに使用するパスワードを変更できます。 \* **サインアウト**: ここをクリックしてサインアウトします。 |
| 8 | **バージョン** | akaBot Center のバージョンを表示する |
