# 役割管理

akaBot には、ロールと権限ベースのアクセス制御フレームワークが組み込まれています。ロールは権限のセットです。つまり、特定のセンター リソースにアクセスして制御するために必要な権限がロールに割り当てられます。言い換えれば、ユーザーには特定のリソースへのアクセス許可が与えられていませんが、ロールを通じて許可を取得することになります。ユーザーは、その役割に応じて、以下にリストされているリソースにアクセスできます。

| いいえ | リソース |
| --- | --- |
| 1 | 管理リソース |
| 2 | 環境資源 |
| 3 | ロボットリソース |
| 4 | パッケージリソース |
| 5 | ワークフローリソース |
| 6 | ジョブリソース |
| 7 | スケジュールリソース |
| 8 | 組織単位のリソース |
| 9 | 資産リソース |
| 10 | キューリソース |
| 11 | ログリソース |

| 権限 | カテゴリ |
| --- | --- |
| なし | ユーザーにはセンター インターフェイスにリソースが表示されません。 |
| 見る | ユーザーはリソースを表示できますが、操作することはできません。 |
| 編集 | ユーザーはリソースを表示および編集できます。 |
| 作成 | ユーザーは新しいリソースを表示および作成できます。 |
| 削除 | ユーザーはリソースを削除できます。 |

この役割管理ページには、使用可能なすべての役割が表示され、管理者は役割の情報を追加または削除したり更新したりできます。 [**役割管理**] タブを選択すると、次のページが表示されます。

![image-20221101164359-16.png](/static/img/d407b3_image-20221101164359-16.png)

| いいえ | コラム | 説明 |
| --- | --- | --- |
| 1 | アクション | ユーザーを管理するために使用できるアクション。内容: 編集: ユーザーがロールの詳細を表示できるようにします。削除: ユーザーが選択した役割を削除できるようにします |
| 2 | 名前 | 役名 |

* デフォルトの役割

akaBot Center には、3 つのデフォルトの役割があります。

- 役割\_管理者

- 役割\_ユーザー

- 役割\_ロボット

**役割\_ADMIN**
デフォルトでは、ロールにはすべてのリソースに対するすべての権限があり、これは変更できません。

**役割\_ユーザー**
デフォルトでは、ロールには次の権限がありますが、これらは編集できます。

![image-20230704162245-1.png](/static/img/27201e_image-20230704162245-1.png)

**役割\_ロボット**
デフォルトでは、ロールには次の権限がありますが、編集することができます。

![image-20230704162317-2.png](/static/img/50ea0e_image-20230704162317-2.png)

## **a.役割の作成/編集**

センターの新しい役割を作成するには、ページの上部にある [新規作成] ボタンをクリックします。

![image-20221101164549-17.png](/static/img/aacffe_image-20221101164549-17.png)

既存の役割を編集するには、「アクション」列の役割に対応する「編集」ボタンをクリックします。

![image-20221101164638-18.png](/static/img/c7556f_image-20221101164638-18.png)

[**新規作成/編集**] ボタンをクリックすると、入力するフォームが表示されます。詳細は以下の表にあります。

![image-20230704162458-3.png](/static/img/96e8d2_image-20230704162458-3.png)

| いいえ | 列/ラベル | 説明 | 種類 | 最大値 | 必須ですか? | 入力要件 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | 名前 | ロール名を入力してください | 文字列 | 100文字 | はい |  |
| 2 | リソース | 各行で選択できるオプションは 1 つだけです ロールの権限を選択してください | 単一選択 |  | はい |  |

## **b.役割を削除します**

ロールを削除するには、対応するロールの [削除] ボタンをクリックします。

![image-20221101164723-19.png](/static/img/f253ab_image-20221101164723-19.png)

確認のポップアップが表示されます。 「削除」を押して操作を完了します。
