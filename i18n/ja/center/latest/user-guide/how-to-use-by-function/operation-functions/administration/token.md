# トークン

トークン ページでは、アカウントとパスワードをサードパーティに送信せずに、API にアクセスするための新しいトークンを簡単に作成できます。

ダッシュボードから「管理」タブをクリックし、「トークン」タブを選択します。クリックすると、作成されたトークンの一覧が表示されます。

![image-20221101170819-30.png](/static/img/284198_image-20221101170819-30.png)

| いいえ | 列/ラベル | 説明 |
| --- | --- | --- |
| 1 | アクション | Webhook を管理するために使用できるアクション。含まれるもの: 編集: ユーザーがトークンを編集できるようにします。削除: ユーザーが選択したトークンを削除できるようにします。 |
| 2 | 別名 | トークンの送信先となるサードパーティの名前。 |
| 3 | 名前 | トークンの名前 |
| 4 | 説明 | 作成/編集するOUの説明を入力します。 |
| 5 | 役割 | トークンにアクセスできるロール |
