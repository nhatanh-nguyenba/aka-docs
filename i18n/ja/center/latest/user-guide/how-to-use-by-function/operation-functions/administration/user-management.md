#  ユーザー管理

ユーザーは、アクセス依存の機能を持つエンティティであり、akaBot Center の視点と制御は、指定された役割に依存します。ユーザーはセンターで作成できます。

このページには利用可能なすべてのユーザーが表示され、管理者はユーザーの情報を追加または削除したり、更新したりできます。

[**ユーザー管理**] タブを選択すると、次のページが表示されます。

![image-20221101163259-10.png](/static/img/348667_image-20221101163259-10.png)

| いいえ | コラム | 説明 |
| --- | --- | --- |
| 1 | アクション | ユーザーを管理するために使用できるアクション。内容: 編集: ユーザーがユーザーの詳細を表示できるようにします。削除: ユーザーは選択したユーザーを削除できます。 |
| 2 | ログイン | サインインに使用されるユーザー名の資格情報。 |
| 3 | 電子メール | ユーザーの電子メール アドレス。 akaBot Center へのログインに使用できます。 |
| 4 | ステータス | ユーザーのステータス。有効化または無効化できます。 |
| 5 | 言語 | ユーザーに対する akaBot Center の表示言語。 |
| 6 | プロフィール | ユーザーに割り当てられた役割。デフォルトのロールは 3 つあります: ROLE\_ADMIN ROLE\_USER \* ROLE\_ROBOT ユーザーは複数のロールを持つことができます。各ロールの権限の詳細については、ロール管理で説明します。 |
| 7 | 作成日 | ユーザーが作成された日付 |
| 8 | 変更者 | ユーザー情報を最後に変更した人 |
| 9 | 変更日 | ユーザー情報が最後に変更された日付 |

## **a. ユーザーを表示**

ユーザーの詳細を表示するには、**目** ボタンをクリックします。

![image-20221101163356-11.png](/static/img/7501e4_image-20221101163356-11.png)

![image-20221101163519-12.png](/static/img/579bc1_image-20221101163519-12.png)

| いいえ | コラム | 説明 |
| --- | --- | --- |
|  | ログイン | サインインに使用されるユーザー名の資格情報。 |
|  | ステータス | ユーザーのステータス。アクティブ化または非アクティブ化できます。 |
|  | 名 | ユーザーの名。 |
|  | 姓 | ユーザーの姓。 |
|  | 電子メール | ユーザーの電子メール アドレス。 akaBot Center へのログインに使用できます。 |
|  | 言語 | ユーザーごとの akaBot Center の表示言語。 |
|  | 作成者 | このユーザーを作成したユーザー。 |
|  | 作成日 | ユーザーが作成された日付。 |
|  | 変更者 | ユーザー情報を最後に変更した人。 |
|  | 変更日 | ユーザー情報が最後に変更された日付 |
|  | プロフィール | ユーザーに割り当てられた役割。デフォルトのロールは 3 つあります: ROLE\_ADMIN ROLE\_USER \* ROLE\_ROBOT ユーザーは複数のロールを持つことができます。各役割の権限の詳細については、「役割管理」セクションで説明します。 |

## **b.ユーザーの作成/編集**

新しいユーザーを作成するには、[**ユーザー**] ページの上部にある [**新規作成**] をクリックします。

![image-20221101163616-13.png](/static/img/36cead_image-20221101163616-13.png)

[**新規作成/編集**] ボタンをクリックすると、入力するフォームが表示されます。詳細は以下の表にあります。

![image-20221101163638-14.png](/static/img/5d2288_image-20221101163638-14.png)

| いいえ | 列/ラベル | 説明 | タイプ | 最大値 | 必須ですか? | 入力要件 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | ログイン | 作成/編集しているユーザーのシステムへのログインに使用するユーザー名を入力します | 文字列 | 50 | はい |  |
| 2 | 名 | ユーザーの名を入力してください | 文字列 | 50 | はい |  |
| 3 | 姓 | ユーザーの姓を入力してください | 文字列 | 50 | はい |  |
| 4 | 電子メール | ユーザーのメールアドレスを入力してください | 文字列 | 100 | はい |  |
| 5 | パスワード | 新規作成でのみ使用可能 ユーザーのパスワードを入力してください | 文字列 | 50 | はい |  |
| 6 | パスワードの確認 | 新規作成時のみ利用可能 ユーザーのパスワードを再入力してください | 文字列 | 50 | はい |  |
| 7 | アクティブ化された | このユーザーを作成時にアクティブ化する場合はこれを選択します | チェックボックス |  | いいえ |  |
| 8 | 言語 | ユーザーの言語を選択します | ドロップダウン リスト、単一選択 |  | はい |  |
| 9 | プロフィール | ユーザーの役割を選択する | 複数選択 |  | はい |  |

フォームに入力した後、[**保存**] ボタンをクリックして保存すると、ユーザーが作成されます。 [**キャンセル**] ボタンをクリックして、キューの作成をキャンセルすることもできます。

## **c.ユーザーの削除**

ユーザーを削除するには、対応するユーザーの「削除」ボタンをクリックします。

![image-20221101163711-15.png](/static/img/c91ba3_image-20221101163711-15.png)

確認のポップアップが表示されます。 [**削除**] を押して操作を完了します。
