# 資産

**アセット** は通常、さまざまな自動化プロジェクトで使用できる共有変数または認証情報を表します。ロボットが簡単にアクセスできるように、特定の情報を保存できます。

さらに、ここに保存されるすべての資格情報が AES 256 アルゴリズムで暗号化されるため、追加のセキュリティ レベルが提供されます。これらは RPA 開発者がプロ​​セスを設計するときに呼び出すことができますが、その値は非表示にすることができます。 **アセット** ページでは、新しいアセットを作成できます。また、以前に作成されたすべてのアセットも表示されます。
編集または削除することができます。

**アセット** ページにアクセスするには、左側のメニューの **アセット** タブをクリックします。クリックすると、すべてのアセットが含まれる**アセット リスト**が開きます。

![image-20221028172721-18.png](/static/img/4a4112_image-20221028172721-18.png)

| いいえ | コラム | 説明 |
| --- | --- | --- |
| 1 | アクション | アセットを管理するために使用できるアクション。内容: *編集: 各アセットの詳細を編集できます* 削除: 選択したアセットを削除できます 各アセットに対応するチェックボックスを選択すると、フィルター オプションの横に削除オプションが表示され、センターからアセットを削除できるようになります。 「アクション」の横にあるチェックボックスを選択すると、表示されているすべてのアセットが選択され、選択したアセットを一括削除できるようになります。 |
| 2 | 名前 | 資産の名前 |
| 3 | タイプ | 必須。次のオプションが利用可能です: ***テキスト** - 文字列値を保存できるアセット。これはデフォルトのオプションです。* **Bool** - true または false の値を保存できるアセット。このオプションを選択すると、「値」フィールドに 2 つのラジオ ボタン (True、False) が表示されます。 ***Integer** - 整数を保存できるアセット。* **Credential** - 認証情報を保存できるアセット。このオプションを選択すると、**値**フィールドは他の2つの**ユーザー名**と**パスワード**に置き換えられます。 |
| 4 | 値 | アセットに割り当てられる値: ***共通値** - 特定の値がエージェントに割り当てられていない限り、すべてのエージェントが受け取るデフォルト値。* **エージェントごとの値** - 指定されたエージェントのみが受け取る値。 **共通値**が無効になっている場合は、少なくとも 1 つのエージェント値を指定する必要があります |
| 5 | 作成者 | アセットを作成したユーザー。 |

## **d.アセットの削除**

タスクを削除するには、対応するアセットの [削除] ボタンをクリックします。

![image-20221101104746-3.png](/static/img/57ab4d_image-20221101104746-3.png)

あるいは、各アセットの前にあるチェックボックスを選択すると、**削除** ボタンが**フィルタ** ボタンの横に表示されます。 [**アクション**] の横にあるチェックボックスをオンにすると、ユーザーは表示されているすべてのアセットを選択して一括削除できるようになります。

![image-20221101104914-4.png](/static/img/714702_image-20221101104914-4.png)

## **e. 2レベルの資産管理**

この機能により、管理者は 2 つのレベルで資産を管理できます。
レベル 1: 管理者は、アセット内のすべての値を個人または個人のグループに割り当てることができます。割り当てられた個人またはグループのみが表示および編集できます。

![1695035724812-354.png](/static/img/3b206f_1695035724812-354.png)

レベル 2: 管理者は、資産セット内の特定の値を個人または個人のグループに割り当てることができます。割り当てられた個人またはグループのみが値を表示および編集できます。

![1695035642475-624.png](/static/img/3cde00_1695035642475-624.png)

**注意**: 管理者は、アセットに対してレベル 1 またはレベル 2 のいずれかのみを使用できますが、同時に使用することはできません。
