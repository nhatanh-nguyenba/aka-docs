スケジュールを設定すると、事前に計画された方法でタスクを実行できます。ニーズに応じてさまざまなスケジュールと時間を実行するようにエージェントに命令することで、タスクを実行できます。 [スケジュール] ページでは、新しいスケジュールを作成できます。また、以前に作成したすべてのスケジュールも表示され、さらに編集、有効化、無効化することができます。 [スケジュール] ページにアクセスするには、左側のメニューの [スケジュール] タブをクリックします。クリックすると、以前に作成したすべてのスケジュールを含むスケジュール一覧ページが開きます。

## **a.スケジュールを表示**

![image-20221028111221-36.png](/static/img/8fa96a_image-20221028111221-36.png)

上記の一般情報に加えて、**詳細**ページには、**スケジュール**の**パラメータ**と**休日設定**も表示されます。

- パラメータ: ここから、この**スケジュール**に関連付けられた**エージェント**と**タスク**のリストも表示できます。エージェントとタスクの詳細については、それぞれのセクションで説明します。

- 休日設定: ここから、エージェントの休日設定オプションを確認できます: 継続的に実行/休日を回避/次の営業日まで延期。

* スケジュールの無効化/有効化

スケジュールを無効にするには、対応する「アクション」ボタンをクリックし、「無効」を選択します。

![image-20221028143344-1.png](/static/img/5bdb2c_image-20221028143344-1.png)

クリックすると、システムはステータスを切り替えるための確認を求めます。

![image-20221028143400-2.png](/static/img/9494f7_image-20221028143400-2.png)

スケジュールを無効にすると、そのスケジュールのステータスが「**無効**」に変わります。

同様に、スケジュールを有効にするには、対応する [**アクション**] ボタンをクリックし、[**有効にする**] を選択します。

![image-20221028143503-3.png](/static/img/b2ba03_image-20221028143503-3.png)

## **d. スケジュールの削除**

削除するには、対応するスケジュールの「アクション」ボタンをクリックし、「削除」ボタンを選択する必要があります。

![image-20221028143538-4.png](/static/img/f0cd15_image-20221028143538-4.png)

あるいは、各**スケジュール**の前にチェックボックスをオンにすると、**スケジュール**の横に**削除**オプションが表示されます。
上部の **フィルタ** オプション。 「**アクション**」の横にある「**チェックボックス**」をクリックすると、表示されているすべてのスケジュールを選択できます
削除用。

![image-20221028143725-5.png](/static/img/337d8f_image-20221028143725-5.png)

クリックすると、システムにより削除が許可され、削除する前に確認が必要になります。

![image-20221028143745-6.png](/static/img/155ffd_image-20221028143745-6.png)

[**削除**] を押して操作を完了します。
