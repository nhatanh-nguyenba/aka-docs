# タスク

タスクは、1 つまたは複数のエージェント上のプロセスの実行インスタンスです。ワークフローを作成した後 (パッケージをエージェント グループに展開した後)、次のステップは、割り当てられたエージェントでワークフローを実行することです。これは、[タスク] ページから手動で行うことも、[スケジュール] ページから事前計画的に行うこともできます。

[タスク] ページには、デフォルトで、手動で開始されたかスケジュールによって開始されたかに関係なく、実行中のすべてのタスクと保留状態のタスクが表示されます。このページでは、ボタンをクリックするだけで、手動でタスクを作成または編集したり、タスクを開始または停止したり、タスクによって生成されたログを表示したりすることもできます。 「タスクの詳細」ウィンドウに詳細が表示され、障害が発生したタスクのトラブルシューティングに役立ちます。 「タスク」ページにアクセスするには、左側のメニューの「タスク」タブをクリックします。クリックすると、以下のように既存のすべてのタスクをリストしたタスク ページが表示されます。

![image-20221028093344-4.png](/static/img/b3516c_image-20221028093344-4.png)

| いいえ | コラム | 説明 |
| --- | --- | --- |
| 1 | アクション/ビュー | タスクのステータスに応じて、システムではタスクを管理するためのいくつかのアクションを実行できます。 VIEW: タスクとタスク ログのプロパティを表示します。 DELETE: タスクを削除します。 CLONE: 既存のタスクから同一のタスクを作成します。 各タスクに対応するチェックボックスを選択すると、フィルター オプションの横に削除オプションが表示され、センターからタスクを削除できます。 ！ 「アクション」の横にあるチェックボックスを選択すると、表示されているすべてのタスクが選択され、選択したタスクを一括削除できます。 |

| 3 | バージョン | akaBot Center に公開されたパッケージのバージョン。 |
| 4 | エージェント | タスクが実行されるエージェント。 |

## **a.タスクを表示**

タスクの詳細を表示するには、いくつかの方法があります。
**[タスク] ページから:** これにより、作成されたすべてのタスクを表示できます。左側の「表示」ボタンをクリックします。

![image-20221028103213-22.png](/static/img/301407_image-20221028103213-22.png)

**ワークフローの詳細ページから**: これにより、特定のワークフローで作成されたすべてのタスクを表示できます。 [**ワークフロー**] タブをクリックし、[**ワークフロー**] を選択し、[**タスク**] タブを選択してタスク アイテムを選択します。

![image-20221028103341-23.png](/static/img/17588c_image-20221028103341-23.png)

**エージェントの詳細ページから**: これにより、特定のエージェントの作成されたすべてのタスクを表示できます。 [**エージェント**] タブをクリック > [**エージェント**] を選択 > 下部にある [**タスク**] タブを選択し、タスク アイテムを選択します。

![image-20221028103513-24.png](/static/img/e55549_image-20221028103513-24.png)

**「スケジュールの詳細」ページから**: これにより、特定のスケジュールで作成されたすべてのタスクを表示できます。 [**スケジュール**] タブをクリック > [**スケジュール**] を選択 > 下部にある [**タスク**] タブを選択し、タスク アイテムを選択します。

![image-20221028103731-25.png](/static/img/8cc834_image-20221028103731-25.png)

クリックすると、[**詳細**] ページにリダイレクトされます。詳細は以下でご覧いただけます。

![image-20221028103810-26.png](/static/img/f325fb_image-20221028103810-26.png)

「**リスト**」ページにもある**一般**情報に加えて、**タスクの詳細**ページには、各タスクの特別な**パラメータ**と**ログの詳細**も含まれています。

![image-20230704144337-1.png](/static/img/ceb01d_image-20230704144337-1.png)

| いいえ | コラム | 説明 |
| --- | --- | --- |
| 1 | アクション | 当該タスクのログを管理するために利用可能なアクション。含まれるもの: 表示: [ログの詳細] ページにリダイレクトできます。削除: 各ログ項目のチェックボックスを選択すると、フィルタ オプションの横に削除オプションが表示され、ログ項目を削除できるようになります。 ！ 「アクション」の横にあるチェックボックスを選択すると、表示されているすべてのログが選択され、選択したログ項目を一括削除できるようになります。 |
|  | タイムスタンプ | アクションが実行された正確な時刻 |
|  | レベル | メッセージが記録された重大度レベル。次のオプションが利用可能です:**オフ、致命的、エラー警告、情報、デバッグ、詳細。** |
|  | メッセージ | ログに記録されたメッセージ。これは、Studio のログ メッセージ アクティビティを通じて記録されたメッセージである場合もあります。 |

akaBot Center では、無料の検索ボックス、エージェント名、タスクの状態によってタスク リストを検索できます。

![image-20221028104328-27.png](/static/img/c8502b_image-20221028104328-27.png)

| いいえ | 列/ラベル | 説明 | 種類 | 最大値 | 入力要件 |
| --- | --- | --- | --- | --- | --- |
| 1 | 検索ボックス | タスクに関連するすべての項目の名前で検索します。 | フリーテキスト |  |  |
| 2 | エージェント | 検索するタスクを実行するために割り当てられているエージェントを選択します | 入力検索 | 制限なし |  |
| 3 | 州 | 検索したいタスクの状態を選択します | 入力検索 | 制限なし |  |

## **c.タスクの作成/編集**

タスクを作成するには 3 つの方法があります。

- [スケジュール] ページでスケジュールを作成するか、エージェントをローカルで実行すると、タスクが自動的に作成されます。

- [タスク] ページでタスクを手動で作成することもできます。

- 既存のタスクのクローンを作成します。

**タスクを手動で作成するには:**
ページ右上の「新規作成」ボタンをクリックします。タスクを作成するためのフォームが表示されます。

![image-20221028104844-28.png](/static/img/ecb6b1_image-20221028104844-28.png)

![image-20221028104910-29.png](/static/img/f6a3d1_image-20221028104910-29.png)

| いいえ | 列/ラベル | 列/ラベル | 種類 | 最大値 | 必須ですか? | 入力要件 |
| --- | --- | --- | --- | --- | --- | --- |
|  | ワークフロー | 作成するタスクのワークフローを選択します | 入力検索 | 制限なし | はい |  |
|  | 検索 | 検索したいエージェントの名前を入力してください | 文字列 | 制限なし | いいえ |  |
|  | エージェント | 選択したワークフローに応じて、作成中のタスクを実行するエージェントを選択します。 | チェックボックス、複数選択 | 制限なし | はい |  |
|  | パラメータ | その他のパラメータ |  |  | いいえ |  |

あるいは、[**アクション**] ボタンをクリックし、[**クローン**] オプションを選択して同一のタスクを作成することもできます。この機能を使用すると、構成プロセス全体を実行することなく、タスク リストからタスクを迅速に実行できます。

このプロセスは、以前に開始したタスクがすでに最終状態に達していることを前提としています。まず、各タスクの開始時に対応する [**アクション**] ボタンをクリックした後、[**クローン**] を選択します。

![image-20221028105120-30.png](/static/img/cd2723_image-20221028105120-30.png)

このオプションをクリックすると、[**CLONE A TASK**] ウィンドウが表示され、タスクのデフォルトのパラメータが表示されます。

![image-20221028105146-31.png](/static/img/e5cfb7_image-20221028105146-31.png)

必要な変更を自由に行ってから、[**保存**] ボタンをクリックして作業内容を保存します。 [**CLONE A TASK**] ウィンドウが閉じ、実行が開始されます。

**注意:** 作成またはクローン作成後、タスクは自動的にエージェントにデプロイされ、すぐに実行されます。 [タスク] ページには、各タスクのステータスがリアルタイムで表示されます。

## **d.タスクを削除します**

タスクを削除するには、リスト内の 1 つのタスクの [削除] ボタンをクリックします。

![image-20221028105307-32.png](/static/img/7d7559_image-20221028105307-32.png)

あるいは、各タスクの前にチェックボックスをオンにすると、**削除** ボタンが **フィルタ** ボタンの横に表示されます。 [**アクション**] の横にある **チェックボックス** をオンにすると、表示されているすべてのタスクをユーザーが選択して一括削除できるようになります。

![image-20221028105402-33.png](/static/img/c69d5a_image-20221028105402-33.png)

システムは確認を求めるメッセージを表示します。 [**確認**] をクリックしてタスクを削除します。

![image-20221028105416-34.png](/static/img/af130a_image-20221028105416-34.png)
