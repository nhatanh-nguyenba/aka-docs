# ワークフロー

**ワークフロー**は、**パッケージ**と**エージェント グループ**の間の関連付けを表します。パッケージが**エージェント グループ**にリンクされるたびに、その**エージェント グループ**に属するすべてのエージェント マシンに自動的に配布されます。

[ワークフロー] ページでは、アップロードされたパッケージをエージェント グループに展開し、以前に作成した関連付けを管理し、すべてのワークフローを最新の状態に保つことができます。これにより、エージェント マシンにパッケージを配布し、[タスク] ページから作業をより迅速に実行できるようになります。 「ワークフロー」ページにアクセスするには、左側のメニューの「ワークフロー」タブをクリックします。クリックすると、以下のような既存のすべてのワークフローをリストするワークフロー ページが表示されます。

![image-20221028093815-5.png](/static/img/image-20221028093815-5.png)

| いいえ | 列/ラベル | 説明 |
| --- | --- | --- |
| 1 | アクション | 当該ワークフローのログを管理するために利用可能なアクション。含まれるもの: \* VIEW: [タスクの詳細] ページにリダイレクトされます。 \* 編集: 各ワークフローの詳細を編集できます。 \* 削除: 選択したワークフローをセンターから削除できます。 \* ジョブの開始: このワークフローに関連する新しいタスクをすばやく作成します。  \*スケジュールの追加: このワークフローに関連するスケジュールをすばやく追加します。各ワークフローに対応するチェックボックスを選択すると、フィルター オプションの横に削除オプションが表示され、ワー​​クフローを削除できるようになります。 「アクション」の横にあるチェックボックスを選択すると、表示されているすべてのワークフローが選択され、選択したワークフローを一括削除できます。 |
| 2 | ワークフロー名 | ワークフローの名前。整理のために、**パッケージ名\_エージェント グループ名** の形式でワークフロー名を作成することをお勧めします。 |
| 3 | パッケージ名 | akaBot Center に公開されたパッケージの名前 |
| 4 | パッケージバージョン | akaBot Center に公開されたパッケージのバージョン。パッケージのバージョンは、akaBot Studio で公開オプションを選択したときに決定されます。 |
| 5 | エージェントグループ | ワークフローが展開されるエージェント グループ |
| 6 | 説明 | パッケージの公開されている説明の現在のバージョン |
| 7 | 作成者 | このワークフローを作成するユーザー。 |

## **a.ワークフローを表示**

ワークフローを表示するには、目のボタンをクリックします。

![image-20221028095243-9.png](/static/img/image-20221028095243-9.png)

**ワークフロー リスト** ページに表示されるすべての一般情報に加えて、**ワークフロー詳細** ページには、ワークフローの作成/編集時に生成されるパラメータとマシン環境情報も含まれています。

さらに、更新履歴や当該ワークフローで実行されたタスクの一覧も確認できます。タスクの詳細については、「タスク」セクションで説明します。

![image-20221028094255-6.png](/static/img/image-20221028094255-6.png)

ここでは、タスクの **時間、名前、状態、エージェント** でタスク リストを検索できます。

![image-20221028094515-7.png](/static/img/image-20221028094515-7.png)

ユーザーは、各タスクの先頭にある**チェックボックス**を選択して、このワークフローの実行済みインスタンスを削除することもできます。 [**削除**] ボタンが表示されます。

![image-20221028094854-8.png](/static/img/image-20221028094854-8.png)

## **c.ワークフローの作成**

ワークフローを作成するには、ページの上部にある**「新規作成」** ボタンをクリックします。

![image-20221028095700-11.png](/static/img/image-20221028095700-11.png)

クリックすると、ワークフローを作成できるフォームが表示されます。

![image-20221028100318-14.png](/static/img/image-20221028100318-14.png)

| いいえ | 列/ラベル | 説明 | 種類 | 最大値 | 必須ですか? | 入力要件 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | パッケージ名 | 作成しているワークフローに必要なパッケージを選択します | 入力検索 | 制限なし | はい |  |
| 2 | パッケージバージョン | 上で選択したパッケージに応じて異なります 選択したパッケージのバージョンを選択してください | ドロップダウンリスト |  | はい |  |
| 3 | エージェントグループ | 選択したパッケージのバージョンに応じて異なります。作成しているワークフローを接続するエージェント グループを選択します。 | 入力検索 | 制限なし | はい |  |
| 4 | 説明 | 作成/編集しているワークフローの説明を入力します | 文字列 | 255文字 | いいえ |  |

## **d.ワークフローを編集する**

ワークフローを編集するには、**3 点** ボタンをクリックし、次に **編集** をクリックします。

![image-20221028095829-12.png](/static/img/image-20221028095829-12.png)

クリックすると、ワークフローの情報を編集できるフォームが表示されます。

![image-20221028100409-15.png](/static/img/image-20221028100409-15.png)

| いいえ | 列/ラベル | 説明 | 種類 | 最大値 | 必須ですか? | 入力要件 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | パッケージバージョン | 上で選択したパッケージに応じて異なります 選択したパッケージのバージョンを選択してください | ドロップダウンリスト |  | はい |  |
| 2 | エージェントグループ | 選択したパッケージのバージョンに応じて異なります。作成しているワークフローを接続するエージェント グループを選択します。 | 編集できません |  |  |  |
| 3 | 説明 | 作成/編集しているワークフローの説明を入力します | 編集できません |  |  |  |

パッケージ バージョンを編集したら、ワークフローの **パラメータ** と **マシン環境** 変数をさらに編集できます。複数の変数を追加できますが、1 つずつしか追加できません (上記と同じ)。

## **e.ワークフローの削除**

削除するには、**3 点** ボタンをクリックし、次に [削除**] をクリックします。**

![image-20221028101551-19.png](/static/img/image-20221028101551-19.png)

あるいは、各ワークフローの前にチェックボックスを選択すると、[フィルター] ボタンの隣に [削除] ボタンが表示されます。 [アクション] の横にあるチェックボックスをオンにすると、ユーザーは表示されているすべてのエージェントを一括削除のために選択できるようになります。

![image-20221028101717-20.png](/static/img/image-20221028101717-20.png)

クリックすると、システムによって確認メッセージが表示されます。

![image-20221028101740-21.png](/static/img/image-20221028101740-21.png)
