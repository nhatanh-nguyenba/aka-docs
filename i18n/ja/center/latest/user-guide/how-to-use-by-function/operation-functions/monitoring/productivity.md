# 生産性

このページでは、カレンダー上でエージェントのアクティビティを追跡および監視することで、エージェントを最大限に活用できます。

![image-20221101151901-1.png](/static/img/2be2de_image-20221101151901-1.png)

## **a. エージェントのアクティビティを表示します。**

エージェントのアクティビティを表示するには、メイン メニューから [監視] タブをクリックしてください。クリックすると、すべてのエージェントのアクティビティが週ごとにすぐに表示されます。

![image-20221101154859-5.png](/static/img/6c9116_image-20221101154859-5.png)

| いいえ | ボタン | 説明 |
| --- | --- | --- |
| 1 | 今日 | 今日のアクティビティビューに戻る |
| 2 | 進む/戻る | *前日/翌日*ビューに移動 |
| 3 | 月と日 | 任意の時期のアクティビティを表示したい場合は、**日付と月** ボタンをクリックするだけで、小さなカレンダーが表示され、希望の時間を選択できます。 |
| 4 | 日、週、月 | ビューを日/週/月に変更します |

## **b. エージェントのアクティビティをフィルタリング**

カレンダーの隅にあるエージェント名とマシン名を選択すると、必要なエージェントの正確なアクティビティを表示できます。![image-20221101154015-3.png](/static/img/5fcc23_image-20221101154015-3.png)

## **c. スケジュールを作成します**

ここから、エージェントの空き時間を確認できます。そのため、カレンダー上の空き時間領域をクリックすることで、エージェントの新しいスケジュールをすばやく作成できます。次に、必要な情報を入力するフォームが表示されます。 「保存」ボタンをクリックすると、新しいスケジュールがカレンダーに表示されます。

![image-20221101160835-6.png](/static/img/b36064_image-20221101160835-6.png)
