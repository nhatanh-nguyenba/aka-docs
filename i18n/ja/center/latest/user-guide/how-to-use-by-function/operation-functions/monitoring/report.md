# レポート

**レポート**は、現在センターに接続されている**エージェントとワークフロー**に関するアクティビティ統計情報を提供します。ここでは、成功したタスクの数、実行中、停止、または障害が発生したタスクの数などの情報を取得できます。選択したエージェントまたはワークフローによって実行されたタスクの合計数を確認することもできます。

レポートは **エージェント** および **ワークフロー** によって取得できます。 [**レポート**] ページにアクセスするには、左側のメニューの [**レポート**] タブをクリックします。

## **a.エージェントへのレポート**

ユーザーは、[**レポート**] タブ > [**エージェント**] タブの選択に移動して、エージェント レポートを表示できます。

![image-20221101161308-7.png](/static/img/550df6_image-20221101161308-7.png)

| いいえ | コラム | 説明 |
| --- | --- | --- |
| 1 | エージェント名 | アクティビティが報告されるエージェントの名前 |
| 2 | ワークフロー名 | アクティビティが報告されるワークフローの名前 |
| 3 | タスクは成功しました | 正常に実行されたタスクの総数。 |
| 4 | タスクの実行中 | 現在実行中のタスクの総数 |
| 5 | タスクが停止しました | 停止されたタスクの総数 |
| 6 | タスクが失敗しました | 実行中にエラーが発生したタスクの総数 |
| 7 | 合計 | これまでに開始されたタスクの合計数。 |

レポートを取得するには、ページの上部にあるフィルターを使用できます。

![image-20221101161709-9.png](/static/img/5b079b_image-20221101161709-9.png)

| いいえ | 列/ラベル | 説明 | 種類 | 最大値 | 入力要件 |
| --- | --- | --- | --- | --- | --- |
| 1 | 開始日 | エージェントのアクティビティ統計を取得する日付。 | 日付と時刻 |  |  |
| 2 | 終了日 | エージェントのアクティビティ統計を取得する日付。 | 日付と時刻 |  |  |
| 3 | エージェントグループの選択 | アクティビティ統計を表示するエージェント グループを選択します。 | 入力検索 | 制限なし |  |
| 4 | エージェントの選択 | アクティビティ統計を表示したいエージェントを選択します | 入力検索 | 制限なし |  |
| 5 | レポートを見る | [**表示**] をクリックしてレポートを生成します |  |  |  |
| 6 | エクスポート | レポートをダウンロードする |  |  |  |

レポートはローカル マシンにダウンロードされ、開かれます。

![image-20230704161007-1.png](/static/img/ee307e_image-20230704161007-1.png)
