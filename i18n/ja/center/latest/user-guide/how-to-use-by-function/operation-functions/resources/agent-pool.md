# エージェントプール

エージェント プールを使用すると、ワークフローに関連するエージェントの「プール」を作成できます。 ワークフローを実行する必要があるときはいつでも、その時点でプール内の利用可能なエージェントがワークフローを自動的に実行するため、エージェントを最大限に活用できます。

## **a.エージェント プールの作成**

ステップ 1: 「プール」内のエージェントが実行する必要があるワークフローを作成します。**オートメーション** タブを開く -> **ワークフロー** タブをクリック -> **新しいワークフローを作成** -> 作成した**正確なエージェント グループ**をフォームに入力 -> 保存

ステップ 2: エージェント プールを作成します: **[リソース**] タブを開きます -> [**エージェント プール**] タブをクリックします -> **[新規作成]** をクリックします (下の図のように):

![image-20221031170544-22.png](/static/img/165fc9_image-20221031170544-22.png)

次に、フォームが表示されますので、記入してください。

![image-20221031170959-23.png](/static/img/4fb8d8_image-20221031170959-23.png)

| いいえ | 列/ラベル | 説明 | 種類 | 最大値 | 必須ですか? | 入力要件 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | 名前 | エージェント プールの名前を入力します | 文字列 | 55 | はい |  |
| 2 | 説明 | 作成/編集しているワークフローの説明を入力します | 文字列 |  | いいえ |  |
| 3 | ワークフロー | 作成したワークフローを選択します | ドロップダウンリスト |  | はい |  |
| 4 | 説明 | 作成/編集しているワークフローの説明を入力します | 文字列 | 255文字 | いいえ |  |

「全般」タブの隣に、**コールバック設定**があります。この機能を使用すると、このプール内のエージェントがタスクを完了したときに、出力をサードパーティに送信できます。

![image-20221031171615-24.png](/static/img/dd3eef_image-20221031171615-24.png)

送信オプションは 3 つあります。

**デフォルトではコールバックなし**: 出力を送信しません

**Restful API (POST メソッド)**: 挿入した API を通じて出力を送信します。

![image-20221101104556-1.png](/static/img/839f13_image-20221101104556-1.png)

**Webhook**: 出力を Webhook アドレスに送信します

![image-20221101104633-2.png](/static/img/b0d4b5_image-20221101104633-2.png)
