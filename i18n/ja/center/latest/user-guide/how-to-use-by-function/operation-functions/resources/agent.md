# エージェント

エージェントは、akaBot Studio で構築されたプロセスを実行する実行ホストです。

[エージェント] ページでは、エージェントの追加、編集、ステータスの確認、実行時設定の変更、エージェントが割り当てられているエージェント グループを行うことができます。

**注意:** エージェントのステータスがビジー (プロセス実行中) の場合、ユーザー名、マシン、またはタイプを編集することはできません。エージェントページにアクセスするには、左側のメニューの「エージェント」タブをクリックします。クリックすると、以下のように既存のすべてのエージェントをリストするエージェント ページが表示されます。

エージェントに関する詳細に加えて、エージェントのすべてのタスク、ログ、接続検証ステータスもここで確認できます。

## **b.エージェントのタスクを表示**

「詳細」ページの下部には、この特定のエージェントによって実行されるすべてのタスクのリストが表示されます。

![image-20221031154931-5.png](/static/img/61d3b2_image-20221031154931-5.png)

タスクの詳細については、タスクのセクションで説明します。

## **c.エージェントのログを表示する**

「エージェントの詳細」ページからエージェントのログを表示するには、「タスク」タブの隣にある「ログ」タブをクリックしてビューを切り替えます。

![image-20221031155216-6.png](/static/img/6634c9_image-20221031155216-6.png)

[**フィルタ**] ボタンの横にある [**ログのエクスポート**] オプションをクリックして、さらに調査するためにログをエクスポートすることもできます。すべてのログまたはフィルタリングされた特定のログ項目をエクスポートできます。

![image-20221031155405-7.png](/static/img/ee2568_image-20221031155405-7.png)

* エージェント接続の確認

センターとエージェントが有効なユーザー名とパスワードを使用して正しく接続されていることを確認するには、[**確認**] ボタンをクリックします。

![image-20221031155517-8.png](/static/img/d5e373_image-20221031155517-8.png)

Agent接続時にユーザーがこのボタンをクリックすると、ID/パスワードが正しくログイン可能かどうかの認証チェックが行われます。

検証タスクの状態としては、次の 3 つが考えられます。

* **保留中**: 検証タスクが作成され、エージェントからの検証を待っています
* **成功**: エージェントは検証され、ユーザー名、パスワードは正しかったです。
* **FAULTED**: エージェントは検証されましたが、ユーザー名、パスワードが正しくありませんでした

検証タスクと検証ボタンはリアルタイムで更新されます。検証タブに保留中の検証タスクがある場合、検証ボタンは無効になります。
ユーザーが検証ボタンをクリックすると、検証タブが自動的に開き、右上隅に「検証中...」という通知も表示されます。

作成されたすべての確認リクエストを表示するには、[**エージェント] タブ > [エージェントの表示**]**詳細** > [**確認**] タブを選択します。

![image-20221031155727-9.png](/static/img/076928_image-20221031155727-9.png)

## **d.エージェントの作成/更新**

エージェントをセンターに接続する最初のステップは、センターにエージェントを作成することです。

**注意**: ステータス = BUSY (タスク実行中) のエージェントは編集できません。新しいエージェントを作成するには、画面の右上隅にある「新規作成」ボタンをクリックする必要があります。

![image-20221031155823-10.png](/static/img/474aef_image-20221031155823-10.png)

エージェントを編集するには、リスト内の 1 つのエージェントの [編集] ボタンをクリックする必要があります。または、エージェントの詳細ページで「編集」ボタンをクリックすることもできます。

![image-20221031155905-11.png](/static/img/5dd967_image-20221031155905-11.png)

* エージェントを検索

必要なエージェントを検索するには、選択したエージェント グループの検索ボックスにエージェント名を入力し、ステータスを検索します。

## **e.エージェントの削除**

削除するには、[**削除**] ボタンをクリックする必要があります。

![image-20221031160620-14.png](/static/img/6be193_image-20221031160620-14.png)

または、各エージェントの前にチェックボックスをオンにすると、[新規作成] ボタンの横に [削除] ボタンが表示されます。 [アクション] の横にあるチェックボックスをオンにすると、表示されているすべてのエージェントをユーザーが選択して一括削除できるようになります。

![image-20221031160701-15.png](/static/img/f5ef98_image-20221031160701-15.png)
