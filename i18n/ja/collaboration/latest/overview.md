# コラボレーションセンター

コラボレーション センターは、人間のオペレーターと自動ボットが連携してエンドツーエンドのビジネス プロセスを完了する akaBot Center 内の中心ハブです。完全に自動化されたボットの実行と並行して、人間の判断、承認、介入が必要なタスクを管理するための統合インターフェイスを提供します。コラボレーション センターは、R​​PA の速度と正確性と人間の意思決定機能を組み合わせることで、組織が完全には自動化できない複雑なワークフローを処理できるようにします。

コラボレーション センター内で、スーパーバイザとチーム リーダーは、すべてのアクティブなプロセスをリアルタイムで監視し、タスクの割り当てを表示し、ボトルネックを追跡し、必要に応じて作業を再割り当てすることができます。すべてのタスクは、ボットによって実行されたか人間によって実行されたかに関係なく、タイムスタンプ、結果、監査証跡とともに記録され、すべての操作にわたってコンプライアンスと説明責任が保証されます。タスクの期限が過ぎた場合、または例外によって人間の対応が必要な場合、通知とエスカレーションが自動的にトリガーされます。

コラボレーション センターは、役割ベースのアクセス制御もサポートしており、各参加者が自分の責任に関連するタスクのみを表示して実行できるようにします。 ERP、CRM、電子メール プラットフォームなどの外部システムとの統合により、コンテキスト データがヒューマン タスク フォームにシームレスに流れるようになり、手動によるデータ入力が減り、意思決定の品質が向上します。

## 主要コンポーネント

| コンポーネント | 説明 |
|---|---|
| タスク受信箱 | ログインしたユーザーに割り当てられたタスクを表示する個人の受信箱 |
| プロセスモニター | 実行中のすべてのプロセス インスタンスのリアルタイム ビュー |
| エスカレーションマネージャー | 期限を過ぎたタスクやエスカレーションされたタスクのダッシュボード |
| 監査ログ | ユーザーとタイムスタンプを含むすべてのタスクアクションの完全な履歴 |
| 通知センター | 新しい課題、期日、例外に関するアラート |
| 役割管理 | どの役割が特定のタスクを表示およびアクションできるかを構成する |

## はじめに

1. **akaBot Center** に移動し、左側のメニューから **Human-Bot Collaboration** を選択します。
2. **コラボレーション センター** をクリックして、メイン ダッシュボードを開きます。
3. 現在自分に割り当てられているタスクについては、[**タスク受信ボックス**] パネルを確認してください。
4. タスクを選択して詳細ビューを開き、コンテキスト データを確認してアクションを実行します。
5. [**プロセス モニター**] タブを使用して、実行中のすべてのプロセスの全体的な状態を表示します。
