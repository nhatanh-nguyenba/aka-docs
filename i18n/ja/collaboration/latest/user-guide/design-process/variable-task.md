# プロセス内で変数タスクを設定する方法

変数タスクは akaBot Center の軽量プロセス要素であり、デザイナーが完全なボット タスクを実行したり人間の入力を必要とせずに、ワークフローの任意の時点でプロセス変数の値を設定、更新、または計算できるようにします。変数タスクは、単純なデータ変換の実行、カウンターの初期化、文字列の連結、日付の計算、または他のプロセス変数から導出された式の結果の保存に使用されます。

バリアブル タスクは、エージェントに作業をディスパッチせずにプロセス エンジン内で完全に実行されるため、即座に完了し、プロセスの実行時間にオーバーヘッドが追加されません。これらは、次のタスクに渡す前にデータを再フォーマットまたは強化する必要があるステップ間で特に役立ちます。たとえば、変数タスクでは、姓と名変数を組み合わせてフルネームを作成したり、請求書日付変数に 30 日を加算して支払い期日を計算したりできます。

変数タスクは、算術演算、文字列操作、日付算術、条件 (3 項) 式などのさまざまな式タイプをサポートします。単一の変数タスク内で複数の変数の割り当てを定義でき、それぞれが順番に評価されるため、1 つの割り当ての結果を次の変数で使用できます。

## 変数タスクの構成

| プロパティ | 説明 |
|---|---|
| タスク名 | デザイナーでのこのタスクの表示名 |
| 変数の割り当て | 1 つ以上の変数名と式のペア |

## サポートされている式の構文

| 種類 | 式の例 | 結果 |
|---|---|---|
| 算数 | `${invoiceAmount} * 1.1` | 10%の税金が適用された請求額 |
| 文字列連結 | `${firstName} + ' ' + ${lastName}` | フルネーム文字列 |
| 日付の追加 | `addDays(${invoiceDate}, 30)` | 支払い期限 |
| 条件付き | `${status} == 'Approved' ? 'Yes' : 'No'` | 派生したブール型の文字列 |
| 型キャスト | `toInt(${quantityStr})` | 文字列を整数に変換する |
| 現在の日付 | `now()` | 現在のタイムスタンプ |
| ストリングアッパー | `toUpperCase(${countryCode})` | 大文字の国コード |

## サポートされている変数データ型

| 種類 | 説明 |
|---|---|
| 文字列 | テキスト値 |
| 整数 | 整数 |
| 10進数 | 浮動小数点数 |
| ブール値 | 真/偽 |
| 日付 | 時間のない日付 |
| 日時 | 日付と時刻 |
| JSON | 構造化されたJSONオブジェクト |

## 変数タスクを追加する手順

1. **プロセス デザイナー**でプロセスを開きます。
2. **変数タスク** 要素をタスク パレットからキャンバスにドラッグします。
3. 変数の計算を行う必要があるタスクの間に配置します。
4. フロー矢印を使用して、変数タスクを前後の要素に接続します。
5. [変数タスク] ノードをクリックして、[**プロパティ**] パネルを開きます。
6. わかりやすい **タスク名** (例: 「支払期日の計算」) を入力します。
7. [**変数の割り当て**] セクションで、[**+ 割り当ての追加**] をクリックします。
8. 各割り当てについて:
   - **変数名** を入力します (新しい変数を作成するか、既存の変数を選択します)。
   - 変数の **データ タイプ** を選択します。
   - **式**を入力して値を計算します。
9. 必要に応じて割り当てを追加します。それらは上から下に評価されます。
10. [**保存**] をクリックして完了します。
