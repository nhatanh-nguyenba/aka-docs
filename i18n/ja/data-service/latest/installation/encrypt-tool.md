# データサービス暗号化ツール
## **1.はじめに**

このドキュメントでは、ファイル \*.yml データ サービスを暗号化する方法について説明します。

## **2.ファイルの暗号化**

![1776047623395-661.png](/static/img/e59a7f_1776047623395-661.png)

## **3.使用方法**

* 要件: Java 8 以降
* 暗号化ツールを使用する (akaBot が提供)

![1776047678106-518.png](/static/img/7b9b68_1776047678106-518.png)

次の情報を入力します。

* 暗号化する文字列
* 暗号化パスワード: 暗号化するすべての文字列に同じパスワードを使用します。
* 暗号化アルゴリズム: デフォルトは`PBEWITHHMACSHA256ANDAES_256`

`application.yml`ファイル内の情報を更新します。

```
password: <the Secret string generated by the tool>
```

![1776047785456-188.png](/static/img/db8e93_1776047785456-188.png)

`application-dev.yml`および`application-prod.yml`で暗号化情報を構成します。

- 暗号化された値を次のように置き換えます:`ENC(`**ツール`)`の暗号化されたテキスト**
