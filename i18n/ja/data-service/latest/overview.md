# アカボット データ サービス

Akabot Data Service は、ロボティック プロセス オートメーション (RPA) プロジェクト向けに設計された一元化されたデータ ストレージ ソリューションです。安全でスケーラブルなコード不要のデータ モデリング機能を提供し、ロボットとワークフローが自動化エコシステム全体でビジネス データを効率的に保存、取得、処理できるようにします。

Data Service を使用すると、組織は構造化されたビジネス データを 1 か所に維持しながら、有人自動化シナリオと無人自動化シナリオの両方をサポートできます。

## 主な特徴

- **一元化されたデータ ストレージ** — akaBot プラットフォーム内でビジネス データを安全に保存および管理します。
- **コード不要のデータ モデリング** — データベース スクリプトを作成せずに、構造化データ エンティティを作成および管理します。
- **ワークフローの統合** — Akabot Studio で作成されたワークフローにデータ操作をシームレスに統合します。
- **リレーショナル データのサポート** — 構造化された関係を使用してデータを整理し、処理とレポートを改善します。
- **安全なデータ管理** — 組み込みのセキュリティ メカニズムで機密性の高いビジネス情報を保護します。
- **迅速な展開** — データ サービスを迅速にプロビジョニングして展開し、自動化プロジェクトですぐに使用できるようにします。
