# APIドキュメント

# **はじめに**

このガイドでは、認証、ユーザーとトークンの管理、エンティティとデータの操作、エージェント、アカウント、構成などのシステム関連 API をカバーするデータ サービス API の概要を説明します。これには、利用可能なエンドポイント、リクエストとレスポンスの構造、アクセス許可が含まれます。

詳細な API 情報については、**[DataService API (Developer Guide & Reference) v1.0.0.](https://ws3.akabot.com/s/ejTe5RHEu7TG3ZE)** を参照してください。
