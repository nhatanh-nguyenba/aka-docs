# データサービスの高度なフィルター

**ステップ 1:** Data Service で目的のエンティティの [**データ**] タブに移動します。

![image-20230630132517-1.png](/static/img/f9122d_image-20230630132517-1.png)

**ステップ 2**: 結果リストにさらにフィルタを適用するには、[**詳細フィルタ**] をクリックします。ポップアップの詳細フィルターが開きます。

![image-20230630132522-2.png](/static/img/bb7b93_image-20230630132522-2.png)

**ステップ 3**: ユーザーの詳細フィルター

1. 演算子を選択します。

   * **AND** - すべての条件に一致するエントリを返します。
   * **OR** - いずれかの条件に一致するエントリを返します。
2. 条件を定義します。

   * **フィールド** - 条件を適用するエンティティ フィールドを選択します。
   * **演算子** - 条件を選択します。
   * **値** - 条件の値を入力します。値はフィールドの型と同じ型である必要があります。たとえば、**Text** フィールドには文字列を使用し、**Number** フィールドには整数を使用します。
3. [**+ 条件**] をクリックして、必要な数の条件を追加します。

条件を削除するには、対応する**削除** アイコンをクリックするか、すべてのフィルタをクリアする場合は、**すべての条件をクリア** ボタンをクリックします。
4. [**フィルタ**] をクリックして検索を再実行します。ページが更新されて新しい結果が表示されます。
