# フィールドの作成

**ステップ 1**: まず、新しいフィールドを追加するエンティティを選択し、[フィールドの追加] ボタンをクリックします。

![image-20230630131330-1.png](/static/img/90375e_image-20230630131330-1.png)

**ステップ 2**: 新しいフィールドに次の情報を入力します。

* **表示名**: このフィールドに表示する名前を入力します。
* **名前**: 文字と数字のみで構成されるフィールドの一意の名前を入力します。この名前は、データベースにデータを保存するために使用されます。
* **タイプ**: テキスト、はい/いいえ、数値、日付/時刻などのオプションからフィールドに適切なデータ タイプを選択します。
  + テキスト: このフィールドを使用して、少量のテキストを保存します。
  + 数値: このフィールドを使用して数値を保存します。希望の小数点以下の桁数を指定します。
  + はい/いいえ: このフィールドを使用して、はいまたはいいえの値を選択します。
  + 日付/時刻: このフィールドを使用して、日付と時刻の両方を含む値を保存します。日付ピッカーを使用して日付と時刻を選択できます。
* **必須フィールド**: フィールドに値が必要な場合は、「必須フィールド」チェックボックスを選択します。
* **一意のフィールド**: フィールド内のデータが一意である必要がある場合は、[一意のフィールド] チェックボックスをオンにします。
* 必要に応じて、「詳細」をクリックして追加フィールドを表示します。

![image-20230630131340-2.png](/static/img/c55496_image-20230630131340-2.png)

**ステップ 3**: 必要な情報を入力したら、[保存] ボタンをクリックして新しいフィールドを保存します。
