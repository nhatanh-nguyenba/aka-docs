# エンティティの削除

**ステップ 1**: エンティティを削除するには、削除するエンティティの横にある 3 つの点をクリックし、[削除] オプションを選択します。

![image-20230630131159-1.png](/static/img/708254_image-20230630131159-1.png)

**ステップ 2**: [削除] ボタンをクリックして削除を確認します。

![image-20230630131206-2.png](/static/img/24d5e4_image-20230630131206-2.png)
