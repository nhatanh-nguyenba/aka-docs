# エンティティの編集

**ステップ 1**: まず、編集するエンティティの横にある 3 つの点をクリックします。

![image-20230630131024-1.png](/static/img/083974_image-20230630131024-1.png)

**ステップ 2**: 要件に応じてエンティティ名を編集します。

![image-20230630131031-2.png](/static/img/83055d_image-20230630131031-2.png)

**ステップ 3**: 必要な変更を加えたら、[保存] ボタンをクリックして変更を保存します。
