# フィールドを更新する

**ステップ 1:** まず、フィールドを編集するエンティティを選択します。 [フィールド] タブで、更新するフィールドの横にある三点アイコンをクリックします。

![image-20230630131504-1.png](/static/img/bb3902_image-20230630131504-1.png)

**ステップ 2:** 必要な変更を加えます

![image-20230630131515-2.png](/static/img/09e670_image-20230630131515-2.png)

**ステップ 3:** [保存] ボタンをクリックして変更を保存します
