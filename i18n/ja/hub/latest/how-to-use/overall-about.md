# 全体的に akaBot Hub について

## アイデアを共有する

**アイデアの共有** 「アイデアの共有」機能は、特定のプロセスに関する情報が限られているものの、自動化の可能性を探りたいと考えているユーザー向けに設計されています。この機能を使用すると、akaBot Hub は提供された情報に基づいてプロセスの自動化の可能性の予備評価を行うことができます。

## CoE 主導のアイデアを共有する
「CoE 主導のアイデアの共有」は、特定のプロセスに関する詳細情報を持っているユーザー向けに設計されています。この機能を使用すると、akaBot Hub は提供された情報に基づいてプロセスの可能性を評価できます。

## オートメーションを共有する
ユーザーはユーザーが実装した自動化を共有できます

> **注:** ユーザーは **アイデア フローのレビュー** にアクセスして詳細を表示できます。

![image-20230222150931-1.png](/static/img/image-20230222150931-1.png)

## ワークスペース
- **自動化パイプライン:** 自動化パイプラインにアクセスして、共有されたすべてのアイデアを確認して評価できます。
- **私の提出物:** 「私の提出物」にアクセスして、共有したすべてのアイデアや自動化を表示できます。
- **マイ コンポーネント:** 「マイ コンポーネント」にアクセスして、アップロードされたコンポーネントとダウンロードされたコンポーネントの両方を表示できます。
- **私のフォロー中:** 「私のフォロー中」にアクセスして、フォローしているアイデアや自動化を表示できます。
- **私のコラボレーション:** 「私のコラボレーション」にアクセスして、コラボレーションしているすべてのアイデアや自動化を表示できます。

![image-20230222151859-2.png](/static/img/image-20230222151859-2.png)

## 探検する
- **自動化の探索:** 「自動化の探索」にアクセスして、akaBot Hub と共有されているすべてのアイデアと自動化を表示できます。
- **コンポーネントの探索:** 「コンポーネントの探索」にアクセスして、akaBot Hub と共有されているすべてのコンポーネントを表示できます。
- **Explore People:** 「Explore People」にアクセスして、akaBot Hub のすべてのメンバーを表示できます。

![image-20230222151954-3.png](/static/img/image-20230222151954-3.png)

## 管理
- **役割の管理:** では、一般役割とコラボレーター役割の権限を追加、管理、設定できます。
- **ユーザーの管理:** を使用すると、akaBot Hub でユーザーを追加および管理できます。
- **アイデア評価:** フォームと式をカスタマイズしてプロセスを評価でき、そのフォームはユーザーがアイデアを共有するときに使用されます。
- **ドキュメント:** では、各アイデアの [ドキュメント] タブで使用できるドキュメント テンプレートを設定および管理できます。
- **カテゴリ:** では、組織に固有のカテゴリ構造を定義できます。
- **アプリ インベントリ:** を使用すると、共有フォームのアプリケーション セクションに表示される IT アプリケーション インベントリを設定および管理できます。
- **コスト設定:** では、各アイデアの [コストと利点] タブに表示されるコストを設定および管理できます。
- **費用便益の設定:** では、アイデアの費用と便益を計算する数式をカスタマイズできます。そして、この公式は akaBot Hub で共有されるすべてのアイデアに適用されます
- **コンポーネントの管理:** では、akaBot Hub にアップロードされたすべてのコンポーネントを表示および確認できます。

![image-20230222152035-4.png](/static/img/image-20230222152035-4.png)

## ダッシュボード
- **自動化プログラムのパフォーマンス:** は、カスタマイズされた期間にわたる自動化数の推移と、既存の自動化の推定利点を反映しています。
- **平均自動化コスト:** プログラムのコストの推移を視覚化します
- **計画レポート:** は、アイデア フェーズを除くあらゆるフェーズのすべての自動化のベースライン サイクル タイム、実際のサイクル タイム、デルタの集計ビューであり、登録された実装に必要な労力を過大評価しているか過小評価しているかを判断するのに役立ちます。
- **コスト レポート:** は、アイデア フェーズを除く、あらゆるフェーズにおけるすべての自動化のコストの集計ビューです。
