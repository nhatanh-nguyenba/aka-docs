# アクティベーション

## ステップ 1:
ブラウザを開きます。 **ハブ URL** または **localhost:8080** を使用して、akaBot Automation Hub に移動します。ログインに成功したら、**管理** オプションを選択し、**ライセンスの管理** をクリックします。

![1693967443811-153.png](/static/img/1693967443811-153.png)
## ステップ 2:
[**ライセンスの追加**] をクリックします。

![1693967526338-637.png](/static/img/1693967526338-637.png)
## ステップ 3:
- ライセンス電子メールから受け取ったライセンス キーを [ライセンス キー] フィールドに入力します。
- 「生成」をクリックします。
- [コピー] ボタンをクリックして、**アクティベーション リクエスト** コードをコピーします。

![1693967608755-831.png](/static/img/1693967608755-831.png)
## ステップ 4:
- インターネットにアクセスできるコンピューターで、**https://aegis.akabot.com/#/activation.** に移動します。
- **アクティベーション リクエスト** コードを **アクティベーション リクエスト** フィールドに貼り付け、**ファイルをダウンロード** ボタンをクリックします。 **アクティベーション応答** コードが生成され、**「Active.txt」** という名前のテキスト ファイルとしてダウンロードされます。

![1693967693505-260.png](/static/img/image-20220506134343-3.png)

## ステップ5:
- akaBot Automation Hub に戻ります。 「アクティブ化」タブを選択します。
- 「Active.txt」を選択してアップロードします。
- 「アクティブ化」ボタンをクリックします。

![1757746057637-852.png](/static/img/image-20220506134349-4.png)
