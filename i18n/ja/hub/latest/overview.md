# アカボットハブ

Akabot Hub は、自動化資産の中央マーケットプレイスおよびコラボレーション プラットフォームです。チームは再利用可能なワークフロー、アクティビティ パッケージ、テンプレート、ベスト プラクティス ドキュメントを Hub に公開して、組織の他のメンバーがそれらを見つけて再利用できるようにすることで、配信を加速し、重複した作業を防止します。

## 主な特徴

- **アセット マーケットプレイス** — チームまたは Akabot コミュニティによって公開されたワークフロー、アクティビティ パッケージ、テンプレート、コネクタを参照および検索します。
- **ワンクリックインポート** — ワンクリックであらゆるアセットを Akabot Studio に直接インポートします。
- **バージョン管理** — 各アセットには完全なバージョン履歴があります。消費者は特定のバージョンに固定することも、常に最新のバージョンを使用することもできます。
- **評価とレビュー** — 他の人が最適なソリューションを選択できるように、アセットに関するフィードバックを残してください。
- **ドキュメント ホスティング** - 各アセット ページには、README、使用例、および変更ログが含まれています。
- **ガバナンス** — 管理者は、資産を組織内で公開する前に承認を要求できます。

## 資産の種類

| タイプ | 説明 |
|------|-------------|
| **ワークフロー テンプレート** | 一般的なユースケース (請求書処理、オンボーディングなど) 向けに事前に構築されたワークフロー スケルトン。 |
| **アクティビティ パッケージ** | Studio に直接インストールできるカスタム NuGet アクティビティ パッケージ。 |
| **コネクタ** — | 一般的な SaaS アプリケーション用に事前構成された HTTP コネクタ。 |
| **プロセスの文書化** | プロセス定義文書 (PDD) とソリューション設計文書 (SDD)。 |

## はじめに

1. Akabot Center から Hub ポータルを開きます。
2. 自動化プロジェクトに関連するアセットを検索します。
3. [**Studio にインポート**] をクリックするか、パッケージを手動でダウンロードします。
4. Studio または Hub ポータルから [**公開**] をクリックして、独自のアセットを公開します。
