# Akabot Hub — リリースノート

## v3.2.0 — 2026 年 4 月

### 新機能
- **AI を活用した検索** — セマンティック検索は、正確なキーワードではなく意図を理解します。必要なものを説明してアセットを見つけます。
- **依存関係グラフ** — 更新を公開する前に、特定のアクティビティ パッケージにどのワークフローとパッケージが依存しているかを表示します。
- **プライベート チャネル** — ハブ内に招待専用チャネルを作成して、特定のチームまたはプロジェクトとアセットを共有します。

### 改善点
- Studio へのアセットのインポートでは、ダウンロード前に現在の Studio バージョンとの互換性が検証されるようになりました。
- パブリッシャーのプロフィールには、ダウンロード数、平均評価、応答時間の SLA が表示されるようになりました。
- Changelog エディターは、ライブ プレビュー パネルを使用して Markdown をサポートします。

### バグ修正
- 固定資産の検索では、新しい資産が公開されてから最大 60 秒間、古い結果が返されます。
- Studio で特定のバージョンに固定すると、代わりに最新バージョンが取得される問題を解決しました。
- 拡張属性を使用して macOS からアップロードされたパッケージの壊れたサムネイル プレビューを修正しました。

## v3.1.0 — 2026 年 1 月

### 新機能
- **ガバナンス承認ワークフロー** - 管理者は複数ステップの承認チェーンを定義します。すべての承認者が承認するまで、アセットは下書きのままになります。
- **使用状況分析** — パブリッシャーは、アセットのダウンロード傾向、アクティブなインストール、およびバージョンの採用率を確認できます。
- **コレクション** — ワンクリックで一括インポートできるように、関連するアセットを名前付きコレクション (例: 「請求書処理スイート」) にグループ化します。

### 改善点
- NuGet パッケージ検証では、既知の脆弱な依存関係をチェックし、発行者に警告するようになりました。
- Hub ポータルの検索結果は、アセット タイプ、評価、Studio の互換性、ライセンスによってフィルタリングできます。
- CI/CD パイプラインの一部としてプログラムによるアセット公開用に REST API が追加されました。

### バグ修正
- コレクションを削除すると、基礎となるアセットも削除される問題を修正しました。
- レビューの送信後に星の評価がリアルタイムで更新されない問題を修正しました。

## v3.0.0 — 2025 年 9 月

### 新機能
- **Akabot Hub v3 のリリース** — NuGet 互換フィード、REST API、Studio ディープリンク統合を備えたマーケットプレイスを再構築しました。
- **ワンクリックインポート** — Hub ポータル内からアクティブな Studio プロジェクトに Hub アセットを直接インストールします。
- **評価とレビュー システム** — 1 ～ 5 つ星の評価と自由記述のレビューを含む構造化されたフィードバックを残してください。

### 重大な変更
- Hub v2 アセット パッケージ (`.abpkg`) を Hub v3 で表示するには、標準の NuGet パッケージとして再公開する必要があります。

### バグ修正
- ゲスト ユーザーが直接 URL 経由でプライベート アセットをダウンロードできるようにする権限の問題が解決されました。
