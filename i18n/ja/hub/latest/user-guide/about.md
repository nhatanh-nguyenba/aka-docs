# akaBot ハブについて

## akaBot Hub の使命は何ですか?

![image-20230222160249-1.png](/static/img/image-20230222160249-1.png)

自動化の人気が高まっている現在の状況では、企業は通常、組織内の従業員が使用する自動化製品の作成を担当する専任の CoE チームを抱えています。従業員がこれらの自動化製品を利用すると、さらなる改善のアイデアが生まれ、それを CoE チームと共有できるようになります。 CoE チームはこれらのアイデアを実装し、自動化製品をよりユーザーフレンドリーで従業員にとって利用しやすいものにすることに取り組みます。時間の経過とともに、一部の従業員は自動化ソリューションを作成して実装する独自の能力を身につけ、レビューとガバナンスのために CoE チームに提出できるようになります。

企業が大規模な自動化を導入し続けるにつれて、自動化のアイデアや製品を効果的に管理する必要性がますます高まっています。ここで、akaBot Hub が登場します。**自動化統合プロセス全体を通じて企業の自動化製品とアイデアの管理を支援する**という使命を持っています。akaBot Hub は、アイデアの生成、実装、ガバナンスのための一元的な場所を提供することで、自動化プロセスを合理化し、企業が自動化のメリットを最大化できるように支援します。

## akaBot Hub が企業にもたらすメリットは何ですか?
akaBot Hub は、あらゆる企業の RPA Center of Excellence (CoE) がプロセスの自動化を推進できるように設計された製品です。

- 同じプラットフォーム上で数十万の自動化プロジェクトを一元管理
- BAが自動化アイデアに関する情報を収集するのを従業員が待たせるのではなく、従業員が自ら自動化アイデアを共有できるプラットフォームを提供することで、運用時間を最適化します。
- 従業員がプロセスの詳細情報の入力を完了した直後に、自動化のアイデアの実現可能性を数秒で評価します
- 自動化プロジェクト向けのドキュメントやソース コードなどの何百万ものファイルを一元管理および保存
- すべての自動化プロジェクトのコストとメリットを簡単に追跡
