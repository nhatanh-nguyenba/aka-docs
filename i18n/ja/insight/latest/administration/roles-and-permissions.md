## ユーザー役割の管理

Insights は、ADMINISTRATOR\_RESOURCE および DATASET\_RESOURCE という 2 つの主要な権限をサポートしています。 2 つの基本的な権限から、ユーザーにさまざまな役割を割り当てることができます。

システムに対する権限の影響を次の表に示します。

|  |  |  |
| --- | --- | --- |
| **権限** | **管理者\_リソース** | **データセット\_リソース** |
| ユーザーの割り当て | × |  |
| ビジネスデータセットの構成 |  | × |

### ロールの作成

1. インサイトへのアクセス。
2. 右上隅にある**設定**メニューに移動し、**管理**を選択します。
3. **役割**に移動し、**新規作成**ボタンを選択します。

![image-20230420153140-2.png](/static/img/image-20230420153140-2.png)

4. 新しいダイアログ ボックスが開き、ロール名を入力し、システムへの影響のレベルを選択します。

**注:** 増加する影響の範囲は、左から右、なしから削除までです。

      5. **保存** を選択してアクションを確認します。

### 役割の編集

1. 役割カードの左端にあるその他メニューに移動し、**更新** を選択します。

![image-20230420154728-3.png](/static/img/image-20230420154728-3.png)

2. 新しいダイアログ ボックスが開き、新しいロール名を入力し、システムへの影響のレベルを再選択します。

3. **保存** を選択してアクションを確認します。

### ビューの役割

1. 役割カードの左端にあるその他メニューに移動し、**表示** を選択します。

![image-20230420155357-4.png](/static/img/image-20230420155357-4.png)

2. 新しいダイアログ ボックスが開きます。[**キャンセル**] を選択してダイアログ ボックスを閉じます。
