# Tableau アカウントのリスト
## Tableau アカウントを管理する

Insights でダッシュボードを表示、編集、変更、または削除するには、ユーザーの Insights アカウントを既存の Tableau アカウントにリンクする必要があります。 Tableau アカウントの各タイプでは、ユーザーにさまざまなレベルのシステム アクセスが提供されます。具体的には次のとおりです。

1. Tableau Viewer: ユーザーはダッシュボードとビジュアライゼーションを表示できますが、データの操作や編集は許可されません。
2. Tableau Explorer: ユーザーは、ダッシュボードの表示と操作、データのフィルター処理、および基本的なデータ操作の実行を行うことができます。
3. Tableau Creator: 最高レベルのアクセスを提供し、ユーザーが新しいビジュアライゼーションやダッシュボードを作成したり、より高度なデータ操作や分析を実行したりできるようにします。
