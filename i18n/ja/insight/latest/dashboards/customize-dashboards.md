# ダッシュボードのカスタマイズ
## 新しいビューの作成

新しいビューを作成するには、次の手順を実行します。

ステップ 1: ユーザーは任意のビューで [**ダッシュボードの編集**] をクリックして、Tableau UI にアクセスします。

![image-20230519090636-1.png](/static/img/image-20230519090636-1.png)

この時点で、画面は組み込みの Tableau プラットフォームのビュー構成画面に切り替わります。

![image-20230519091334-2.png](/static/img/image-20230519091334-2.png)

**注:** akaBot Insights のビューは、ワークシートとダッシュボードの 2 つのタイプに分類されます。ダッシュボードには、作成者の好みや目的に応じて、1 つまたは複数のワークシートを含めることができます。

ステップ 2.1: 新しいワークシートを作成するには、ページの下部にある [**新しいワークシート**] アイコンをクリックするか、キーボード ショートカット Ctrl + Alt + T を入力します。

![image-20230519092843-3.png](/static/img/image-20230519092843-3.png)

または、メニュー バーの **[ワークシート]** をクリックし、**、[**新しいワークシート**] をクリックします。

![image-20230519094334-4.png](/static/img/image-20230519094334-4.png)

ステップ 2.2: 新しいダッシュボードを作成するには、ユーザーはページの下部にある **新しいダッシュボード** アイコンをクリックします。

![image-20230519095146-6.png](/static/img/image-20230519095146-6.png)

または、メニュー バーの [**ダッシュボード**] をクリックし、[**新しいダッシュボード**] をクリックします。

![image-20230519095546-8.png](/static/img/image-20230519095546-8.png)

ステップ 3: [**公開**] をクリックして、新しいビューを akaBot Insights のビューのリストに保存します。

![image-20230519103017-10.png](/static/img/image-20230519103017-10.png)

次に進む方法の詳細な手順については、以下にある Tableau のダッシュボード作成ガイドを読むことをお勧めします。

次のページは、Tableau UI を使用してダッシュボードを作成するのに役立ちます。

* [Tableau Learning](https://www.tableau.com/learn/training/20201): ユーザーに Tableau の基本的な知識と使用方法を提供します。
* [Tableau Documentation](https://www.tableau.com/support/help?_ga=2.148610935.1319140124.1684467311-897470386.1679390771): Tableau の公式ドキュメントと教材を提供します。
* [Tableau Gallery](https://public.tableau.com/en-us/gallery): Tableau ギャラリーは、他の人がダッシュボードをどのように設計しているかを学ぶことができる場所です。主に、アイデア、設計テクニック、さまざまな種類のチャートの組み合わせについて学ぶことに重点を置いています。
* [Tableau Viz Cookbook](https://public.tableau.com/app/profile/josh.weyburne/viz/CookBook/VizCookbook): 最も基本的なものから 52 種類のチャートをまとめたもの。
* [Tableau Chart Catalog](https://public.tableau.com/views/TheTableauChartCatalog/TableauChartExamples?:showVizHome=no): さまざまな目的に対応する幅広いクリエイティブなチャートを含む、基本的なタイプから高度なタイプのチャートまで 100 種類をまとめたもの。

## ビューの複製

ビューを複製するには、次の手順を実行します。

ステップ 1: ユーザーは任意のビューで [**ダッシュボードの編集**] をクリックして、Tableau UI にアクセスします。

ステップ 2: 複製するシートを右クリックし、[**複製**] をクリックします。

![image-20230519110519-11.png](/static/img/image-20230519110519-11.png)

ステップ 3: [**公開**] をクリックして、複製したシートを akaBot Insights のビューのリストに保存します。

## ビューのエクスポート

ユーザーはビジュアライゼーションをさまざまな形式（PDF ファイル、画像ファイル、CSV ファイル、Excel ファイル、PowerPoint ファイルなど）でエクスポートできます。

ステップ 1: ビューをエクスポートするには、**ダウンロード** アイコンをクリックします。

![image-20230519132033-1.png](/static/img/image-20230519132033-1.png)

ステップ 2: ダウンロードするファイル形式を選択します。

![image-20230519132129-2.png](/static/img/image-20230519132129-2.png)

## ビューの削除

akaBot Insights UI で削除されたビューは完全には削除されませんが、Tableau サイトに保存されます。

ビューを完全に削除するには、次の手順に従います。

ステップ 1: ユーザーは任意のビューで [**ダッシュボードの編集**] をクリックして、Tableau UI にアクセスします。

ステップ 2: 複製するシートを右クリックし、[**削除**] をクリックします。

![image-20230519132810-3.png](/static/img/image-20230519132810-3.png)

## 独自のダッシュボードを作成および編集する

テンプレートの使用に加えて、好みに応じてダッシュボードをカスタマイズできます。

**注**: ダッシュボードの作成/編集に習熟するには、[Tableau Documentation](https://help.tableau.com/current/pro/desktop/en-us/gettingstarted_overview.htm)(akaBot Insight の組み込みプラットフォーム) を参照してください。
