# キュー

![image-20230413161244-1.png](/static/img/image-20230413161244-1.png)

デフォルトでは、**キュー** ダッシュボードは次の編集可能なウィジェットで構成されています。

| ウィジェット名 | ウィジェットの種類 | 説明 |
| --- | --- | --- |
| **タスクの数** | 入力 | 現在までに処理されたすべてのタスクの累計を表示します。 |
| **キュー** | 入力 | キューの合計数。 |
| **ステータスごとのタスク** | 円グラフ | 各状態のタスクの合計数 (再試行されたタスクを含む): * **成功** * **放棄** * **失敗** |
| **日付ごとのタスク** | 縦棒グラフ | 1 日あたりのタスクの合計数と、**成功**と**失敗**のステータス別の内訳。 |
| **毎週の例外の内訳** | 縦棒グラフ | 各週の例外の合計数と、例外タイプ別の内訳: **ApplicationException** または **BusinessException**。 |
| **キューあたりの平均タスク期間** | 棒グラフ | キューごとの各アイテムの平均処理時間を秒単位で表示します。 |
