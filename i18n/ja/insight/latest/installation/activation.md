# アクティベーション
**ステップ 1**: ログインに成功すると、アクティベーション用に**ライセンス ファイル**を**アップロード**するように求められます。前に受け取った **ライセンス電子メール**からこのファイルを**ダウンロード**し、**アップロード**に進み、**確認ボタン**をクリックしてください。

![1693969773750-736.png](/static/img/1693969773750-736.png)

**ステップ 2:** アクティブ化後、ユーザーが akaBot Insights を使用できるようにするには、**ユーザーの割り当て**という追加の手順が 1 つ必要です。詳しい手順については、「[Users](#)」タブを参照してください。
