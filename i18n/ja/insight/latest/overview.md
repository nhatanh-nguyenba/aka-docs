# アカボット・インサイト

Akabot Insight は、生の実行データを実用的なビジネス インテリジェンスに変える分析およびレポート モジュールです。運用マネージャー、CoE チーム、経営幹部は、Insight ダッシュボードを使用して ROI を追跡し、ボトルネックを特定し、自動化投資についてデータに基づいた意思決定を行います。

## 主な特徴

- **事前に構築されたダッシュボード** — プロセスのパフォーマンス、ロボットの使用率、キューの健全性、エラー傾向をすぐに表示できるビュー。
- **カスタム ダッシュボード** — グラフ、KPI カード、テーブル、フィルターを備えたドラッグ アンド ドロップ ダッシュボード ビルダー。
- **リアルタイム データ** — ロボットが実行されるとメトリクスが自動的に更新されます。手動で更新する必要はありません。
- **ビジネス指標** — 自動化の結果を、FTE の削減、処理時間、トランザクションあたりのコストなどのビジネス KPI にマッピングします。
- **スケジュールされたレポート** - PDF または Excel レポートをエクスポートし、スケジュールに従って関係者に配信します。
- **ドリルダウン** — メトリックをクリックすると、根本原因分析のための個々の実行ログに移動します。

## 追跡される主要な指標

| メトリック | 説明 |
|--------|-------------|
| **実行成功率** | エラーなしで完了したワークフロー実行の割合。 |
| **平均処理時間** | 1 つのトランザクションまたはキュー項目を処理する平均時間。 |
| **ロボットの活用** | 各ロボットがアイドル状態でアクティブに実行されていた時間の割合。 |
| **FTE相当額の節約** | 期間ごとに自動化された手動作業の推定時間。 |
| **キューのスループット** | すべてのアクティブなキューで 1 時間あたりに処理されるアイテム。 |
| **例外率** | 100 トランザクションごとのビジネスおよびアプリケーションの例外。 |

## はじめに

1. Akabot Center を開き、**Insight** に移動します。
2. 事前に構築されたダッシュボードを選択するか、カスタムのダッシュボードを作成します。
3. 日付範囲、環境、およびプロセス フィルターを構成します。
4. チームのダッシュボードを共有またはスケジュールします。
