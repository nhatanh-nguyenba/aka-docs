# インサイトデータモデル
## はじめに

**akaBot Insights** は、利用可能なビジネス指標と運用上の洞察を組み合わせて使用する、データ モデリングと分析のプラットフォームとして機能する Web アプリケーションです。事前にロードされたダッシュボード テンプレートと、必要なメトリクスにわたって企業データを視覚化するユーザー定義のダッシュボードを使用すると、新しい分析洞察を発見し、パフォーマンス指標を追跡し、エラーを警告することができます。

**インサイト** ホームページには、ユーザーがアクセスできるすべての利用可能なダッシュボードが表示されます。デフォルトでは、すべてのユーザーは、製品に含まれるエージェント**s**、**プロセス**、**キュー、およびビジネス**のデフォルト ダッシュボードにアクセスできます。各ダッシュボードはカスタマイズできるほか、さまざまな形式でダウンロードできます。

## 洞察のためのデータ取り込み

テナントで Insights を有効にすると、センター データベースに存在する履歴データが取り込まれ、数分以内に Insights で使用できるようになります。

この初期セットアップの後、Center からの新しいデータが約 20 分ごとに取り込まれます。エージェント ログも約 20 分ごとに取り込まれます。最後のレコードやエラー ログだけでなく、すべてのエージェント ログが取り込まれることに注意してください。

注: Insights に取り込まれたデータは履歴的なものであり、変更することはできません。そのため、Center でプロセス、キュー、ロボットに変更が加えられた場合（名前の変更など）、既存のデータには影響しません。
