# スケールフロー - リリースノート

## v1.0.0 - 初期ドキュメントのベースライン

**ハイライト**
- Scale Flow モジュールに`overview`を追加しました。
- 中核となる範囲を定義しました: アカウント設定、チャネル統合、AI 自動応答、レスポンダーのワークフロー、チケットの使用、連絡先管理。

**注意事項**
- このページは、Scale Flow の機能が進化するにつれて継続的に更新されます。
