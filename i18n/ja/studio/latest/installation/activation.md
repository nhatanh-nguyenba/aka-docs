# ライセンスのアクティベーション

**以下から適切なアクティベーション方法を選択してください:**

**1.標準 (またはスタンドアロン) ライセンス:**

- akaBot Studio コンピュータ **(デバイス A)** に **アクティブなインターネット接続**がある場合は、**オンライン アクティベーション** 方法を使用して、便利で迅速なアクティベーションを行うことができます。

- akaBot Studio コンピューターがインターネットに接続できない場合は、オフライン アクティベーション方法を使用してください。正常にアクティブ化するには、次のことを確認する必要があります。

  1. インターネットに接続された別のデバイス **(デバイス B)** を使用して、akaBot ライセンス管理システムからライセンス ファイルを取得します。
  2. ライセンス ファイルをデバイス B からデバイス A に転送するには、デバイス A とデバイス B を接続します。

**2.エンタープライズ (またはネットワーク) ライセンス**

- 独自の akaBot Center を通じてアクティブ化できます。 akaBot Studio コンピュータが akaBot Center にアクティブに接続されていることを確認してください。
- ネットワークがプロキシ サーバー経由でインターネットまたは akaBot Center に接続する必要がある場合は、アクティブ化する前にプロキシ情報があることを確認してください。

### オンラインアクティベーション

**ステップ 1** - Studio を開始します。 Windows のインストールでは、オプション「LMS (ライセンス管理システム)」を選択し、「次へ」ボタンを押します。

![image-20230717142628-1.png](/static/img/image-20230717142628-1.png)

**ステップ 2** – オンラインアクティベーションオプションを選択し、「次へ」ボタンを押します。

![image-20230717142628-2.png](/static/img/image-20230717142628-2.png)

**ステップ 3** – ライセンス情報を入力し、「次へ」ボタンを押します。

1 - **ライセンス サーバー**: https://aegis.akabot.com/

2 - **ライセンス キー**: ライセンス電子メールに記載されているライセンス キーを入力してください。

3 - **ネットワーク オプション** (オプション): プロキシ サーバーを使用する場合は、**ネットワーク オプション** の下にプロキシ情報を入力してください。

![image-20230717142628-3.png](/static/img/image-20230717142628-3.png)

**ステップ 4** - ボタン **「アクティブ化」** を押します。 **アクティベーションが成功**すると、akaBot Studio に以下のメッセージが表示されます。

![image-20230717142628-4.png](/static/img/image-20230717142628-4.png)

以下のエラーメッセージが表示された場合は、ネットワーク接続を確認してください。

![image-20230717142628-5.png](/static/img/image-20230717142628-5.png)

### オフラインアクティベーション

**akaBot Studio コンピューター (デバイス A):**

**ステップ 1** - Studio を開始します。 Windows のインストールでは、オプション「LMS (ライセンス管理システム)」を選択し、「次へ」ボタンを押します。

![image-20230717142905-6.png](/static/img/image-20230717142905-6.png)

**ステップ 2** - オフラインアクティベーションオプションを選択し、「次へ」ボタンを押します。

![image-20230717142905-7.png](/static/img/image-20230717142905-7.png)

**ステップ 3** – ライセンス電子メールにライセンス キーを入力し、[次へ] ボタンを押します。

![image-20230717142905-8.png](/static/img/image-20230717142905-8.png)

**ステップ 4** - 以下の **アクティベーション リクエスト**を **コピー**し、[次へ] をクリックします。

![image-20230717142905-9.png](/static/img/image-20230717142905-9.png)

![image-20230717142905-10.png](/static/img/image-20230717142905-10.png)

**ステップ 5** – **アクティベーション リクエスト**を **インターネット アクセス デバイス (デバイス B)** にコピーします。

**デバイス B:**

**ステップ 6** - https://aegis.akabot.com/#/activation に移動します。

AkaBot Studio コンピュータ (デバイス A) からコピーした **アクティベーション リクエスト** を貼り付け、[キャプチャ] を選択して、**ファイルをダウンロード**して **アクティベーション レスポンス ファイル** (**activate.txt** という名前) をダウンロードします。

![image-20230717142905-11.png](/static/img/image-20230717142905-11.png)

**ステップ 7** – アクティベーション応答ファイルをデバイス B からデバイス A にコピーします。

**akaBot Studio コンピューター (デバイス A):**

**ステップ 8** – アクティベーション レスポンス ファイルを参照するか、ファイル内のアクティベーション レスポンスをアクティベーション ボックスにコピーして、「アクティベート」ボタンを押します。

![image-20230717142905-12.png](/static/img/image-20230717142905-12.png)


**アクティベーションが成功**すると、akaBot Studio に以下のメッセージが表示されます。

![image-20230717142905-13.png](/static/img/image-20230717142905-13.png)

### akaBot センターを通じてアクティブ化する

**ステップ 1** - Studio を開始します。 Windowsのインストールでは、オプション「akaBot Center」を選択し、「次へ」ボタンを押します。

![image-20230717142905-14.png](/static/img/image-20230717142905-14.png)

**ステップ 2** – ライセンス情報を入力します。

- **センター URL**: akaBot センターの URL
- **ライセンス キー**: ライセンス電子メールに記載されているライセンス キーを入力してください。
- **ネットワーク オプション (オプション)**: プロキシ サーバーを使用する場合は、**ネットワーク オプション** の下にプロキシ情報を入力してください。
- 「有効化」ボタンを押します

![image-20230717142905-15.png](/static/img/image-20230717142905-15.png)

**アクティベーションが成功**すると、akaBot Studio に次のメッセージが表示されます。

![image-20230717142905-16.png](/static/img/image-20230717142905-16.png)

以下のエラーメッセージが表示された場合は、ネットワーク接続を確認してください。

![image-20230717142628-5](/static/img/image-20230717142628-5.png)
