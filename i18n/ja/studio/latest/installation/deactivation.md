# ライセンスの無効化
## 始める前に
akaBot Studio は、アクティベーション方法と **同じ** 非アクティベーション方法を使用します。

1. akaBot Studio が **オンライン**でアクティブ化されている場合、非アクティブ化方法は **オンライン非アクティブ化** になります。

akaBot Studio コンピュータがインターネットに接続されていることを確認してください。
2. akaBot Studio が **オフライン**でアクティブ化されている場合、非アクティブ化方法は **オフライン非アクティブ化** になります。
必ずご確認ください:
    - akaBot ライセンス管理システムからライセンス ファイルを取得するには、別のインターネット アクセス デバイスを用意します。
    - akaBot Studio コンピュータとインターネット アクセス デバイスを接続して、非アクティブ化ライセンス ファイルを転送します。
3. akaBot Studio が **akaBot センターを通じて**アクティブ化されている場合、非アクティブ化も **akaBot センターを通じて**されます。
akaBot Studio コンピューターが akaBot センターに接続されていることを確認してください。
## オンラインでの無効化
akaBot Studio をオンラインで無効化するには、以下の手順に従ってください。

1 - 「ファイル」を選択します

2 - 「概要」を選択します

3 - 「ライセンスの非アクティブ化」リンクをクリックします。

4 – 非アクティブ化アクションを確認します。

![image-20230717145333-2.png](/static/img/image-20230717145333-2.png)

![image-20230717145333-3.png](/static/img/image-20230717145333-3.png)

**正常に**非アクティブ化されると、akaBot Studio に以下のメッセージが表示されます。

![image-20230717145333-4.png](/static/img/image-20230717145333-4.png)

コンピュータがインターネットに接続されていない場合、akaBot Studio は以下の **エラー** メッセージを表示します。ネットワーク接続を確認してください。

![image-20230717145333-5.png](/static/img/image-20230717145333-5.png)

## オフラインでの非アクティブ化
akaBot Studio をオフラインで非アクティブ化するには、以下の手順に従ってください。

**akaBot Studio コンピューター (デバイス A):**

**ステップ 1** - akaBot Studio を開き、ライセンス機能の非アクティブ化に移動します。

1 - 「ファイル」を選択します

2 - 「概要」を選択します

3 - 「ライセンスの非アクティブ化」リンクをクリックします。

4 – 非アクティブ化アクションを確認します。

![image-20230717145539-7.png](/static/img/image-20230717145539-7.png)

![image-20230717145539-8.png](/static/img/image-20230717145539-8.png)

**ステップ 2** - 非アクティブ化リクエストをテキスト ファイルにコピーし、ファイルに Deactivation.txt という名前を付けて、「次へ」ボタンを押します。

![image-20230717145539-9.png](/static/img/image-20230717145539-9.png)

次のページで「次へ」ボタンを押してください

![image-20230717145539-10.png](/static/img/image-20230717145539-10.png)

**ステップ 3** – ファイルを akaBot Studio コンピューターから **インターネット アクセスのあるデバイス (デバイス B)** にコピーします。

**デバイス B:**

**ステップ 4** – https://aegis.akabot.io/#/deactivation にある akaBot ライセンス管理システムに移動します。

1 – 非アクティブ化ファイルの非アクティブ化リクエストをテキスト ボックスに貼り付けます。

2 – キャプチャを確認する

3 – 「ライセンスの非アクティブ化」ボタンを押して、非アクティブ化応答ファイル (Deactivate.txt という名前) をダウンロードします。

**ステップ 5** – 非アクティブ化応答ファイルをデバイス A にコピーします。

**デバイス A:**

**ステップ 6** – 非アクティブ化応答ファイルを参照するか、ファイル内の非アクティブ化応答を非アクティブ化ボックスにコピーして、「非アクティブ化」ボタンを押します。

![image-20230717145539-12.png](/static/img/image-20230717145539-12.png)
**正常に**非アクティブ化されると、akaBot Studio に以下のメッセージが表示されます。

![image-20230717145539-13.png](/static/img/image-20230717145539-13.png)

## akaBot センターを通じて非アクティブ化する
akaBot センターを通じて akaBot Studio を非アクティブ化するには、以下の手順に従ってください。

1 - 「ファイル」を選択します

2 - 「概要」を選択します

3 - 「ライセンスの非アクティブ化」リンクをクリックします。

4 – 非アクティブ化アクションを確認します。

![image-20230717150234-14.png](/static/img/image-20230717150234-14.png)

![image-20230717150234-15.png](/static/img/image-20230717150234-15.png)

![image-20230717150234-16.png](/static/img/image-20230717150234-16.png)

**正常に**非アクティブ化されると、akaBot Studio に以下のメッセージが表示されます。

![image-20230717150234-17.png](/static/img/image-20230717150234-17.png)
コンピュータが akaBot Center に接続されていない場合、akaBot Studio には以下のエラー メッセージが表示されます。ネットワーク接続を確認してください。

![image-20230717150234-18.png](/static/img/image-20230717150234-18.png)
