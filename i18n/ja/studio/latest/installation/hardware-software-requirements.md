# ハードウェアとソフトウェアの要件

## ハードウェア要件

akaBot Studio と Agent を使用するには、コンピューターが以下のハードウェア要件を満たす必要があります。

|  | 前提条件 | おすすめ |
|---|---|---|
| **CPU** | 2 × 1.8 GHz 64 ビット | 4×2.4GHz 64ビット |
| **RAM** | 4 GB の利用可能なシステム メモリ | 8 GB の利用可能なシステム メモリ |

## ソフトウェア要件

akaBot Studio と Agent を使用するには、コンピュータが以下のソフトウェア要件を満たす必要があります。

| | サポートされているバージョン |
|---|---|
| **オペレーティング システム** | Windows 10<br/>Windows 10N<br/>Windows 11<br/>Windows Server 2016<br/>Windows Server 2019<br/>Windows Server 2022<br/>Windows Server 2025 |
| **.NET フレームワーク** | 最小: 4.5.2 |
