# インストール

## 始める前に

1. akaBot Studio と Agent は **同じ** インストーラーに含まれています。このガイドラインは、両方のコンポーネントを **スタンダード** (または **スタンドアロン**) と **エンタープライズ** (または **ネットワーク**) **エディション**の両方にインストールするために使用されます。

2. インストールする前に、お使いのコンピュータが **ハードウェアおよびソフトウェアの要件**を満たしていることをご確認ください。

3. インストーラーのダウンロード リンクを含む**akaBot ライセンス電子メール**をお持ちであることをご確認ください。まだの場合は、一時的な使用のために **2 か月の試用ライセンス**をリクエストしてください。

### インストールガイド
#### 1.ビデオガイド

より直感的なインストール手順については、以下のビデオガイドをご覧ください。

#### 2. 詳細な手順

**ステップ 1** - ライセンス電子メールに記載されているリンクからインストーラーをダウンロードします。

**ステップ 2** - zip ファイルを解凍し、管理者としてセットアップを実行します。


![1689060128056-950.png](/static/img/1689060128056-950.png)

コンピュータに akaBot プラットフォームがすでにインストールされている場合はステップ 3 に進み、そうでない場合はステップ 4 に進みます。

**ステップ 3** - [akaBot プラットフォームの削除] オプションを選択し、[完了] ボタンを押して、以前の akaBot エディションを削除します。

![image-20230711142249-1.png](/static/img/image-20230711142249-1.png)

手順 2 に戻ってインストールを再開します。

**ステップ 4** - [次へ] ボタンを押してインストールを開始します。

![image-20230711142249-2.png](/static/img/image-20230711142249-2.png)

**ステップ 5** - 使用許諾契約をよくお読みください。規約に同意する場合は、「同意する」を選択してから「次へ」を選択し、同意しない場合は「キャンセル」を押してください。

![image-20230711142249-3.png](/static/img/image-20230711142249-3.png)

**ステップ 6** - インストールフォルダーを選択します。

1 – デフォルトでは、akaBot はフォルダー「C:\Program Files\FPT Software\akaBot Platform」にインストールされます。 「参照」ボタンをクリックしてからフォルダーを選択するように変更できます。

2 – コンピュータ上の誰でも (任意のプロファイル) akaBot を使用できるようにしたい場合は、「Everyone」を選択し、それ以外の場合は「Just me」を選択します。

3 – 「次へ」をクリックします

![image-20230711142249-4.png](/static/img/image-20230711142249-4.png)


**ステップ 7** – 「次へ」ボタンを押してインストール設定を確認します。

![image-20230711142249-5.png](/static/img/image-20230711142249-5.png)

**ステップ 8** - インストールが進行するまで 1 ～ 2 分待ちます。

インストールが成功したら、「閉じる」ボタンを押して終了します。

![image-20230711142249-6.png](/static/img/image-20230711142249-6.png)

これで、**akaBot Studio と Agent** のショートカットがコンピュータ上に表示されるはずです。

![image-20230711142249-7.png](/static/img/image-20230711142249-7.png)

**次のステップ**

- 機能のロックを解除するには、**Studio と Agent をアクティブ化**する必要があります。
- Web ブラウザ上で akaBot を自動化するには、**akaBot Web 拡張機能をインストール**する必要がある場合があります。
