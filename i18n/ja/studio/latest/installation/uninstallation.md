# アンインストール
以下のこの手順では、**akaBot Studio** がデバイスから削除されます。

**ステップ 1** - akaBot を**アンインストール**するには、検索バーに「**コントロール パネル**」と入力して開きます。

![image-20220506111003-1.png](/static/img/image-20220506111003-1.png)


**ステップ 2** - **表示方法:「大きいアイコン」** を選択し、**「プログラムと機能」** を選択します。

![image-20220506111112-2.png](/static/img/image-20220506111112-2.png)

**ステップ 3** - 次に、**akaBot Platform** を **ダブルクリック**して **アンインストール**します

![image-20220506111126-3.png](/static/img/image-20220506111126-3.png)
