# アカボットスタジオ

Akabot Studio は、自動化ワークフローを構築、テスト、公開するビジュアル ワークフロー デザイナーです。ドラッグ アンド ドロップ キャンバス、豊富なアクティビティ ライブラリ、組み込みのデバッグ ツールを提供するため、開発者やシチズン オートメータは定型コードを記述せずに堅牢なボットを作成できます。

## 主な特徴

- **ビジュアル ワークフロー キャンバス** - アクティビティをドラッグ アンド ドロップして接続し、あらゆるビジネス プロセスをモデル化します。
- **豊富なアクティビティ ライブラリ** — UI オートメーション、ファイル処理、データ操作、Web サービスなどのための何百もの組み込みアクティビティ。
- **統合デバッガー** — デザイナー内で直接実行をステップ実行し、変数値を検査し、ブレークポイントを設定します。
- **バージョン管理** — チームコラボレーションのために、ワークフローを Git リポジトリにコミットしてプッシュします。
- **センターに公開** — ワンクリックでワークフローをパッケージ化し、Akabot Center に公開します。
