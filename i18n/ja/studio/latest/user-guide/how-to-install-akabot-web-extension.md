# Akabot Web Extension のインストール方法

## 1. はじめに
Akabot Studio 内で Web 自動化機能を有効にするには、Akabot ブラウザ拡張機能が必要です。拡張機能がユーザーの Web ブラウザーにインストールされて有効になると、Akabot Studio は自動化の記録と実行中に Web 要素をネイティブに検出し、操作できるようになります。

## 2. 前提条件
Akabot ブラウザ拡張機能をインストールする前に、次の条件が満たされていることを確認してください。

### 2.1 システム要件
- Windows 10/11、Windows Server 2016/2019/2022。
- Akabot Studio がインストールされている (サポートされているバージョン)
- ユーザーはローカルインストール権限を持っています
### 2.2 ブラウザの要件
- Google Chrome (サポートされているバージョン: 3.0.0、3.0.3) または
- Microsoft Edge (Chromium ベース、サポートされているバージョン: 3.0.3)
- Firefox (サポートされているバージョン: 1.1.6、2.0.0、2.0.1)
## 3. Google Chrome 用の Akabot 拡張機能をインストールする
### 3.1 Akabot Studio からインストールを起動する
- Akabot Studio を開く
- **オプション / 拡張機能 / Chrome** に移動します
- **「インストール」** をクリックします

![9453eb_1773283802504-331.png](/static/img/9453eb_1773283802504-331.png)

- 「インストール」をクリックすると、システムは実行中のブラウザ インスタンスを確認します。
- ブラウザが現在開いている場合は、インストールを続行する前に、「OK」をクリックして実行中のブラウザ インスタンスをすべて閉じるように求める確認ダイアログが表示されます。

![1773283826554-551.png](/static/img/1773283826554-551.png)

- ブラウザ インスタンスが実行されていない場合、インストールは自動的に続行され、成功メッセージが表示されます。 「OK」をクリックして完了します。
### 3.2 Chrome で拡張機能を有効にする
- Chromeを再度開きます。拡張機能を有効にするように求める通知ポップアップが表示されます。 **「拡張機能を有効にする」** をクリックして有効にします。

![1773283936766-583.png](/static/img/1773283936766-583.png)

- または、[拡張機能] に移動し、トグル ボタンを使用して拡張機能をオンにすることもできます。

![1773283989799-603.png](/static/img/1773283989799-603.png)

トグルを有効にすると、AkaBot はブラウザと対話して自動化アクティビティを実行できるようになります。
