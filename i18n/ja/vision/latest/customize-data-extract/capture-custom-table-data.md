# カスタムテーブルデータのキャプチャ
抽出スキーマの基本要素はデータ フィールドです。ただし、akaBot Vision を使用すると、テーブルなどのさらに複雑な構造をキャプチャできます。

## 事前定義されたテーブルフィールドの追加
ドキュメントからデータをキャプチャするときに一部のフィールドが欠落している場合は、キュー設定の [キャプチャするフィールド] タブに移動します。このタブでは、事前トレーニングされたデータ フィールドを管理し、どのフィールドを抽出するかを選択できます。

![image-20220421003652-1.png](/static/img/image-20220421003652-1.png)

## 新しいテーブルの追加
新しいテーブルを作成するには、[フィールドの追加]をクリックし、データ型=「テーブル」を選択し、新しいテーブルを作成する値を入力します。
