# データ抽出用のフィールドを構成する
各パイプラインは、akaBot Vision が抽出するデータ フィールドの構造を定義します。

## 説明
この構造を編集する場合、次の 2 つのオプションがあります。

- 事前トレーニングされたデータ フィールドを使用する – AkaBot Vision の汎用 AI エンジンは、特定のデータ フィールドを認識するように事前トレーニングされており、AI の追加トレーニングなしでデータの抽出を開始できます。
- カスタム データ フィールドを定義し、専用 AI エンジンをトレーニングしてそれらを自動的に抽出します。 (このオプションは 2022 年第 3 四半期に利用可能になります)

## 事前トレーニングされたデータフィールドを使用してデータ抽出をカスタマイズする方法
事前トレーニングされたデータ フィールドの使用は簡単です。この機能はすぐに使用できるため、セットアップが簡単です。

akaBot Visionにログイン後、右上隅の「設定」ボタンをクリックして設定画面に移動します。

![image-20220421002856-1.png](/static/img/image-20220421002856-1.png)

1. 構成するパイプラインを選択します。
2. これにより、パイプライン設定画面が開き、「キャプチャするフィールド」タブを開くことができます。
3. このタブでは、処理しているドキュメントとの関連性に応じてフィールドを選択または選択解除できます。
4. 完了したら、「保存」をクリックして設定を保存します。

![image-20220421002856-2.png](/static/img/image-20220421002856-2.png)
