# 手動でエクスポートする
スタッフは、文書をエクスポートすることで、akaBot Vision 内の文書の結果を取得できます。

書類の確認後、スタッフは結果をエクスポートできます。この記事では、ドキュメントをエクスポートする方法を説明します。

**ステップ 1**: 「確認済み」タブに移動します

![image-20221021140511-1.png](/static/img/image-20221021140511-1.png)

**ステップ 2**: エクスポートする必要があるドキュメントを選択し、[エクスポート] アイコンをクリックします。

![image-20221021140817-3.png](/static/img/image-20221021140817-3.png)

**ステップ 3:** スタッフがエクスポートするファイルの種類を選択し、[OK] ボタンをクリックします。

![image-20221021143844-4.png](/static/img/image-20221021143844-4.png)
