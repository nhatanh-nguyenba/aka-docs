# 新しいユーザーを作成する
**説明**

登録フォームを使用して akaBot Vision で新しいアカウントを作成すると、テナント アカウントと管理者権限を持つユーザー アカウントの両方が作成されます。

その後、管理者ユーザーとして、パイプラインやドキュメントへのアクセスをチームの他のメンバーと簡単に共有し、組織内での役割を指定できます。

## 新しいユーザーアカウントを作成する

**ステップ 1:** ログイン後、上部のメニュー バーで **テナント設定** を選択し、左側のメニューで **ユーザー** をクリックします。

![1704972347578-788.png](/static/img/1704972347578-788.png)

**ステップ 2**: 右上隅にある **ユーザーの追加** ボタンをクリックします。ポップアップが表示されます。次に、すべてのフィールドにアカウント情報を入力します

![image-20220420094636-4.png](/static/img/image-20220420094636-4.png)

**ステップ 3**: [OK] ボタンをクリックして確認します。 その後、招待メールが新しいユーザーの受信箱に送信されます。

![image-20220420094720-9.png](/static/img/image-20220420094720-9.png)
