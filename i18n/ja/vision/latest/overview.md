# アカボットビジョン

Akabot Vision は、スキャンされた請求書、契約書、フォーム、電子メール、画像などの非構造化ソースから構造化データを読み取り、分類し、抽出するインテリジェントな文書処理 (IDP) モジュールです。 OCR、機械学習、構成可能な抽出テンプレートを組み合わせて、紙ベースの文書とデジタル文書をワークフローで使用できる実用的なデータに変換します。

![Akabot Vision](/static/img/akabot-vision-01.png)

## 主な特徴

- **OCR エンジン** — スキャンされた PDF、画像 (PNG、JPEG、TIFF)、デジタル ドキュメントから高精度のテキスト抽出を行います。
- **ドキュメント分類** — 受信したドキュメントを内容に基づいて正しい抽出テンプレートに自動的にルーティングします。
- **フィールド抽出** — ビジュアル テンプレート エディターを使用して抽出ルールを定義します。 ML の専門知識は必要ありません。
- **テーブル検出** — 請求書、注文書、レポートから複数行のテーブルを識別して抽出します。
- **信頼スコア** - 抽出されたすべてのフィールドには信頼スコアが含まれるため、信頼性の低いアイテムは人間によるレビューにルーティングできます。
- **ヒューマンインザループ** — データが下流システムに入力される前に、オペレーターが不確実な抽出をレビューして修正できる組み込みの検証ステーション。

## サポートされているドキュメントの種類

- 請求書と注文書
- 身分証明書（パスポート、国民ID）
- 契約と法的合意
- 銀行取引明細書と財務報告書
- 医療記録とフォーム
- テンプレートトレーニングによるカスタムドキュメントタイプ

## はじめに

1. サンプル ドキュメントを Vision ポータルにアップロードします。
2. テンプレート エディターを使用して、ドキュメント タイプとラベル フィールドを作成します。
3. モデルをトレーニングし、抽出精度を検証します。
4. Studio の **Vision** アクティビティを使用して、ドキュメントを送信し、抽出されたデータを取得します。
