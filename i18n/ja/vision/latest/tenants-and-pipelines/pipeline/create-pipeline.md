# 新しいパイプラインの作成
パイプラインは、akaBot Vision の基本的なドキュメント処理ユニットです。

すべてのドキュメント タイプ (請求書、パッキング リスト、クレームなど) に対して、個別のパイプラインを指定する必要があります。

新しいものを作成する場合は、次の手順に従ってください。

**ステップ 1**: ログイン後、ホーム画面の左上隅にある「+ Pipeline」ボタンをクリックします。

![1704877216968-111.png](/static/img/1704877216968-111.png)

**ステップ 2**: パイプライン名を入力します

![1704877336918-513.png](/static/img/1704877336918-513.png)

**ステップ 3**: ドキュメントの種類を選択します。ドキュメント タイプは、事前定義された抽出スキーマです (これは、akaBot Vision がドキュメントから何を抽出するかを定義しますが、後で変更することもできます)。

注: 新しいテナントのデフォルトのドキュメント タイプは「一般請求書」です。あなたの会社で他の種類のドキュメントの使用を希望する場合は、akaBot サポート チーム（[support@akabot.com](mailto:support@akabot.com)）にお問い合わせください。その他のアシスタントが必要です。

![1704877579310-775.png](/static/img/1704877579310-775.png)

**ステップ 4**: ドキュメントをレビューするためにこのパイプラインに参加する 1 人以上のユーザーを選択します (必要な場合)。

![1704879120761-244.png](/static/img/1704879120761-244.png)

ステップ 5: 「OK」ボタンをクリックして、新しいパイプラインの作成を完了します。

![1704879194659-943.png](/static/img/1704879194659-943.png)
