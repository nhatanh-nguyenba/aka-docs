# クッキーポリシー

最終更新日: 2023 年 9 月 17 日

## 1. はじめに
akabotvision.com では、閲覧エクスペリエンスを向上させるために Cookie またはその他の情報収集テクノロジーを使用します。 Cookie により、お客様が当社 Web サイトをどのように使用しているかを把握し、お客様の興味に合わせて Web サイトを調整することができます。ログイン、コンテンツの適切な表示、または設定の保存を可能にするために、一部の Cookie が必要になる場合があります。この Cookie ポリシーは、当社 Web サイト www.akabotvision.com の使用に関連して、akaBot Vision およびその関連会社 (総称して「当社」、「弊社」、または「当社」) に適用されます。当社における個人情報の取り扱いについて詳しくは、当社のプライバシーポリシーをご覧ください。

## 2.「クッキー」とは何ですか?
Cookie は、さまざまな目的で Web ブラウザによってデバイスに保存される小さなテキスト ファイルです。 Cookie はインターネット上のほぼすべての Web サイトで使用されており、akabotvision.com も例外ではありません。通常、お使いのブラウザまたはデバイスでは、Cookie の使用に関する設定を変更できます。設定の構成方法を理解するには、ブラウザまたはデバイスのリファレンス情報を参照してください。
 
## 3. Cookie にはどのような種類がありますか?

**必ず必要な Cookie。**

これらの Cookie は、セキュリティ、ネットワーク管理、アクセシビリティなどの Web サイトの中核機能を有効にするために不可欠です。必要な Cookie を使用すると、ページの読み込み、Web サイト内での移動、およびその機能の使用が可能になります。これらの Cookie を無効にすることもできますが、Web サイトの機能に影響を与える可能性があります。機能性クッキー。これらの Cookie は、ユーザー名、言語、お住まいの地域などの設定を記憶し、強化されたパーソナライズされた機能を提供するために使用されます。これらの Cookie は、Web サイト上のフォームに自動入力するために使用される場合もあります。

**分析クッキー**

これらの Cookie は、訪問者が Web サイトとどのようにやり取りするかを理解するために使用されます。これらの Cookie は、訪問者数、直帰率、トラフィック ソースなどの指標に関する情報を提供するのに役立ちます。 パフォーマンス Cookie。これらの Cookie は、Web サイトの主要なパフォーマンス指標を理解および分析するために使用され、訪問者により良いユーザー エクスペリエンスを提供するのに役立ちます。
 

**マーケティング Cookie。**

これらの Cookie は、お客様にとってより関連性が高く、お客様の興味に合ったオファーやサービスを改善するために使用されます。 Cookie は、広告が表示される回数を制限したり、広告が適切に表示されるようにしたり、マーケティング キャンペーンの効果を測定したりすることもできます。

**ファーストパーティ Cookie とサードパーティ Cookie。**

ファーストパーティ Cookie は、お客様が直接アクセスした Web サイトによって設定される Cookie です。サードパーティ Cookie は、お客様がアクセスする Web サイトを通じて、サードパーティ (当社以外のサービスプロバイダー) のドメインおよび Web サイトによって設定されます。

## 4. 当社が使用するクッキー
追加情報を含むすべての Cookie は、Cookie バナーにリストされます。

## 5. ログファイルとWebビーコンの使用
当社は、お客様が電子メールやリンクなどのコンテンツにアクセスしたことを示す手法として Web ビーコンを使用する場合があります。当社は一部の情報を収集し、ログ ファイルに保存する場合があります。ほとんどの場合、この情報には、Web ブラウザーの種類、インターネット サービス プロバイダー、オペレーティング システム、タイムスタンプ、またはクリックストリーム データ (訪問したページや各ページの滞在時間など) などの匿名の使用状況データのみが含まれます。
