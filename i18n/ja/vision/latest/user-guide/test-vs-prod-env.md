# プロダクション vs.テスト環境のセットアップ
akaBot Vision は数回クリックするだけで開始できます。http://idp.akabot.com にアクセスし、電子メール アドレスを入力し、簡単なフォームに記入します。これで、当社のプラットフォームとテクノロジーを試すことができます。

基本をテストし、akaBot Vision に真剣に取り組みたい場合は、akaBot Vision を使用したテスト環境を構築するためのベスト プラクティスは何なのか、また将来テスト環境から本番環境にどのように移行するのかを疑問に思っているでしょう。

## 本番環境とテスト環境のセットアップ

**http://idp.akabot.com/account/register** でアカウントを登録すると、クラウドに akaBot Vision アカウントが作成されます。これをテナントと呼びます。これは、さまざまな文書処理ワークフローや環境に対応するためにさらに分割できる基本単位です。

テスト環境を実稼働環境から分離するには、主に 2 つのオプションがあります。

* 1 つのテナント内に 2 つの PipelineGroup を作成し、PipelineGroup の 1 つを Test として指定し、もう 1 つを Prod として指定します。
* サインアップ ページから 2 つの別個のテナントを作成し、そのうちの 1 つをテストとして、もう 1 つを製品として指定します。

どちらの場合でも、最初は同じ akaBot Vision クラウドとプラットフォームの同じ機能にアクセスできます。ただし、2 つのアプローチには微妙な違いがあるため、長期的に検討する必要があります。

|  |  |  |
| --- | --- | --- |
|  | **2 つの別個のテナント** | **1 つのテナント内の 2 つのパイプライン グループ** |
| ユーザーアクセス | akaBot Vision ユーザーを作成するには、電子メール アドレスをテナントに追加します。同じ人が両方の環境に参加する必要がある場合、2 番目の環境ではエイリアス電子メールを使用し、異なるユーザー名 ([support@akabot.com](/cdn-cgi/l/email-protection)など) を使用する必要があります。 | テナントを使用してパイプラインごとにユーザー アクセスを簡単に管理できます。ユーザーは、Test のすべてのパイプラインにアクセスできますが、Prod のパイプラインにはアクセスできない場合があります。管理者ユーザーは、すべての PipelineGroup のすべてのパイプラインにアクセスできることに注意してください。 |
| セットアップの移行 | オブジェクトはテナント間で移動できないため、運用テナントでテスト パイプラインのレプリカが必要な場合は、スキーマとパイプラインのセットアップを再作成する必要があります。 | パイプライン間でスキーマをコピーしたり、PipelineGroupgroup 間でパイプラインを移動したりすることが簡単にできます。 |
| データの整合性管理 | ユーザーは、異なる認証情報セットを使用して本番環境にアクセスし、テストすることになるため、本番環境に影響を与える人的エラーのリスクが最小限に抑えられます。 | 管理者アクセス権を持つユーザーは、すべての PipelineGroups、パイプライン、ドキュメントに完全にアクセスでき、無制限にアクセスできます。 |
| 特徴 | テストと運用の PipelineGroups の両方で、サブスクリプションで利用可能なすべての機能を使用できます。そのために 2 つの別個の組織がある場合は、Test に必要な機能についてカスタマー サクセス マネージャーと話し合ってください。 | 両方の PipelineGroup で同じ機能を簡単にセットアップできます。 |

## 実稼働 (有料) サブスクリプションへの切り替え

異なる環境として機能する複数の PipelineGroups を持つ単一のテナントを選択した場合、組織全体が請求システムの有料サブスクリプションに関連付けられます。

別のテスト テナントを選択した場合、それはトライアル テナントとなり、毎月最大 200 ドキュメントを無料で使用できます。この制限を超えると、テスト テナントがブロックされる可能性があります。テスト テナントで必要な機能に確実にアクセスできるようにするには、使用するすべてのテナントについて akaBot Vision のカスタマー サクセス マネージャーに通知してください。

有料サブスクリプションへの移行はバックグラウンドでサイレントに行われ、アプリのエクスペリエンスにはまったく影響しません。
