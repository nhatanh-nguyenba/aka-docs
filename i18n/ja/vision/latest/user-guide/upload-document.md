# ドキュメントのアップロード
これで、「インポート」ボタンを選択するか、指定された指定の電子メール アドレスにドキュメントを送信することで、ドキュメントのアップロードを開始できます (これは API 経由でも行うことができます)。 PDF またはドキュメントのスキャン画像をインポートできます。

![1704876509645-404.png](/static/img/1704876509645-404.png)

![image-20220420184331-2.png](/static/img/image-20220420184331-2.png)

「OK」をクリックすると、AI が自動的にデータフィールドの抽出を開始します。 AI 抽出には通常、A4 1 ページあたり最大 5 秒かかります。

![image-20220420184331-3.png](/static/img/image-20220420184331-3.png)

処理後、人間によるレビューが必要な場合に備えて、文書は「レビュー対象」タブに表示されます。

![image-20220420184331-4.png](/static/img/image-20220420184331-4.png)
