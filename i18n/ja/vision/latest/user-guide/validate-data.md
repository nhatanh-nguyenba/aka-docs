# データを検証する

レビューするドキュメントを選択すると、検証画面が表示されます。

左側のパネルには、抽出されたデータ フィールドの事前定義されたリストが表示されます。

![image-20220420185243-1.png](/static/img/image-20220420185243-1.png)

Tab キーを押してフィールドを 1 つずつ確認します。赤いフラグが付いているフィールドは確認して修正する必要があります。そうしないと、データをエクスポートできません。データ フィールドが正しく抽出されていないことに気付いた場合は、次の方法で調整できます。

* クリックしてポイントします。
* 長方形をドラッグします。
* テキストを直接入力します。
